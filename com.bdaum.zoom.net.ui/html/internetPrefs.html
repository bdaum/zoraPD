Internet preferences

Internet preferences determine how the system connects to the Internet, for example when displaying maps or when geo-naming.

  • System proxy configuration. Enable this option to use the proxy configuration of the platform. 
  • Direct connection to the Internet.  Enable this option to have all connections to remote systems happen directly without involving a proxy server. 
  • Manual proxy configuration. Use this option if connections to remote systems should happen through a proxy server other than the platform proxy.
    • HTTP Proxy. Specify the server for HTTP connections. 
    • Port. Specify the port that is to be used  for HTTP connections. If the port field is empty the default port of 80 is used.
  • FTP accounts. To create an FTP account, press the button New... . To edit an existing account, press the button Edit... . To delete an existing account, press the button Remove.
