Internet-Einstellungen

Die Internet-Einstellungen bestimmen, wie sich das System mit dem Internet verbindet, z.B. beim Anzeigen von Karten und beim Geonaming.

  • System-Proxy-Konfiguration. Aktivieren Sie diese Option, wenn Sie die Proxy-Einstellungen des Betriebssystems verwenden wollen.
  • Direkte Verbindung zum Internet.  Aktivieren Sie diese Option, wenn sämtliche Kommunikation direkt (ohne die Verwendung eines Proxy) erfolgen soll. 
  • Manuelle Konfiguration. Aktivieren Sie diese Option, wenn die Datenübertragung über einen speziellen Proxy-Server erfolgen soll.
    • HTTP-Proxy. Der Proxy-Servername für die HTTP-Kommunikation 
    • Port. Die Port-Einstellung für die HTTP-Verbindung. Wird dieses Feld leer gelassen, wird der Standardport 80 verwendet.
  • FTP-Konten. Um ein neues FTP-Konto anzulegen, betätigen Sie die Taste Neu... . Um ein bestehendes FTP-Konto zu bearbeiten, betätigen Sie die Taste Bearbeiten.... Um ein bestehendes FTP-Konto zu entfernen, betätigen Sie die Taste Entfernen.
