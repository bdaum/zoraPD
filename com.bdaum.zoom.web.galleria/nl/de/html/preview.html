Galleria


http://galleria.io/

Features

  • Intelligentes Vorabladen von Bildern 
  • Navigation mit Hilfe von Vorschaubildern (mit Seitennummerierung) 
  • Diashow 
  • Unterstützung von Bildbeschriftungen 
  • "Download Originals"-Link 
  • EXIF-Anzeige 
  • Themen (zusätzliche Themen können bei http://galleria.io/ erworben werden)
  • Javascript/AJAX basiert. Akzeptable Degradation, wenn Javascript nicht verfügbar. 
  • Multiple Storyboards werden unterstützt
