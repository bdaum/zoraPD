RawTherapee 3 Support

ZoRa Photo Director is able to reflect many of the development settings for RAW and DNG files made with RawTherapee 3.

These settings (recipes) are stored by Raw Therapee in (usually hidden) system folders (cache) and/or as a PP3 sidecar alongside the raw image file. ZoRa Photo Director is able to detect and read these development recipes and apply them when importing RAW or DNG files, refreshing or viewing them, or when deriving images from them during export operations.

To enable this support go the Window>Preferences>Import>Raw Conversion* and check the RawTherapee 3 recipe detector. Make also sure that recipe detection is enabled (Process recipes when importing RAW images).

Im addition, you may also specify which recipe file takes precedence if a recipe file is found both in the cache and as a sidecar.

RawTherapee 3 usually saves this information when you open another image in RawTherapee or when you close the application. If the edited image happens to be located in a watched folder (see Catalog Properties), ZoRa Photo Director will automatically refresh its own catalog thumbnail with the new development recipe.
