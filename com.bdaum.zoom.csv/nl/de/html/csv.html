CSV-Assistent


Die Funktion zum Exportieren von kommagetrennten Werten wird über die Menüfunktion Datei>Exportieren ...>Andere>Kommagetrennte Werte (CSV) aufgerufen.  Sie wird benutzt, um eine Liste von kommagetrennten Werten mit den Metadaten der ausgewählten Bilder zu erzeugen. Solche Listen können leicht in Tabellenkalkulations- und andere Programme importiert werden.

Sie können entweder ausgewählte Bilder oder aber alle Bilder einer Diashow, Ausstellung oder Webgalerie, die Sie im Katalog-Sichtfenster selektiert haben, exportieren.

  • Auf der ersten Seite des Assistenten wählen Sie die Zieldatei aus. Wahlweise können Sie als erste Zeile eine Zeile mit den Namen der ausgewählten Metadatenfeldern erzeugen lassen (empfohlen).
  • Auf der zweiten Seite des Assistenten wählen Sie die Metadatenfelder aus, die Sie exportieren möchten.
  • Auf der dritten Seite des Assistenten können Sie wahlweise jedem Feld einen neueen Namen zuordnen.
