CSV wizard

The function for exporting Comma Separated Values (CSV) is invoked via menu function File>Export...>Others>Comma Separated Values (CSV). It is used to generate a list of comma separated values from the metadata of the selected images. Such lists can be easily imported into spreadsheet and other applications.

You may either export selected images or export all images of a slide show, exhibition, or web gallery selected in the Catalog View.

  • On the first wizard page select the target file. Optionally you can create a first line containing the labels of all selected meta data fields (recommended).
  • On the second wizard page select the metadata fields you want to export.
  • On the third wizard page you can optionally relabel each metadata field.
