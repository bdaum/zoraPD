Manuelles Geotagging


Geotagging kann manuell durchgeführt werden. Wählen Sie die in Frage kommenden Bilder aus und schalten Sie dann auf das Karten-Sichtfenster. Klicken Sie nun auf die Markierungsnadel in der oberen Region. Folgen Sie dann den Anweisungen. Sie können diesen Vorgang auf ein einzelnes Bild oder in einem Schritt auf mehrere Bilder anwenden.

Wenn ein Bild bereits Geotags besitzt, wird deren Lage als rote Markierungsnadel gezeigt. Sie können auch dann auf die beschriebene Weise eine neue Markierung setzen, jedoch nur, wenn ein einzelnes Bild bearbeitet wird, oder wenn alle Bilder die gleichen Koordinaten besitzen.

In der Karte arbeiten Sie in der Regel im verriegelten Modus. Durch Umschalten der Verriegeln-Schaltfläche () in der Sichtfenster-Werkzeugleiste, können Sie die Verriegelung aufheben Dies bedeutet, dass die Karte der Bildauswahl nicht mehr folgt, sondern ihren alten Inhalt beibehält. So können Sie die Karte auch für andere Zwecke einsetzen.
