Manuelles Geotagging

Geo-Tags können manuell angewendet werden. Wählen Sie die betreffenden Bilder aus und wechseln Sie in die Kartenansicht.

Im oberen Bereich sehen Sie einen oder zwei Pins:

  • Kamera-Pin (). Verwenden Sie diesen Pin, um den Ort festzulegen oder zu ändern, an dem das Bild erstellt wurde. Nach einem Klick auf den Pin erscheint dieser in der Karte. Ziehen Sie ihn auf seinen Bestimmungsort. Nach dem Ziehen wird er rot.
  • Richtungs-Pin (). Verwenden Sie diesen Pin, um entweder die Kamerarichtung oder sowohl die Kamerarichtung als auch die angezeigte Position festzulegen. Klicken Sie auf den Pin und wählen Sie im nachfolgenden Dialog den Modus aus. Ein Pin wird in der Karte angezeigt. Ziehen Sie ihn in die gewünschte Richtung bzw. den gewünschten Ort.
  • Gezeigter-Ort-Pin (). Dieser Pin erscheint anstelle des Richtungs-Pins wenn die Kameraposition unbekannt ist. Nach einem Klick auf den Pin erscheint dieser in der Karte. Ziehen Sie ihn auf seinen Bestimmungsort.

Der IPTC-Standard erlaubt einen Standort für die Kamera, aber mehrere im Bild gezeigte Orte. Ein Panorama-Foto kann beispielsweise mehrere interessante Orte zeigen. Durch das Setzen eines neuen Kamera-Pins wird daher ein neuerKamerastandort für alle ausgewählten Bilder definiert (und somit alle vorhandenen Kamera-Pins in der Karte ersetzt). Durch das Setzen einer neuen gezeigten Ortes wird im Gegensatz dazu einfach ein neuer Pin für den gezeigten Ort zur Karte  hinzugefügt.

Bereits in der Karte angezeigte Pins können an eine neue Position gezogen werden. Pins können gelöscht werden, indem man sie zunächst mit einem Einfachklick selektiert (ein bläulicher Punkt erscheint auf dem Pin) und dann mit einem Klick auf die Löschtaste () im oberen Bereich des Kartensichtfensters löscht.

Durch Doppelklicken auf einen Pin können Sie zusätzliche Informationen über den Pin anzeigen.
