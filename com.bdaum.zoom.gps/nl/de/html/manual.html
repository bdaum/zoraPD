Manuelles Geotagging


Geotagging kann manuell durchgeführt werden. Wählen Sie die in Frage kommenden Bilder aus und schalten Sie dann auf das Karten-Sichtfenster. Klicken Sie nun auf die linke Markierungsnadel in der oberen Region. Sie können diesen Vorgang auf ein einzelnes Bild oder in einem Schritt auf mehrere Bilder anwenden.

Im  Falle eines Einzelbildes können Sie auch die Bildrichtung (also die Himmelsrichtung, in die die Kamera ausgerichtet war) mit einem Klick auf die benachbarte Richtungsnadel bestimmen.

Wenn ein Bild bereits Geotags besitzt, wird deren Lage als rote Markierungsnadel gezeigt. Sie können auch dann auf die beschriebene Weise eine neue Markierung setzen, jedoch nur, wenn ein einzelnes Bild bearbeitet wird, oder wenn alle Bilder die gleichen Koordinaten besitzen.
