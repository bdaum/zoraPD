Das Kartensichtfenster


Die Karte zeigt den Standort der derzeit ausgewählten Bilder. Darüber hinaus ermöglicht sie das manuelle Hinzufügen von Geo-Tags zu den ausgewählten Bildern, oder die geografische Lage eines Bildes zu verändern. Abhängig von der aktuellen Auswahl zeigt sie die Position der Bilder als rote Markierungen. Sie können diese Markierungen auf andere Positionen ziehen, um die jeweilige Bildposition zu verändern.

Alernativ können neue Koordinaten können für ALLE selektierten Bilder gesetzt werden, indem Sie auf die linke Markierungsnadel im oberen Bereich klicken.

Das Kartensichtfenster kann frei mit anderen Sichtfenstern kombiniert werden. Standardmäßig befindet es sich hinter dem jeweiligen Galleriesichtfenster. Wenn es nicht sichtbar sein sollte, können Sie es mit Fenster>Sicht anzeigen>Andere ...>GPS>Karte sichtbar machen.

Lokale Werkzeugleiste und lokales Drop-down-Menü

  • Klicken Sie  In externem Webbrowser anzeigen (), um die aktuelle Position in einem externen Webbrowser anzuzeigen. Das ist sinnvoll, um weitergehende Kartenfunktion wie z.B. die Routenplanung zu verwenden.
  • Klicken Sie Karte aktualisieren (), um neue Marker in die Standardform zu überführen.
  • Außerdem befindet sich in der lokalen Werkzeugleiste das Auswahlfeld für den zu verwendenden Kartendienst.

Kopfbereich des Sichtfensters

  • Links befinden sich eine oder zwei Markierungsnadels. Mit einem Klick auf die linke Nadel können Sie das manuelle Geo-Taggen von selektierten Bildern anstossen. Mit einem Klick auf die rechte Nadel können Sie die Bildrichtung festlegen.
  • Rechts befindet sich das Suchfeld. Durch Eingabe von Ortsnamen könne Sie diese Orte auf der Karte suchen. Die Suche erfolgt dabei mit Hilfe des ausgewählten Geo-Naming Webdienstes. Um einen Webdienst auszuwählen und mit Parametern zu konfigurieren, klicken Sie auf die Suchen-Taste während Sie Strg gedrückt halten.
    Ersatzweise können Sie auch Koordinaten oder Google Plus Codes eingeben. Gültige Formate sind:
    • Grad, Minuten und Sekunden: 41°24'12.2"N 2°10'26.5"E
      oder: N41°24'12.2" E2°10'26.5"
    • Grad und Dezimalminuten: 41 24.2028, 2 10.4418
    • Dezimalgrad: 41.40338, 2.17403
    • Google Plus code: C53F+5M
    Frühere Eingaben können Sie über die Drop-down-Liste des Textfeldes abrufen.
  • Mit derZurück-Taste () können Sie zur vorigen Kartenposition zurückgehen, z.B. nachdem Sie mit einer Suche eine andere Position aufgesucht hatten. 
  • Mit der Vorwärts-Taste () können Sie zur nächsten Kartenposition gehen, z.B. nachdem Sie die Rückwärts-Taste benutzt hatten.

Mausgesten in Karten

  • Mit einem Klick auf eine Markierungsnadel können Sie näher heranzoomen. Abhängig von der anfänglichen Zoomstufe kann das auch wiederholt angewendet werden, um noch näher heranzuzoomen.
  • Mit einem Doppelklick auf eine Markierungsnadel wird das zugeordnete Bild in einer Info-Blase angezeigt. Sind mehrere Bilder der Nadel zugeordnet, wird nur das erste Bild gezeigt.
