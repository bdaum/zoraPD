Standorte


Der IPTC-Standard definiert verschiedene Standorte, die für ein Bild eingetragen werden können:

  • Erstellungsort (IPTC>Verwaltung>Aufnahme-Standort) zeigt die Position der Kamera. Diese Einträge werden automatisch beim Geotagging erstellt.
  • Gezeigter Ort (IPTC>Herkunft>gezeigter Ort). Hier können mehrere Orte eingetragen werden. Diese Standorte werden in der Regel manuell eingetragen.

Jeder Standort besteht aus hierarchischen Ortsnamen (Weltregion, Land, Bundesland, Stadt, Details) und/oder Koordinaten. ZoRa Photo Director führt eine Liste der bereits definierten Standorte für die schnelle Auswahl. Neue Standorte können definiert werden, indem Sie die Add ... -Taste betätigen und die Daten manuell eingeben oder auf der Karte markieren. Betätigen Sie hierzu zunächst die Karte ... -Taste, und klicken Sie dann auf die grüne Markierungsnadel in der oberen Region des Kartendialogs, dann auf die Position in der Karte.

Standortangaben können mit Hilfe der Menüfunktion Katalog>Eigenschaften>Standorte betrachtet und bearbeitet werden.
