Import von Geodaten


Importieren von Geodaten erfolgt entweder über die Menüfunktion Metadaten> Geotagging oder einfach, indem Sie eine unterstützte Tracker-Datei auf eine Kollektion oder in einen Galeriebereich ziehen. Zu Zeit werden die folgenden Formate unterstützt:
  • GPX. mit der Dateinamenerweiterung .gpx
  • NMEA. mit den Dateinamenerweiterungen .nmea und .log. $GPRMC Nachrichten  in diesen Dateien werden ausgewertet, andere Nachrichten werden ignoriert.

Den Importprozess können Sie unter Fenster>Benutzervorgaben>GPS* konfigurieren.

*) Auf dem Mac: ZoRa Photo Director>Einstellungen>...
