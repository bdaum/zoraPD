Manual Geotagging

Geo-tags can be applied manually. Select the images in question, then switch to the Map view.

In the top region you will see one or two pins:

  • Camera pin (). Use this pin to set or modify the location where the image was created. Click the pin and it will appear in the map. Then drag it to the desired location. After dragging it will turn red.
  • Direction pin (). Use this pin to set either the camera direction or both the camera direction and the location shown. Click the pin and select the desired mode from the dialog box. A pin will appear in the map. Drag it to the desired direction or location.
  • Location show pin (). This pin appears instead of the direction pin of the camera location is undefined. Click the pin and it will appear in the map. Then drag it to the desired direction.

The IPTC standard allows for one location for the camera but for multiple locations shown in the image. For example, a panorama photo may show several interesting locations. Therefore, setting a new camera pin will define a new camera location for all selected images (and thus replace all existing camera pins in the map). Setting a new location shown, in contrast, will simply add a new location shown pin to the map.

Pins already displayed in the map can be dragged to a new position. Pins can be deleted by first selecting the pin with a single click (a blueish dot is displayed on the pin) and then click the Delete button () in the top region of the map view.

By double clicking a pin you can display additional information about the pin.
