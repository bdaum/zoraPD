Manual Geotagging

Geo-tags can be applied manually. Select the images in question, then switch to the Map view. Click on the pin in the top region, then follow the instructions. You may apply this operation to a single image or to several images in one step.

If an image already is geo-tagged, its location is displayed as a marker. You may set a new marker to change the current position. However, this is only possible with single images or with images that all have the same coordinates.

In the Map view you usually work in the Lock mode. By toggling the Lock button () in the views tool bar you can unlock the view. This means that the view content will not change when the image selection changes. This mode facilitates the Map view for other purposes than geo-tagging.
