Searching geographically

There are several options to utilize geographical data when searching for images:

  • Using geographical data as search criterion in a Collection or an Adhoc Query. 
    • You may search directly for values of GPS altitude, GPS latitude, GPS longitude and GPS speed (Group Exif) .
    • You may search for locations (Location created and Location shown). In this case you cannot enter data freely. Instead you select from a list of defined locations (Group IPTC).
    • You may search for place names in the keywords (Group IPTC), provided the option Include place names into keywords was switched on while importing images (Window>Preferences>GPS*).
  • When viewing meta data in the IPTC view or the Metadata view you can click on the arrow buttons at the fields IPTC>Administration>Location created and IPTC>Origin>Location shown to find matching images.
  • Using menu function Metadata>Proximity search you can find images in the vicinity of selected images. All images will be shown with a distance shorter or equal than the specified distance to at least one selected image.
  • You may use the keyword search (Metadata>Keyword Query...) to find images by place names (city, street, country, etc). To enable this possibility, it is necessary to keep option Include place names into keywords switched on while importing images (Window>Preferences>GPS*).
  • You may search for place names using the function Metadata>Text Query... .

*) On Mac use ZoRa Photo Director>Preferences>...
