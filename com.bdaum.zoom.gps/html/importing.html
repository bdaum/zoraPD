Importing geo-data

Importing of geo-data is performed either through the menu function Metadata>Geotagging or simply by dropping a supported tracker file onto a collection or into a gallery area. Currently the following file formats are supported:

  • GPX. with file extension .gpx
  • NMEA. with file extensions .nmea and .log. $GPRMC messages contained in these files are evaluated, other messages are ignored.
You may configure the import process through Window>Preferences>GPS*.
