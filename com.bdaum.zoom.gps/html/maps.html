The Map View

The Map view shows the location of currently selected images. In addition it allows to manually add geo-tags to selected images, or to modify the geographical location of an image. Depending on the current selection it shows the location of images as red or yellow markers. New coordinates can be applied by clicking on the green marker in the top section, then dragging it to the desired position.

The Map view can be freely combined with other views. By default, it is located behind the Lightbox view and the Sleeves view. If not visible, you can open it with Window>Show view>Other...>GPS>Map.

Local toolbar buttons and local drop down menu

  • By clicking Show in external web browser () you can display the current location in an external web browser. This makes sense if you want to use advanced mapping functions such as route planning.
  • Click the Refresh map button () to convert new markers into their standard representation.
  • On the local toolbar you also find the field for selecting the map service of your choice.

View header area

  • An the left side you find one or two pin symbols. With a click one of these symbols you can define the geographic position for the selected images or define the image direction (the direction into which the camera was turned).
  • On the right hand side there is a text field for entering queries. By entering location names you can position the map to these locations. These search operations are performed through the selected geo-naming web service. You can choose a web service and configure it with parameters by Ctrl-clicking the Search button.
    Alternatetively, you can enter coordinates or Google Plus Codes, too. Valid formats are:
    • Degrees, minutes and seconds: 41°24'12.2"N 2°10'26.5"E
      or: N41°24'12.2" E2°10'26.5"
    • Degrees and decimal minutes: 41 24.2028, 2 10.4418
    • Decimal degrees: 41.40338, 2.17403
    • Google Plus code: C53F+5M
    You can recall previous entries via the drop-down list of the text field.
  • By clicking the Back button () you can return to the previous position shown in the map for example after performing a location search. 
  • By clicking the Forward button () you can move to the next position shown in the map for example after clicking the Back button.

Map mouse gestures

  • By clicking on a marker you can zoom in on the marker point. Depending on the initial zoom level, this may be applied twice for moving even closer.
  • With a double click on a marker you can display the associated image in an info pop-up. If several images are associated with a marker, only the first image is shown.
