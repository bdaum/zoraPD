Advanced Search

The optionally installed advanced search functions comprise the following:
  • Free text search. You can search for any text occuring in the context of an image: e.g. keywords, headlines, text in slides, exhibits, and other texts. Which text elements are indexed for search can be configured in the Catalog Properties, section Indexing. You can use search expressions and the results will contain a score to indicate the likeliness of a match.
  • Content based search. You can search for images based on the visual similarity of a given image. For each search request it is possible to combine keyword and content based search and to modify the given image using paint tools. The algorithms used can be configured in the Catalog Properties, section Indexing.

    Image similarity is based on color, texture and shape. Different algorithms interpret those features in different ways. Which algorithm is suited best depends on your image set. Landscape images, for example, will require a different algorithm as company logos. You can index your images with multiple algorithms and then, when searching, try out several algorithms to find out which works best for your images. But beware of using to many algorithms for indexing because this slows down the indexing process.

    A special case are AI algorithms (marked with ) such as the Clarifai algorithm. These algorithms don't use low level functions but compare high level image concepts obtained from a deep neural network. Here the similarity is rather semantic than visual. The downside is that these services typically require a web service. When indexing, each image will generate a web service request. For large catalogs this may exceed the capacity of free web service accounts, so a paid account would be needed.

The search field in the main toolbar accepts both keyword expressions and images which you can simply drag into the field. If the Advanced Search feature is not installed, the search field performs keyword searches.

Both text search and content based search use the Lucene search engine. The content based search, in addition, is based on the Lire libraries.
