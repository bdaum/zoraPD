Netzwerkeinstellungen

Die Netzwerkeinstellungen werden über Fenster>Einstellungen>Netzwerk* aufgerufen.

Eigene Adresse. 

  • Hier finden Sie Ihre eigene Netzwerkadresse.
    • Name des Computers (IP-Adresse). Nur zur Information. Dieses Feld kann nicht geändert werden.
    • Port. Der verwendete Kommunikationsport. Normalerweise sollten Sie den bereitgestellten Standardwert verwenden. In einigen Fällen kann der Port jedoch bereits in Verwendung sein, zum Beispiel, wenn ein anderer Peer auf demselben Computer ausgeführt wird. In diesem Fall wählen Sie einen anderen Wert. Die Anwendung muss nach einer Änderung der Portnummer neu gestartet werden.

Freigegebene Kataloge

Hier listen Sie alle Kataloge, auf die von Klienten (aber bei einer Netzwerksuche auch von Ihnen selbst) zugegriffen werden kann, auf.

  • Verwenden Sie Schaltfläche Hinzufügen, um der Liste einen neuen Katalog hinzu zu fügen. Das können sowohl der aktuell geöffnete Katalog als auch alle anderen .zdb-Kataloge auf Ihrem Computer sein. Mit einem Klick in die Spalte "Datenschutz" können Sie die Datenschutzeinstellungen für jeden einzelnen Katalog anpassen:
    • Alle. Klienten dürfen auf alle Bilder im Katalog zugreifen.
    • Medium. Klienten dürfen nur auf Bilder zugreifen, die als "Öffentlich" oder "Medium" klassifiziert wurden.
    • Nur öffentliche Elemente. Klienten dürfen nur auf Bilder zugreifen, die als "Öffentlich" klassifiziert wurden.
    • Lokal. Der Katalog ist nicht für Klienten zu erreichen. Dies ist die typische Einstellung für die lokale Suche über mehrere Kataloge, ohne Katalog für andere Klienten frei zu geben.
  • Verwenden Sie die Schaltfläche Zugriffsrechte, um die Zugriffsrechte eines bestimmten Klienten für den ausgewählten Katalog zu reduzieren oder zu erweitern. (Solche Einschränkungen können Katalogen mit der Datenschutz-Einstellung "Lokal" nicht hinzu gefügt werden).
  • Standardmäßig kann jeder Peer einen Katalog durchsuchen und die Ergebnisse in voller Auflösung ansehen:
    • Markieren Sie Suchen, um dem Klienten die Suche im Katalog zu ermöglichen.
    • Markieren Sie  Betrachten, um dem Klienten das Ansehen in voller Auflösung zu ermöglichen.
    • Markieren Sie Sprachnotizen, um dem Klienten die Wiedergabe von Sprachnotizenzu ermöglichen.
    • Markieren Sie Kopieren, um dem Klienten das Kopieren der Bilddateien zu ermöglichen. (Wenn Sprachnotizen nicht markiert ist, können die zugehörigen Sprachnotizen nicht kopiert werden.)
    Wenn Sie alle Markierungen entfernen, wird der Katalog für den angegebenen Klienten geschlossen.
    Achtung: Zugriffsrechte betreffen immer den gesamten Peer-Computer und nicht einen einzelnen Kommunikationsport!
  • Verwenden Sie die Schaltfläche Entfernen, um die ausgewählten Zugriffsrechte oder den ausgewählten Katalog aus der Liste zu entfernen.

Peers

  • Peers die in eine Netzwerk-Suche eingeschlossen werden
    Hier legen Sie eine Liste mit aller Peerknoten an, auf die Sie im Rahmen einer Netzwerksuche zugreifen möchten:
    • Verwenden Sie die Schaltfläche Hinzufügen einen neuen Knoten zur Liste hinzu zu fügen. Dazu wird der Computername und Kommunikationsport angeben. Wenn Sie eine IP-Adresse anstelle des Computernamens verwenden möchten, stellen Sie sicher, dass der Knoten eine feste IP-Adresse verwendet. Einige Betriebssysteme weisen nach jedem neuen Start eine andere IP-Adresse zu.
    • Verwenden die Schaltfläche Entfernen, um die ausgewählte Peer-Definition aus der Liste entfernen.


Eingehende Anrufe

Hier können Sie überwachen, welche Klienten auf Ihre Kataloge zugegriffen haben. Wenn Sie nicht möchten, dass ein bestimmter Klient auf Ihre Kataloge zuzugreift, wählen Sie ihn einfach in der Liste aus und klicken Sie auf die Schaltfläche Schwarze Liste an/aus. Ein weiterer Klick stellt die Zugriffsrechte dieses Klienten wieder her.

Achtung: Ein Eintrag in der Schwarzen Liste betrifft immer den gesamten Peer-Computer und nicht einen einzelnen Kommunikationsport!

Durch Klicken auf die Schaltfläche Löschen können Sie alle Einträge aus der Liste entfernen - mit Ausnahme der Einträge in die schwarze Liste.
