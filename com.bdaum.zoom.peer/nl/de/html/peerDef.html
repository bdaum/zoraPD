Peer-Definition

Hier definieren Sie welche anderen Knoten in eine Netzwerksuche eingeschlossen werden:

  • Spitzname. Das ist ein optionaler Name, der Sie daran erinnern soll, welcher Peer zur im nächsten Feld folgenden Netzwerkadresse gehört.
  • Computername oder feste IP-Addresse. Name des Peer-Computers.  Wenn Sie eine IP-Adresse anstelle des Computernamens verwenden möchten, stellen Sie sicher, dass der Knoten eine feste IP-Adresse verwendet. Einige Betriebssysteme weisen nach jedem neuen Start eine andere IP-Adresse zu.
  • Port. Der Kommunikationsport des Peers auf dem angegebenen Computer.
