Zugriff auf Ressourcen

Nach einer erfolgreichen Netzwerksuche sind weitere Zugriffe auf die gefundenen Einträge möglich:

  • Es ist möglich, das Objekt in voller Auflösung anzusehen.
  • Sprachnotizen  können abgespielt werden.
  • Die Funktion Verschieben kann verwendet werden, um ein Element und dessen Dateien von der Gegenstelle zu kopieren.
  • Was Sie nicht tun können, ist, ein entferntes Objekt in irgendeiner Weise zu bearbeiten (z.B. Drehen, Farbkodierung, Bewertung). Stattdessen müssen Sie das Objekt zunächst (über Datei>Verschieben) auf Ihr lokales Dateisystem kopieren. Dann können Sie die lokale Kopie bearbeiten.

Alle diese Operationen können durch Vergabe von Zugriffsrechten bei der Definition der freigegebenen Katalogen kontrolliert werden. Standardmäßig hat jeder Klient das Recht, im Katalog nach Bildern zu suchen und Bilder in voller Auflösung anzusehen.
Jeder Anbieter kann jedoch nach Belieben diese Rechte einschränken (z.B. das Ansehen in voller Auflösung zu sperren) oder zu erweitern (z.B. das Abspielen von Sprachnotizen und das Kopieren von Bildern zu ermöglichen). Dies erfolgt in der Definition der gemeinsamen Kataloge (über Fenster> Einstellungen>Netzwerk>Gemeinsame Kataloge*).

Zusätzlich kann jeder Anbieter andere Klienten komplett vom Zugriff auf seine Kataloge ausschließen. Diese erfolgt unter Fenster> Einstellungen>Netzwerk>Eingehende Anrufe*.
*) Auf dem Mac benutzen Sie ZoRa Photo Director>Einstellungen>...
