Zugriffsrechte

Hier definieren Sie die Zugriffsrechte eines bestimmten Peers für den ausgewählten Katalog. Sind keine Zugriffsrechte angegeben, kann jeder Klient sowohl den freigegebenen Katalogen suchen als auch Objekte in voller Auflösung ansehen.

  • Computername oder feste IP-Addresse. Name des Peer-Computers. Wenn Sie eine IP-Adresse anstelle des Computernamens verwenden möchten, stellen Sie sicher, dass der Peer-Knoten eine feste IP-Adresse verwendet. Einige Betriebssysteme weisen nach jedem neuen Start eine andere IP-Adresse zu.
  • Markieren Sie Suchen, um dem Klienten die Suche im Katalog zu ermöglichen.
  • Markieren Sie  Betrachten, um dem Klienten das Ansehen in voller Auflösung zu ermöglichen.
  • Markieren Sie Sprachnotizen, um dem Klienten die Wiedergabe von Sprachnotizenzu ermöglichen.
  • Markieren Sie Kopieren, um dem Klienten das Kopieren der Bilddateien zu ermöglichen. (Wenn Sprachnotizen nicht markiert ist, können die zugehörigen Sprachnotizen nicht kopiert werden.)
    Wenn Sie alle Markierungen entfernen, wird der Katalog für den angegebenen Klienten geschlossen.
