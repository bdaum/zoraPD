Access rights definition

Here you define access rights for specific peers for the selected catalog. By default, each peer can search catalogs and view the results in full resolution.

  • Computer name or fix IP address. Name of the peer's computer. If you want to use an IP address instead of the computer name make sure that the peer uses a fix IP address. Some operation systems allocate a different IP address after each new start.
  • Search. Allow the peer to search the catalog.
  • View. Allow the peer to view items in full resolution.
  • Voice note. Allow the peer to replay voice notes.
  • Copy. Allow the peer to copy an item to his own node. (If Voice note is not marked, attached voice notes are not copied.)
                Removing all marks will close the catalog for this peer.
