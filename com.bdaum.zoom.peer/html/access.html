Accessing Resources

After a successful network search further accesses to the items found are possible:
  • It is possible to view found items in full resolution
  • Voice note annotations can be played
  • The Move function can be used to copy an item and its files from the remote peer.
  • What you cannot do is to modify a remote item in any way (e.g. Rotate, Color code, Rate). Instead, you need to copy the remote item (via File>Move) to your local filesystem. Then you can edit the local copy.

All these operations can be controlled by defining rights in the definition of shared catalogs. By default, each consumer has the right to search a catalog and to view the search results in full resolution.
Each provider is free to further restrict these rights (e.g. inhibit viewing in full resolution) or to extend them (e.g. allow voice note replay or copying of files). This is done through the Shared Catalog definition in Window>Preferences>Network>Shared Catalogs*.

Furthermore, each provider can block another peer from accessing his catalogs altogether by blacklisting him. This is done in Window>Preferences>Network>Incoming Calls*.
