Network troubleshooting

If a network search or another network access does not perform as expected, check the following:
  1. Make sure that the peers you want to access are in the list of peers. ( See Window>Preferences>Network>Peer nodes*). Make sure that the computer name and the port number are valid. If you use IP addresses make sure that the IP address of the peer has not changed.
  2. Make sure that the peers are online. In the Status column in Window>Preferences>Network>Peer nodes* you can see which peers are online and which are not.
    It is a good idea to start all peer ZoRa PhotoDirector applications when the operating system starts (Autostart) and to execute the program in Tray mode (Window>Preferences>Application*).
  3. Make sure that the required catalogs are shared by your peers and that you have the necessary rights to perform the operations wanted. You need to talk to your peers or to your network administrator.
