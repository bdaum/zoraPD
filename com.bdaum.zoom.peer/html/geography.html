Defining the network geography

In a network a location of a node is defined by the computer name and the communication port used. Alternatively, an IP address can be specified instead of the computer name, but it must be made sure that the IP address does not change when the computer is restarted. 

Each network node can define a list of peer nodes that the node whishes to access. Vice versa, other nodes can put this node on their lists of accessed nodes. See Window>Preferences>Network>Peer nodes*
