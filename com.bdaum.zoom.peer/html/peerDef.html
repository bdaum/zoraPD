Peer definition

Here you define a peer to be included into network search operations:

  • Nickname. This is an optional name which helps you to remember the peer belongig to the network address.
  • Computer name or fix IP address. Name of the peer's computer. If you want to use an IP address instead of the computer name make sure that the peer uses a fix IP address. Some operation systems allocate a different IP address after each new start.
  • Port. The peer's communication port on the specified computer.
