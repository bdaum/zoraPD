Clarifai

Clarifai ist einer der führenden Dienstleister für die Analyse von Bildinhalten. Clarifai bietet mehrere Preispläne, beginnend mit einem freien Plan. Zum Zeitpunkt des Schreibens sind beim freien Plan 5000 Anfragen pro Monat erlaubt.
Bevor Sie den Service nutzen können, müssen Sie ein Konto bei Clarifai eröffnen. Um dies zu tun, besuchen Sie die Clarifai-Website unter https://www.clarifai.com oder klicken Sie auf den Link in den Clarifai-Benutzereinstellungen. Sie müssen dann Ihre Anmeldeinformationen (Client-ID und Client-Secret) in die Clarifai-Präferenzseite eingeben.

Clarifai unterstützt mehrere "Modelle" wie Allgemein, Food, Wedding und Travel. Das Modell, das in ZoRa PhotoDirector verwendet wird, wird aus dem Thema des aktuellen Katalogs abgeleitet (Katalog>Eigenschaften ...).

Optionen

  • Modell. Das verwendete Clarifai-Wissensmodell wird normalerweise aus dem Katalogthema (siehe oben) abgeleitet. Es besteht allerdings die Möglichkeit, diese Einstellung temporär zu überschreiben, z.B. wenn eine Reise-Album innerhalb eines Katalogs mit allgemeinem Theme kategorisiert werden soll.
  • Sprache. Wählen Sie eine Ausgabesprache. Zur Zeit unterstützt nur das allgemeine Modell andere Sprachen als Englisch. Für andere Modelle können Sie den Übersetzungsdienst verwenden.
  • Maximale Anzahl von Konzepten. Für jedes Bild schlägt Clarifai etwa zwanzig verschiedene Kategorien vor (auch Konzepte genannt). Sie können diese Anzahl durch eine entsprechende Angabe einschränken.
  • Minimales Konfidenzniveau. Alle Vorschläge werden mit einem Vertrauensniveau angegeben. Sie können einen Wert zwischen 0 und 100 (in Prozent) eingeben, um alle Vorschläge mit einem niedrigeren Konfidenzniveau zu unterdrücken.
  • Schlagworte vormarkieren bei Vertrauensnievau über. Geben Sie ein Konfidenzniveau an, für das das System automatisch Vorschläge als Schlagworte markieren soll. Geben Sie einen Wert von 100 an, wenn Sie keine automatische Markierung wünschen.
  • Nur bekannte Schlagworte vormarkieren. Diese Option beschränkt die automatische Kennzeichnung auf Schlüsselwörter, die bereits im Katalog definiert sind.
  • Auf sichere Inhalte überprüfen. Markieren Sie diese Option, um eine Prüfung auf  sexuelle oder rassistische BildInhalte zu veranlassen. Die Ergebnisse werden verwendet, um die Datenschutzeinstellungen des Bildes zu setzen. Wenn diese Option gesetzt ist, wird für jedes Bild eine zusätzliche Clarifai-Anfragen erzeugt.
  • Gesichter erkennen. Markieren Sie diese Option, um eine Gesichtserkennung durchzuführen. Werden Gesichter erkannt, werden diese durch grüne Rahmen im Bild gekennzeichnet. Sie haben dann die Möglichkeit, diese Gesichtsregionen zu akzeptieren oder zu verwerfen. Wenn diese Option gesetzt ist, wird für jedes Bild eine zusätzliche Clarifai-Anfragen erzeugt.
  • Übersetzen. Markieren Sie diese Option, wenn Sie Englisch als Standard-Clarifai-Sprache verwenden, aber die Ausgabe in einer anderen Sprache wünschen. Der Microsoft-Translator-Dienst wird verwendet, um die übermittelten Texte zu übersetzen.
