Clarifai

Clarifai is one of the leading service providers for image content analysis. They offer several pricing plans, starting with a free plan. At the time of writing the free plan allowed for 5000 images per month.
Before you can use the service you must open an account with Clarifai. To do so, visit their website at https://www.clarifai.com or click the link in the Clarifai preference page. You must then enter your credentials (Client ID and Client secret) into the Clarifai preference page.

Clarifai supports several "models" such as General, Food, Wedding, and Travel. The model used in ZoRa Photo Director is derived from the theme set for the current catalog (Catalog> Properties...).

The Clarifai service is also used to establish a semantic similarity between images or faces (optionally). This allows to search for images that depict similar concepts or faces. The service is used, too, for automated image rating.

Options

  • Model. This is by default set by the catalog theme (see above). However, you may override that setting temporarily, for example, if you want to categorize a travel album within a catalog with a general theme.
  • Language. Choose an output language. Currently, only the General model supports other languages than English. For other models you may want to use the translation service.
  • Maximum number of concepts. For each image Clarifai proposes around twenty different categories (also called concepts). You may restrict this by setting a maximum number of proposals.
  • Minimum confidence level. All proposals are given with a confidence level. You may enter a value between 0 and 100 (in percent) to supress all proposals with a lower confidence level.
  • Pre-mark keywords when confidence is above. Specify a confidence level for which the system should automatically mark proposals as keywords. Specify a value of 100 if you do not wish automatical marking.
  • Pre-mark only known keywords. This option will restricts automatical marking to keywords that are already defined in the catalog.
  • Check "safe for work". Mark this option to check if an image contains sexual or racist contents. The results will be used to set the images privacy settings. If this option is set an additional Clarifai request is issued for each image.
  • Detect faces. Mark this option if you want the service to detect faces in the image. If faces are found bounding boxes in green color will be drawn in the image. You have then the option to accept those face regions or to dicard them. If this option is set an additional Clarifai request is issued for each image.
  • Detect celebrities. Mark this option if you want the service to detect faces of celebrities in the images. If this option is set an additional Clarifai request is issued for each image.
  • Translate. Mark this option if you use English as your default Clarifai language but want the output in another language. The Microsoft Translator Service will be used to translate the returned texts.
