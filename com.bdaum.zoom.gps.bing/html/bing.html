Bing Maps

Bing Maps - provided by Microsoft - feature a road view and an aerial (satellite) view. By obtaining (http://www.microsoft.com/maps/developers/) and specifying a Bing Map key (Window>Preferences>GPS>Bing*) you can activate additional views such as the Bird Eye's view. However, the mapping system can be operated without entering a key, too.

Maps can be displayed in 2D-mode and 3D-mode. Geo-tagging, however, is supported only in the 2D-mode.

There are a few limitations: Markers for existing locations cannot be dragged, and dragging for new markers cannot be aborted with ESC.
