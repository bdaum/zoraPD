Email wizard

The email function is invoked via menu function File>Export...>Email>Images. It is used to transmit the selected images to the platform's email client.

You may either export selected images or export all images of a slideshow, exhibition, or web gallery selected in the Catalog View.

You have the choice to send either originals (which can be rather large and maybe unreadable by the receiver) or to downsize the images and send them in the popular JPEG format. 

The size of these JPEGs can be adjusted using the scale. Alternatively, you may enter a maximum size (either the width for landscape orientation or the height for portrait orientation).

You can influence the Output Quality with these setttings.

You may also add a subject and a message. The mail must still be completed with the receivers email address but this will be done in the email client.

The following options are available for the export operation:

  • Optionally, you may add watermarks for the protection of you intellectual property to images (not when uploading originals).
  • You can track the export. Later, you may view these records in the Properties view under ZoRa Photo Director>Track.
  • If you want to include meta data (EXIF and IPTC) into the generated JPEG files, check the option Include metadata. On the following wizard page you can then select the meta data that you want to transmit.
  • You can filter images and send only public or public+medium images.
