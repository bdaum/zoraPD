HTML wizard

The HTML function is invoked via menu function File>Export...>Others>HTML It is used to generate a HTML page from the selected images into the chosen folder, which can be a local or a folder on an FTP account.

You may either export selected images or export all images of a slide show, exhibition, or web gallery selected in the Catalog View.

You may change the proposed layout and the image quality. You can influence the Output Quality with these setttings.

The layout options follow mainly the Print dialog.

Additional formatting can be achieved via the generated CSS stylesheet styles.css. Existing stylesheets are not overwritten.
