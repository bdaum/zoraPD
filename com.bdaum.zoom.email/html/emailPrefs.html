Email preferences

For sending emails from ZoRa PhotoDirector you have two options:

  • Use the platforms email client. The advantage is that you have access to your address book when specifying receivers. The disadvantage is that you must perform an additional step. Also, under some platforms, the process is a bit cumbersome. Under Windows, ZoRaPD generates due to technical limitations an email message which you must then explicitly forward to the receivers,
  • Don't use the platforms email client. In this case ZoRaPD does the email sending itself. To enable ZoRaPD for this task you must specify the necessary parameters: Host address, Port number, Security method, User ID, Password, and Sender email. Use the values obtained by your email service provider.
    In addition, you may specify a signature to be appended to the message text and a vCard to be attached to each email.
