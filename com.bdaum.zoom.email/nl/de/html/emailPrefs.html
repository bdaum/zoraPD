E-Mail Einstellungen

Zum Versenden von E-Mails aus ZoRa PhotoDirector haben Sie zwei Möglichkeiten:

  • Sie verwenden den E-Mail-Client der Plattform. Der Vorteil ist, dass Sie bei der Angabe von Empfängern Zugriff auf Ihr Adressbuch haben. Der Nachteil ist, dass Sie einen zusätzlichen Schritt durchführen müssen. Außerdem ist der Vorgang unter manchen Plattformen etwas umständlich. Unter Windows z.B. generiert ZoRaPD auf Grund technischer Einschränkungen eine E-Mail-Nachricht, die Sie dann explizit an die Empfänger weiterleiten müssen,
  • Sie verwenden nicht den E-Mail-Client der Plattform. In diesem Fall übernimmt ZoRaPD den E-Mail-Versand selbst. Um ZoRaPD für diese Aufgabe zu konfigurieren, müssen Sie die notwendigen Parameter angeben: Host-Adresse, Port-Nummer, Sicherheitsmethode, Benutzer-ID, Passwort und Absender-E-Mail. Verwenden Sie die Werte, die Sie von Ihrem E-Mail-Dienstanbieter erhalten haben.
    Zusätzlich können Sie eine Signatur angeben, die an den Nachrichtentext angehängt werden soll, und eine vCard, die an jede E-Mail angehängt werden soll.
