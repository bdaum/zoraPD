E-Mail-Assistent


Die E-Mail-Funktion wird über die Menüfunktion Datei>Exportieren...>E-Mail>Bilder aufgerufen. Sie wird verwendet, um Bilder an E-Mail-Empfänger zu übermitteln. 

Sie können entweder ausgewählte Bilder oder aber alle Bilder einer Diashow, Ausstellung oder Webgalerie, die Sie im Katalog-Sichtfenster selektiert haben, exportieren.

Sie haben die Wahl, entweder Originale (die sehr groß sein können und vielleicht vom Empfänger nicht zu lesen sind) zu versenden oder die Bilder zu verkleinern und im populären JPEG-Format zu versenden. 

Die Größe der JPEG-Dateien kann mit Hilfe des Schiebereglers eingestellt werden. Alternativ können Sie eine maximale Größe angeben (entweder Breite beiQuerformat oder Höhe bei Hochformat).

Sie können zusätzlich die Ausgabequalität mit diesen Einstellungen beeinflussen.

Die folgenden Optionen sind für die Exportoperation verfügbar
  • Der Export kann aufgezeichnet werden. Später können Sie diese Aufzeichnungen im Sichtfenster Eigenschaften unter ZoRa Photo Director>Track einsehen.
  • Wenn Sie wollen, dass die generierten JPEG-Dateien Metadaten (EXIF und IPTC) enthalten, markieren Sie die Option einschließlich Metadaten. Im folgenden Dialogschritt des Assistenten können Sie dann die Metadaten, die Sie mit versenden möchten, im einzelnen festlegen..
  • Wahlweise können Wasserzeichen in Bilder eingefügt werden, um das intellektuelle Eigentum zu schützen (nicht beim Upload von Originalen).
  • Sie können Bilder ausfiltern und nur Bilder exportieren, die als öffentlich oder öffentlich+medium klassifiziert sind.Diese Optionen werden nur angezeigt, wenn moderate oder private Bilder zu exportieren sind.

Auf der folgenden Assistentenseite können Sie auch einen Betreff und einen Text  hinzufügen.

  • Wenn Sie den E-Mail-Client Ihrer Plattform zum Versenden von Bildern verwenden (siehe E-Mail-Einstellungen), werden die Empfänger im E-Mail-Client der Plattform eingetragen.
  • Wenn nicht, müssen Sie mindestens einen Empfänger angeben. Sie können mehrere Empfänger durch Semikolon getrennt angeben.
