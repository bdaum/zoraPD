HTML-Assistent


Die HTML-Funktion wird über die Menüfunktion Datei>Exportieren ...>Andere>HTML aufgerufen. Sie wird benutzt, um die ausgewählten Bilder in einer HTML-Seite zusammenzufassen. Das kann entweder in einem lokalen Ordner oder einem Ordner auf einem FTP-Konto sein. 

Sie können entweder ausgewählte Bilder oder aber alle Bilder einer Diashow, Ausstellung oder Webgalerie, die Sie im Katalog-Sichtfenster selektiert haben, exportieren.

Sie können das vorgeschlagene Layout und die Bildqualität abändern. Sie können die Ausgabequalität mit diesen Einstellungen beeinflussen.

Die Layoutoptionen entsprechen im wesentlichen denen des Drucken-Dialogs.

Zusätzliche Formatierungen können über das generierte CSS-Stylesheet styles.css erfolgen. Existierende Stylesheets werden nicht überschrieben.
