Configure Search Dialog

This dialog is used to specify the search options for the global search field in the main menu bar. This affects both similarity search and text retrieval. The settings are similar to the Similarity Search Dialog without the possiblity to edit the reference image and the keywords. Instead, these items are taken from the image dropped into the global search field.
 

Search algorithm

Which search algorithm is the best, depends very much on the character of your images. For example, in a catalog containing mostly texture images, you may want to use the Tamura texture filters. With the average photographic content we found the methods CEDD and FCTH (or their combination JCD) quite efficient. The Auto Color Correlogram is a modern search algorithm also providing very good results.

Please note, that you can only use algorithms in a query that are supported by your catalog. Which algorithms are supported can be configured in the catalog properties.

Each query returns with each image a score between 0 and 100 percent. You may limit the amount of returned images either by number or by specifying a minimum score that must be satisfied by each matching image.

Search for

To search only for visual similarity specify a keyword weight of 0, to search only for keywords specify a weight of 100. For a combined search specify any value between 1 and 99.
