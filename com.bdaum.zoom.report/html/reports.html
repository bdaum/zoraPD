Reports

Creating a report in ZoRa PhotoDirector is easy. Just call menu function Catalog>Reports... .
A wizard guides you through the five steps of report creation:

  1. Source selection. You may either start by creating a new report or by selecting an existing report. To select an existing report simply click the Browse... button to display all reports stored in the current catalog. Select one from the list.
    Otherwise, specify a name and a description for the new report. Name and description will later show up as title and subtitle in the generated report.
    The specify the data source for the report. This may be all images in the catalog or images from a specified collection.

  2. Mode selection. First specify one or several values fror the charts y-Axis.  You may select from image count, number of sales and earnings.
    Then specify a value category for the x-Axis. This may be
    • The time of day. This is derived from the IPTC creation data and the EXIF creation date with precendence of the IPTC date over the EXIF date. The value ranges from 0:00 am to 11:59 pm.
    • The combined creation date. This is derived from the IPTC creation data. the EXIF creation date, and the last file modification date with precendence of the IPTC date over the EXIF date, and the EXIF date over the file modification date.
    • Numeric values. When selecting this option you will be able to select a numeric field as an x-Axis value source in the next step. A typical example would be the field "Model age".
    • Discrete values. When selecting this option you will be able to select a non-numeric field as an x-Axis value source in the next step. This can be a field with a fixed value set (enumeration) such as the field "Light source" or a field with an unlimited value range such as "Artist".
    Finally, for discrete values you have the possibilty to specify how to sort the items on the x-Axis: either use the natural order (no sort), or sort by name (value), by image count, by sales, or by earnings.

  3. Value selection. Here you specify a value source for the x-Axis (Numeric and Discrete values only), a value range, and an interval to determine the number values in the generated chart (e.g. number of bars). Images outside the value range will not appear int the report. For discrete values with a fixed value set you have the possibility to exclude certain values from the report. You may also specify a threshold. Items with a y-Axis value below the specified threshold will not appear in the chart but will be folded into a separate item name "Others".

  4. Chart type selection. Depending on the specified report mode you can select between a Pie chart, a Bar chart, a Line chart, a Step chart, and an Area chart. Depending on that choice you may select additional options such as 3D, cylindrical (for Bar charts), and cumulated (for date and time reports). The preview area shows you the effect of these choices. To keep the responses fast, the preview is limited to data from 50 images.

  5. Finished. The report is generated. By right clicking the finished chart you obtain a menu with additional options. You may save the chart as a PNG image file or print it. By selecting Properties you can add further customization to your chart. You may change  titles, labels, colors, fonts and other properties.
    Finally, you have the choice to either store your definitions under the name specified in Step 1 by clicking Finish or to discard them by clicking Cancel.
