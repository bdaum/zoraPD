Berichte

Das Erstellen eines Berichts in ZoRa PhotoDirector ist einfach. Rufen Sie einfach die Menüfunktion Katalog>Berichte... auf. Ein Assistent führt Sie durch die fünf Schritte der Berichtserstellung:

  1. Quellenauswahl. Sie können entweder einen neuen Bericht erstellen oder einen vorhandenen Bericht auswählen. Um einen bestehenden Bericht auszuwählen, klicken Sie einfach auf die Schaltfläche Durchsuchen..., um alle im aktuellen Katalog gespeicherten Berichte anzuzeigen. Wählen Sie einen aus der Liste aus.
    Geben Sie andernfalls einen Namen und eine Beschreibung für den neuen Bericht an. Name und Beschreibung werden später als Titel und Untertitel im generierten Bericht angezeigt.
    Geben Sie die Datenquelle für den Bericht an. Dies können alle Bilder im Katalog oder Bilder aus einer bestimmten Sammlung sein.

  2. Modusauswahl. Geben Sie zunächst einen oder mehrere Werte für die y-Achse des Diagramms an. Sie können aus Bildanzahl, Anzahl der Verkäufe und Einkünfte auswählen.
    Geben Sie dann eine Wertkategorie für die x-Achse an. Möglich sind:
    • Die Tagezeit. Dies wird aus dem IPTC-Erstellungsdatum und dem EXIF-Erstellungsdatum mit Vorrang des IPTC-Datums über das EXIF-Datum abgeleitet. Der Wert reicht von 0:00 Uhr bis 23:59 Uhr.
    • Das kombinierte Erstellungsdatum. Dies wird aus dem IPTC-Erzeugungsdatum, dem EXIF-Erstellungsdatum und dem letzten Dateimodifikationsdatum mit Vorrang des IPTC-Datums über das EXIF-Datum und des EXIF-Datum über das Dateimodifikationsdatum erstellt.
    • Numerische Werte. Wenn Sie diese Option auswählen, können Sie im nächsten Schritt ein numerisches Feld als x-Achsen-Wertquelle auswählen. Ein typisches Beispiel wäre das Feld "Modellalter".
    • Diskrete Werte. Wenn Sie diese Option auswählen, können Sie im nächsten Schritt ein nicht-numerisches Feld als x-Achsen-Wertquelle auswählen. Dabei kann es sich um ein Feld mit einem festen Wert (Enumeration) wie dem Feld "Lichtquelle" oder einem Feld mit einem unbegrenzten Wertebereich wie z.B. "Künstler" handeln.
    Schließlich können Sie für diskrete Werte festlegen, wie die Elemente auf der x-Achse sortiert werden sollen: entweder die natürliche Reihenfolge (keine Sortierung) oder die Sortierung nach Namen (Wert), nach Bildanzahl, nach Verkauf oder Durch Erträge.

  3. Wertauswahl. Hier legen Sie eine Wertequelle für die x-Achse fest (nur numerische und diskrete Werte), einen Wertebereich und ein Intervall zur Bestimmung der Anzahl der Werte im generierten Diagramm (z. B. Anzahl der Balken). Bilder außerhalb des Wertebereichs erscheinen nicht im Bericht. Für diskrete Werte mit einem festen Wertesatz haben Sie die Möglichkeit, bestimmte Werte aus dem Bericht auszuschließen. Sie können auch einen Schwellenwert angeben. Elemente mit einem y-Achsen-Wert unterhalb des angegebenen Schwellwerts werden nicht im Diagramm angezeigt, sondern in einen separaten Elementnamen "Andere" gefaltet.

  4. Diagrammtypauswahl. Je nach dem angegebenen Berichtsmodus können Sie zwischen einem Kreisdiagramm, einem Balkendiagramm, einem Liniendiagramm, einem Stufendiagramm und einem Bereichsdiagramm wählen. Je nach Auswahl können Sie zusätzliche Optionen wie 3D, zylindrisch (für Balkendiagramme) und kumuliert (für Datums- und Zeitberichte) auswählen. Der Vorschaubereich zeigt Ihnen die Auswirkungen dieser Auswahlmöglichkeiten. Um die Antworten schnell zu halten, ist die Vorschau auf Daten von maximal 50 Bildern beschränkt.

  5. Fertig. Der Bericht wird generiert. Durch Rechtsklick auf die fertige Grafik erhalten Sie ein Menü mit zusätzlichen Optionen. Sie können das Diagramm als PNG-Bilddatei speichern oder ausdrucken. Wenn Sie Eigenschaften auswählen, können Sie Ihrem Diagramm weitere Anpassungen hinzufügen. Sie können Titel, Bezeichner, Farben, Schriftarten und andere Eigenschaften ändern.
    Schließlich haben Sie die Wahl, entweder Ihre Definitionen unter dem in Schritt 1 angegebenen Namen zu speichern, indem Sie auf Fertig klicken oder die Änderungen zu verwerfen, indem Sie auf Abbrechen klicken.
