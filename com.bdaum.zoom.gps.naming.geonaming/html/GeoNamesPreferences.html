GeoNames web service

Geonaming is performed with the help of Web services. Under Window>Preferences>GPS>Webservices you can select the geonaming (reverse geocoding) web service of your choice.

The GeoNames web service offers free and premium services. The latter allow you increasing the volume of your queries.

GeoNames.org account parameters

We recommend registering with GeoNames.org for a free or a premium account. After you have registered please enter your user name under Window>Preferences>GPS>Webservices like this:

Example: username=myname
or this
Example: myname

If you have additional parameters separate them with '&' like this:

Example: username=myname&password=mytoken.

GeoNames.org search parameters

GeoNames.org allows you to fine tune your queries. You can specify how the search is performed (within the whole text or only the name, etc.) and for what location types the search should be performed.
