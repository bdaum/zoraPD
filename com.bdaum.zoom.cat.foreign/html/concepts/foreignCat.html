Importing foreign catalogs

The items contained in some foreign catalogs from Raw Developers such as Adobe Lightroom, Corel AfterShot (Bibble), or Capture One (v4-v6) from Phase One can be imported into the current catalog via Catalog>Import...>xxxx.

There are some settings in both Raw Developer and ZoRa PhotoDirector that influence the import process:
  • Make sure that the Raw Developer externalizes image metadata in some way. 
    • In Lightroom make sure that the option Store changes automatically in XMP is enabled in the Lightroom Catalog Settings.
    • In Bibble/AfterShot set the option Allow Bibble/AfterShot .xmp creation... .
    • No special settings are necessary for Capture One.
  • No conversion to DNG is performed independent of the settings under Window>Preferences>Import>DNG Conversion*.
  • Raw Recipe Reflection is only performed if the option Process recipes when importing RAW images under Window>Preferences>Import>Raw Conversion* is checked. The settings under Active Recipe Detectors are ignored. Instead, the best matching recipe detector is used during the import process. 

*) On Mac use ZoRa Photo Director>Preferences>...
