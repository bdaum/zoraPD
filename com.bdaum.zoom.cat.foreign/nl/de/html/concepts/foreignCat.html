Fremde Kataloge importieren

Die Elemente fremder Kataloge von Raw-Entwicklungsprogrammen wie Adobe Lightroom, Corel AfterShot (Bibble) oder Capture One (v4-v6) von Phase One können in den aktuellen Katalog mittels Katalog>Import...>xxxx importiert werden.

Dabei beeinflussen sowohl die Einstellungen des fremden Programms als auch die Einstellungen von ZoRa PhotoDirector den Importprozess:
  • Das fremde Programm sollte die Metadaten externalisieren: 
    • In Lightroom setzen Sie die Option Store changes automatically in XMP in den Lightroom Catalog Settings.
    • In Bibble/AfterShot setzen Sie die Option Allow BibbleAfterShot .xmp creation... .
    • Für Capture One sind keine speziellen Einstellungen erforderlich.
  • Beim Import von fremden Katalogen werden grundsätzlich keine Bilder in das DNG-Format konvertiert, unabhängig von den Einstellungen unter Fenster>Benutzervorgaben>Import>DNG-Konvertierung*.
  • Raw Recipe Reflection wird nur durchgeführt, wenn die Option Entwicklungsrezepte beim Import von RAW-Bildern berücksichtigen unter Fenster>Benutzervorgaben>Import>Raw-Konvertierung* markiert ist. Die Einstellungen unter Aktive Rezept-Detektoren werden ignoriert. Stattdessen wird der jeweils am besten passende Rezeptdetektor während des Importprozesses verwendet. 

*) Auf dem Mac: ZoRa Photo Director>Einstellungen>...
