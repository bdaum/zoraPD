Microsoft Computer Vision

At the time of writing the Microsoft Computer Vision was still in an early state. Nevertheless it was useful. In addition in providing categories and keywords, it can also check for safe content, detect faces, and provide image description one-liners.
Microsoft offers several pricing plans, starting with a free plan. At the time of writing the free plan allowed for 5000 image requests per month. Microsoft also limits the number of requests to 20 per minute. Therefore, ZoRa PhotoDirector waits three seconds until issueing the next request.

Before you can use the service you must open an account with Microsoft Cognitive Services To do so, visit their website at https://www.microsoft.com/cognitive-services or click the link in the Microsoft Vision preference page. You must then enter your credentials (Key1 or Key2) into the Microsoft Vision preference page.

Microsoft Vision supports at the time of writing only English and Chinese. We suggest to use English and to use the translation options below if you require a different output language.

Options

  • Maximum number of proposals. For each image Microsoft Computer Vision proposes a small number of categories and a larger number of keywords. You may restrict both by setting a maximum number of proposals.
  • Minimum confidence level. All proposals are given with a confidence level. You may enter a value between 0 and 100 (in percent) to supress all proposals with a lower confidence level. Note that this setting applies only to keywords but not to categories.
  • Pre-mark keywords when confidence is above. Specify a confidence level for which the system should automatically mark proposals as keywords. Specify a value of 100 if you do not wish automatical marking.
  • Pre-mark only known keywords. This option will restricts automatical marking to keywords that are already defined in the catalog.
  • Check "safe for work". Mark this option to check if an image contains sexual or racist contents. The results will be used to set the images privacy settings.
  • Detect faces. Mark this option if you want the service to detect faces in the image. If faces are found bounding boxes in green color will be drawn in the image. You have then the option to accept those face regions or to dicard them.
  • Detect celebrities. Mark this option if you want the service to detect celebrities in the image. At the time of writing the celebrity names appear only in the generated image description (below).
  • Generate description. An image description one-liner is generated. Please note that we skip the generation of image description if an image has already a description.
  • Translate categories. Mark this option if you want the outputted categories in another language than English. The Microsoft Translator Service will be used to translate the returned texts.
  • Translate keywords. Mark this option if you want the outputted keywords in another language than English. 
  • Translate description. Mark this option if you want the outputted description in another language than English.
