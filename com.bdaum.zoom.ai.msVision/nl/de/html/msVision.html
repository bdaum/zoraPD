Microsoft Computer Vision

Über die Bereitstellung von Kategorien und Schlagworten hinaus, kann Microsoft Azure Computer Vision Bilder auch auf sichere Inhalte prüfen, Gesichter, Prominente und Sehenswürdigkeiten erkennen und Bildbeschreibungs-Einzeiler liefern.
Microsoft bietet mehrere Preispläne an, beginnend mit einem freien Plan. Zum Zeitpunkt des Schreibens erlaubte der freie Plan 5000 Bildanfragen pro Monat. Microsoft begrenzt außerdem die Zahl der Anfragen auf 20 pro Minute. Deshalb wartet ZoRa PhotoDirector drei Sekunden, bevor es die nächste Anfrage sendet.
Bevor Sie den Dienst verwenden können, müssen Sie ein Konto mit Microsoft Cognitive Services eröffnen. Um dies zu tun, besuchen Sie die Website unter https://azure.microsoft.com/de-de/services/cognitive-services/ oder klicken Sie auf den Link in der Microsoft-Vision-Seite der Benutzereinstellungen. Sie müssen dann Ihre Anmeldeinformationen (Key1 oder Key2 und Endpunkt) in die Microsoft-Vision-Seite der Benutzereinstellungen eingeben.

Microsoft Computer Vision unterstützt zum Zeitpunkt des Schreibens nur Englisch, Spanisch, Chinesisch und einige wenige andere Sprachen als Ausgabesprachen. Wir empfehlen, Englisch zu benutzen und die Übersetzungsoptionen unten anzuwenden, wenn Sie eine andere Ausgabesprache wünschen.

Optionen


  • Maximale Anzahl von Vorschlägen. Für jedes Bild schlägt Microsoft Computer Vision eine kleine Anzahl von Kategorien und eine größere Anzahl von Schlagworten vor. Sie können beides beschränken, indem Sie eine maximale Anzahl von Vorschlägen festlegen.
  • Min. Konfidenzniveau. Alle Vorschläge werden mit einem Vertrauensniveau angegeben. Sie können einen Wert zwischen 0 und 100 (in Prozent) eingeben, um alle Vorschläge mit einem niedrigeren Konfidenzniveau zu unterdrücken. Beachten Sie, dass diese Einstellung nur für Schlagworte gilt, aber nicht für Kategorien.
  • Schlagworte vormarkieren bei Konfidenzniveau über. Geben Sie ein Konfidenzniveau an, für das das System automatisch Vorschläge als Schlagworte markieren soll. Geben Sie einen Wert von 100 an, wenn Sie keine automatische Markierung wünschen.
  • Nur bekannte Schlagworte vormarkieren. Diese Option beschränkt die automatische Kennzeichnung auf Schlüsselwörter, die bereits im Katalog definiert sind.
  • Auf sichere Inhalte überprüfen. Markieren Sie diese Option, um eine Prüfung auf  sexuelle oder rassistische BildInhalte zu veranlassen. Die Ergebnisse werden verwendet, um die Datenschutzeinstellungen des Bildes zu setzen.
  • Gesichter erkennen. Markieren Sie diese Option, um eine Gesichtserkennung durchzuführen. Werden Gesichter erkannt, werden diese durch grüne Rahmen im Bild gekennzeichnet. Sie haben dann die Möglichkeit, diese Gesichtsregionen zu akzeptieren oder zu verwerfen.
  • Promis und Sehenswürdigkeiten entdecken. Markieren Sie diese Option, wenn Sie den Dienst zur Erkennung von Prominenten im Bild verwenden möchten. Zum Zeitpunkt des Schreibens erscheinen die Namen der Prominenten nur inder generierten Bildbeschreibung (siehe unten).
  • Beschreibung generieren. Ein Bildbeschreibungs-Einzeiler wird erzeugt. Bitte beachten Sie, dass die Erzeugung der Bildbeschreibung übersprungen wird, wenn ein Bild bereits eine Beschreibung hat.
  • Kategorien übersetzen. Markieren Sie diese Option, wenn die ausgegebenen Kategorien in einer anderen Sprache als Englisch angezeigt werden sollen. Der Microsoft Translator Service wird verwendet, um die zurückgegebenen Texte zu übersetzen.
  • Schlagworte übersetzen. Markieren Sie diese Option, wenn die ausgegebenen Schlagworte in einer anderen Sprache als Englisch angezeigt werden sollen.
  • Beschreibung übersetzen. Markieren Sie diese Option, wenn die ausgegebene Beschreibung in einer anderen Sprache als Englisch angezeigt werden soll.
