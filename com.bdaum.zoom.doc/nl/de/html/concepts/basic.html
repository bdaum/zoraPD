Grundlegende Konzepte

  • Bilder werden durch ZoRa Photo Director nie geändert. Bei der bildverändernden Operation wie z.B. der Rotation oder der Berücksichtung von RAW-Entwicklungsrezepten werden auschließlich die Vorschaubilder im Katalog und die Anzeige eines Bildes im Bildbetrachter geändert, nie jedoch das Bild in seiner gespeicherten Form.
Ausnahme: Optional können XMP-Metadaten auch in originale JPEG-Dateien exportiert werden. Siehe (Fenster>Benutzervorgaben>Metadaten>Export*)
  • Keine Bildbearbeitung. ZoRa Photo Director bietet keine Möglichkeiten für die Bildverarbeitung an. Statt dessen verwenden Sie bitte Ihren bevorzugten Bild-Editor, den Sie leicht in das System einbinden können (Fenster>Benutzervorgaben>Datei-Zuordnungen*). Dabei kann je nach Dateikennung ein anderer Editor verwendet werden.
  • Metadaten werden ausschließlich in Form von beigeordeneten XMP-Dateien (sidecar) exportiert. Sie haben die Option vor dem Aufruf eines externen Editors die Metadaten automatisch exportieren zu lassen  (Fenster>Benutzervorgaben>Allgemein*), so dass der externe Editor alle Änderungen, die Sie an den Metadaten angebracht haben, erhält (sofern der externe Editor XMP unterstützt). Die Ausnahme sind von ZoRa Photo Director generierte JPEG-Bilder für den Export. In diesem Fällen werden die ausgewählten Metadaten in die JPEG-Datei hineingeschrieben.

*) Auf dem Mac: ZoRa Photo Director>Einstellungen>...
