Farbverwaltung

  • Beim Import werden Vorschau-Bilder vom ursprünglichen Farbraum in den sRGB-Farbraum konvertiert.. 
  • Das Anzeigen von Bildern kann über Fenster>Benutzervorgaben>Allgemein>Ausgabe-ICC-Profile* kontrolliert werden. 
    • CMS deaktivieren. Dies ist die schnellste Betriebsweise, kann aber dazu führen, dass falsche Farben dargestellt werden.
    • sRGB. Verwenden Sie diese Einstellung für Standard-Bildschirme (Standard) 
    • Adobe RGB. Verwenden Sie diese Einstellung bei der Arbeit mit Wide-Gamut-Monitoren.
    • Rec. 709, Rec. 2020, DCI-P3, P3-D60, P3-65. Verwenden Sie diese Einstellungen bei der Arbeit mit Cine-Monitoren.
  • Bilder, die für E-Mail oder Web exportiert werden, werden immer in den sRGB-Farbraum konvertiert.
  • Auch das Drucken von Miniaturansichten erfolgt im sRGB-Farbraum.

*) Auf dem Mac: ZoRa Photo Director>Einstellungen>...
