Häufig gestellte Fragen (FAQ)


Frage: Wann greift ZoRa Photo Director auf das Internet zu und welche Informationen werden dabei übermittelt?

Antwort: ZoRa Photo Director versucht, die Interaktion mit dem Internet auf ein Minimum zu reduzieren. Auf das Internet wird nur in den folgenden Fällen zugegriffen:
  • Anzeige von Karten (über Google Maps) 
  • Hinzufügen von geographischen Namen zu geografischen Positionen (über Geonames.org)
  • Hochladen von Bildern zu einem FTP-Konto
  • Hochladen von Bildern zu einer Webgemeinschaft
  • Anmelden an eine Webgemeinschaft
  • Aktualierung der Kontoinformation von Webgemeinschaften (Alben, Kategorien usw.)
  • Import von Kategorien aus einer Webgemeinschaft
  • Prüfen auf eine neue Programmversion (auf Verlangen oder automatisch)
