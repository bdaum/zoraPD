Erstellen eines neuen Katalogs

Beim ersten Start der Anwendung oder wenn ein zuvor verwendeter Katalog nicht mehr existiert, werden Sie aufgefordert, den Speicherort eines vorhandenen Katalog einzugeben oder einen neuen Katalog am gewählten Speicherort anzulegen. Beim Neuanlegen eiens Katalogs werden Sie zur Eingabe der Katalogeigenschaften aufgefordert. Zu diesem Zeitpunkt sollten Sie die folgenden Einträge noch vor dem Importieren von Bildern überprüfen bzw. vervollständigen:

  • Sicherung (Kopfbereich)
    • Sicherungsdatei
  • Allgemeines
    • Informationen zum Eigentümer
    • Automatisches Erstellen einer Zeitlinie
    • Definition von Benutzerfeldern
  • Miniaturansichten
    • Auflösung
    • Miniaturansichten aus Vorschau-Bildern ableiten
    • Schärfen von Miniaturansichten

Eine ausführliche Erläuterung finden Sie unter Katalogeigenschaften.

Diese Katalogeigenschaften können Sie später jederzeit mit Katalog>Eigenschaften oder Alt+Eingabe aufrufen.
