Vorlagendefinition

Benutzerdefinierte Vorlagen erhalten einen Namen, der im ersten Feld eingetragen werden muss.

Im zweiten Feld erfolgt der Aufbau des Vorlageninhaltes. Konstante Texte werden direkt in das Feld eingetippt. Auch Variablen können eingetippt werden, es ist jedoch komfortabler und sicherer, den Taste Variable einfügen... zu betätigen und die Variable aus einer Liste auszuwählen.

Im dritten Feld wird das Ergebnis der eingegeben Vorlage als Anschauungsmaterial angezeigt. Die Werte basieren auf dem ersten Bild einer vorhandenen Bildauswahl.

Sind die Vorlagen für das explizite Umbenennen (massenweises Umbenennen) vorgesehen, so besteht auch die Möglichkeit, neben den vordefinierten Variablen auch fast jedes Feld der Bild-Metadaten als Variable in die Vorlage einzufügen. Dazu betätigen Sie die Taste Metadaten einfügen...
