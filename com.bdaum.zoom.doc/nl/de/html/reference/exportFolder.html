In Ordner exportieren (Assistent)


Die Funktion In Ordner exportieren wird über die Menüfunktion Datei>Exportieren...>Andere>Lokale Ordner oder FTP aufgerufen. Sie wird verwendet, um die ausgewählten Bilder in den gewählten Ordner zu kopieren. Das kann entweder ein lokaler Ordner oder ein Ordner auf einem FTP-Konto sein. 

Sie können entweder ausgewählte Bilder einer Galerie oder aber alle Bilder einer Diashow, Ausstellung oder Webgalerie, die Sie im Katalog-Sichtfenster selektiert haben, exportieren.

Sie haben die Wahl, entweder Originale (die sehr groß sein können) zu verwenden oder die Bilder zu verkleinern und in das populäre JPEG-Format zu komprimieren. 

Die Größe der JPEG-Dateien kann mit Hilfe des Schiebereglers eingestellt werden. Alternativ können Sie eine maximale Größe angeben (die Breite bei Querformat oder die Höhe bei Hochformat).

Sie können die Ausgabequalität mit diesen Einstellungen beeinflussen.

Die folgenden Optionen sind für die Exportoperation verfügbar:

  • Wasserzeichen in Bilder einfügen, um das intellektuelle Eigentum zu schützen (nicht möglich beim Export von Originalen).
  • Wenn Sie wollen, dass die generierten JPEG-Dateien Metadaten (EXIF und IPTC) enthalten, markieren Sie die Option "einschließlich Metadaten". Im folgenden Dialogschritt des Assistenten können Sie dann die Metadaten, die Sie exportieren möchten, im Einzelnen festlegen..
  • Sie können Bilder ausfiltern und nur Bilder exportieren, die als öffentlich oder öffentlich+medium klassifiziert sind.

Wahlweise kann das exportierte Bild ebenfalls in den Katalog aufgenommen werden. In diesem Fall wird automatisch eine Beziehung "Original->Derivat" erstellt.
