Aktualisieren-Dialog

Die Menüfunktion Datei>aktualisieren... re-importiert ausgewählte Dateien. Im Aktualisieren-Dialog können Sie festlegen, was aktualisiert werden soll.

  • Vorhandene Bilder erneut importieren. Damit können Sie Bilder aktualisieren, die bereits im Katalog vorhanden sind. Normalerweise wird diese Option verwendet, um die Katalogeinträge von Bildern zu aktualisieren, die von anderen Programmen verändert worden sind. Eine andere Möglichkeit, das zu erreichen, ist die Verwendung von Überwachten Ordnern.
  • Auch nicht veraltete Bilder aktualisieren. Normalerweise wird diese Option nicht verwendet. Sie sollte nur dann eingeschaltet werden, wenn eine neue Programmversion installiert wurde, welche neue oder korrigierte Bildattribute eingeführt, oder wenn die Größe oder das Schärfen der Miniaturansichten in den Katalogeigenschaften verändert wurde.
  • Fehlende Bilder aus dem Katalog entfernen. Damit können Katalogeinträge entfernt werden, für die keine Bilder mehr existieren (Waisen).
  • Nicht-lokale Bilder ebenfalls aktualisieren. Mit dieser Option können Sie auch Bilder aktualisieren, die auf einem Internet-Server gespeichert sind. Die Aktualisierungsoperation wird allerdings beträchtlich verlangsamt, da die Bilder zunächst auf den lokalen Computer heruntergeladen werden müssen.

Zusätzlich können Sie festlegen, welche Informationensgruppen aktualisiert werden sollen.So wird man normalerweise keine Informationsgruppen aktualisieren, die man vorher manuell bearbeitet hatte.

  • Miniaturansicht aktualisieren. Aktivieren Sie diese Option, um die Miniaturansicht im Katalog zu aktualisieren - besonders nachdem Sie ein Bild mit einem Bildbearbeitungsprogramm modifiziert hatten oder wenn die Größe oder das Schärfen der Miniaturansichten in den Katalogeigenschaften verändert wurde. Ohne diese Option verläuft der Aktualisierungsvorgang allerdings erheblich schneller.
  • Bild-Status aktualisieren. Deaktivieren Sie diese Gruppe, wenn Sie manuell Status, Farbcode, Bewertung, etc. geändert haben und Sie nicht wollen, dass diese Informationen zurückgesetzt werden.
  • EXIF-Daten aktualisieren. Deaktivieren Sie diese Gruppe, wenn sich die EXIF-Daten der Bilder nicht verändert hatten. Beispielsweise möchten Sie nur die GPS-Daten aktualisieren.
  • GPS-Daten aktualisieren. Deaktivieren Sie diese Gruppe, wenn Sie das Bild mit Geotags versehen habem und Sie nicht wollen, dass diese Informationen zurückgesetzt werden.
  • IPTC-Daten aktualisieren. Deaktivieren Sie diese Gruppe, wenn Sie manuell Schlagworte, Kategorien und andere IPTC-Daten eingegeben haben und Sie nicht wollen, dass diese Informationen zurückgesetzt werden.
