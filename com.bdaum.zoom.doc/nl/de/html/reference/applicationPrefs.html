Anwendung

    • Die Gruppe Vollbild erlaubt die Konfiguration des Vollbildmodus: ob die Menüleiste und/oder die Randleisten im Vollbildmodus gezeigt werden sollen.
    • Die Option Programm im Taskleistenmodus ausführen bewirkt, dass die Anwendung beim Start zunächst in die Taskleiste minimiert wird. Ein Doppelklick auf das Icon in der Taskleiste macht die Anwendung sichtbar. Ein Klick auf den Schließknopf beendet die Anwendung nicht, sondern minimiert sie wieder in die Taskleiste.
    • Die Option Den Mülleimer leeren, wenn das Programm in die Taskleiste geschickt wird  ist nur aktiv wenn das Programm im Taskleistenmodus ausgeführt wird und tut das nämliche. Andernfalls wird der Mülleimer nur geleert, wenn das Programm beendet wird.
