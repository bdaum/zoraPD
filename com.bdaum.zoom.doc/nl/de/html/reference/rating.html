Automatische Bildbewertung

Die automatische Bildbewertung wird mit Hilfe von internetbasierten Bewertungsdiensten durchgeführt. Derzeit wird nur Clarifai als Rating-Service unterstützt. Clarifai bietet mehrere so genannte Modelle wie Landschaft, Portrait oder Fokus. Bilder werden immer in Bezug auf ein solches Modell bewertet.

Das Ergebnis einer Bildbewertung ist ein Wert zwischen 0 und dem maximalen Bewertungswert. Wir empfehlen als maximalen Bewertungswert den Wert 3. So kann der automatische Bewertungsprozess die Werte 0,1,2 und 3 besetzen, während die Werte 4 und 5 nach der visuellen Inspektion manuell angewendet werden können.
Sie können sich dafür entscheiden, alle ausgewählten Bilder zu bewerten oder nur diejenigen, die noch nicht bewertet wurden.
