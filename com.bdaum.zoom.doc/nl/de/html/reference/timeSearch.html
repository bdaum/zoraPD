Zeitsuche

Die Zeitsuche kann Bilder finden, die in der gleichen Zeitspanne wie die selektierten Bilder aufgenommen wurden.

  • Die Gruppe Basis-Intervall wird normalerweise aus den selektierten Bildern abgeleitet. Wurde nur ein Bild selektiert, sind die Einträge unter von und bis gleich. Wurde kein Bild selektiert, ist dort das aktuelle Datum eingetragen. Sie können diese Einträge nach Belieben modifizieren.
  • Die Gruppe Abgeleitetes Intervall bestimmt, wie das Basis-Intervall erweitert werden soll.
