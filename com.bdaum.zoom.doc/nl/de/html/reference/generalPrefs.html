Allgemeine Einstellungen

  • Anzeige
    • Ausgabe-ICC-Profil. Bei der Verwendung von Standard-Monitoren sollten Sie diese Einstellung auf sRGB lassen. Falls Sie einen Wide-Gamut-Monitor verwenden, ändern Sie diesen Eintrag auf Adobe RGB ab. Andernfalls könnten Farben übertrieben dargestellt werden. Wenn es auf korrekte Farben nicht ankommt, wechseln Sie zu CMS deaktivieren, womit die Darstellung der Bilder beschleunigt wird. Siehe auch Farbverwaltung.
    • Hochwertige Interpolation verwenden. Aktivieren Sie diese Option, um eine qualitativ hochwertige Interpolation für das Anzeigen von hochauflösenden Bildern zu verwenden. Andernfalls wird der schnellere Nächste-Nachbar-Algorithmus verwendet, der jedoch zur Pixelierung des Bildes führen kann. Je nach verfügbarem Grafikspeicher  könnten sehr große Bilder überhaupt nicht angezeigt werdem, wenn die hochwertige Interpolation verwendet wird. In diesem Fall wechseln Sie bitte zur normalen Interpolation.
    • Diashow Inaktivitäts-Zeitüberschreitung (min). Hier können Sie eine maximale Zeitspanne für Pausen in einer Diashow angeben. Nach Ablauf der spezifizierten Zeitspanne wird die Diaschau abgebrochen.
  • Interne Betrachter
    • Im internen Betrachter wenn möglich vor jedem hochauflösenden Bild ein Vorschaubild zeigen.  Ist diese Option eingeschaltet, wird ein qualitativ gutes Bild (das Vorschaubild) relativ früh sichtbar, bis zur Anzeige des hochauflösenden Bildes dauert es jedoch wegen der zusätzlichen Verarbeitung etwas länger. Außerdem kann das Vorschaubild in Farbe und Helligkeit stark von der hochauflösenden Ausgabe abweichen.
    • JPeG-Artefakte der Miniaturansichten im internen Bildbetrachter kaschieren. Wenn diese Option eingeschaltet ist, wird den vergrößerten Vorschaubildchen im internen Bildbetrachter Rauschen zugefügt, um die häßlichen JPeG-Artefakte zu kaschieren.
    • Kleine Bilder auf Vollbild vergrößern. Markieren Sie diese Option, wenn Sie kleine Bilder in größerem Format ansehen möchten. Die Qualität könnte dabei allerdings schlecht sein.
    • Sekundären Monitor für Bildbetrachter verwenden. Diese Option kann auf Dual-Monitor-Systemen gewählt werden, wenn die Workbench auf dem primären Monitor läuft. Der interne Bildbetrachter öffnet sich dann  abhängig von Ihrer Auswahl auf dem primären Monitor, dem sekundären Monitor oder abwechselnd auf dem primären und sekundären Monitor.
  • Sicherung und Programmaktualisierung
    • Sicherungsintervall. Geben Sie die Anzahl der Tage zwischen jeder Sicherung an. Die Sicherungen werden automatisch durchgeführt. Der Speicherort für die Sicherungen kann für jeden Katalog separat in den  Katalogeigenschaften angegeben werden.
    • Maximale Sicherungsgenerationen. Geben Sie hier an, wieviele Sicherungsgenerationen Sie behalten möchten. Ältere Sicherungsgenerationen werden nach jeder Sicherung gelöscht.
    • Auf neue Programmversion prüfen. Bestimmt, wann auf eine neue Programmversion geprüft werden soll. Das kann rein manuell erfolgen (Hilfe>Auf neue Programmversion prüfen) oder automatisch bei jeder regulären Katalalogsicherung oder bei jedem Programmstart.
  • Verschiedenes
    • max. Verwerfen-Schritte. Bestimt, wieviele Schritte rückgängig gemacht werden können. Ein hoher Wert braucht mehr Speicherplatz. Der Standardwert ist 9.
    • Statusanzeige für Hintergrundaktivitäten unterdrücken. Markieren Sie diese Option, um Hintergrundaktivitäten unsichtbar zu machen.
