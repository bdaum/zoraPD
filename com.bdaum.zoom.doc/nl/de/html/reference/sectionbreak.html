Abschnittstiteldia

Der Abschnittswechsel-Dialog ermöglicht es Ihnen, mehrere Eigenschaften für einzelne Abschnittswechsel, auch Titelfolien genannt, zu bearbeiten oder zu definieren. Es beschreibt typischerweise den folgenden Abschnitt von DIas bis zum nächsten Abschnittswechsel oder dem Ende der Diashow. Die folgenden Eigenschaften können über die standardmäßigen Diaeigenschaften hinaus definiert werden:

  •     Titel. Ein Titel für den folgenden Abschnitt.
  •     Beschreibung. Text, der den folgenden Abschnitt beschreibt.
  •     Layout:
    •         Nur Text. Es werden nur der Titel und die Beschreibung angezeigt.
    •         Miniaturansichten links/rechts/oben/unten. Dies zeigt eine Übersicht über die folgenden Dias innerhalb der Titelfolie.
    •         Karte links/rechts. Dies zeigt eine Karte der Bildstandorte der folgenden Dias, sofern diese Bilder geotagged sind. Diese Funktion ist nur verfügbar, wenn die Geokomponente installiert wurde. Beachten Sie, dass für die Wiedergabe der Diashow eine Internetverbindung erforderlich ist.


Der Dialog enthält auch ein Feld Zusammenstellen bis. Wählen Sie eine Anzahl von Dias aus, die analysiert werden sollen. Informationen wie der Standort, das Erstellungsdatum, Modellinformationen und gezeigte Kunstwerke oder Objekte werden gesammelt und in das Feld Beschreibung gestellt. Zusätzlich wird ein Layout vorgeschlagen. Sie sollten diese Funktion vor der Eingabe anderer Informationen verwenden, da das Feld Beschreibung und das Feld Layout überschrieben werden.
