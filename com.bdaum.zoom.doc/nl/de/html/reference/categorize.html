Kategorisieren

Das Kategorisieren von Bildern wird durch den IPTC-Standard geregelt. Dieser Standard ermöglicht es, für ein Bild höchstens eine Hauptkategorie aber außerdem eine beliebige Anzahl von zusätzlichen Kategorien zu definieren.

Die Kategorien werden in Zora PhotoDirector durch die Katalogeigenschaften definiert. Diese Definitionen resultieren in Kategoriekollektionen im Katalog-Sichtfenster. Jede dieser Sammlungen zeigt Bilder mit den jeweiligen Hauptkategorien.

Der Kategorisieren-Befehl ermöglicht eine schnelle Kategorisierung einer großen Anzahl von Bildern. Zunächst wählen Sie einige Bilder aus, dann rufen Sie die Kontextfunktion  Kategorisieren ... auf. Im Anschluss sehen Sie im Dialog das aktuelle Bild auf der linken und den Kategoriebaum in der Mitte. Sie können nun Haupt- und Zusatzkategorien mit einem Klick auf Baumknoten zuweisen. Baumknoten können fünf Zustände haben:

Auf der rechten Seite sehen Sie eine Liste der automatisch abgeleiteten Kategorievorschläge mit ihren Vertrauensnoten. Passende Kategorien sind bereits als Vorschläge im Kategorienbaum in der Mitte markiert. Nicht passende Vorschläge können Sie entweder auf eine Kategorie ziehen, um ein Synonym zu dieser Kategorie hinzuzufügen, oder mit Strg ziehen, um den Vorschlag als Kategorie oder Unterkategorie hinzuzufügen.
Alternativ können Sie einen Vorschlag in der Vorschlagsliste markieren, um ihn zum aktuellen Bild als Schlagwort hinzuzufügen.
Dieser Bereich ist nur sichtbar, wenn die KI-Komponente installiert ist.

Unter dem Kategorienbaum finden Sie den Verweis Konfigurieren.... Dadurch werden die Katalogeigenschaften zum Konfigurieren der Kategorien aufgerufen. Auf der rechten Seite wird ein Accept-Verweis angezeigt, wenn Kategorien als Vorschläge markiert sind. Wenn Sie auf diesen Link klicken, werden alle Vorschläge im Baum in primäre oder zusätzliche Kategorien umgewandelt.
Ein anderer Konfigurieren...-Verweis befindet sich unterhalb der Vorschlagsliste. Dies ruft die Benutzereinstellungen für die Konfiguration der AI-Dienste auf.

Zusätzlich zur Definition von Kategorien und Schlagworten können Sie auch Bildbeschreibungen eingaben und den Datenschutz festlegen.

Die Schaltflächen in der Schaltflächenleiste ermöglichen das Navigieren zwischen Bildern:

  • Zurück. Zurück zum vorherigen Bild. Alternativ können Sie die CURSOR-LINKS-Taste drücken.
  • Weiter. Weiter zum nächsten Bild. Alternativ können Sie die CURSOR-RECHTS-Taste drücken.
  • Klonen. Drücken Sie diese Taste, um einer  Reihe von Bildern schnell die gleichen Kategorien zuzuweisen. Es wird die Kategoriedefinitionen aus dem vorherigen Bild kopiert, dann auf das aktuelle Bild angewandt, dann wird zum nächsten Bild weitergeschaltet. Alternativ können Sie die STRG + CURSOR-RECHTS-Tastenkombination drücken.
  • Fertig. Alle Änderungen werden in den Katalog gespeichert und der Dialog wird geschlossen. (Sie können später noch die gesamte Operation verwerfen).
  • Abbrechen. Alle Änderungen werden verworfen und der Dialog wird geschlossen.
