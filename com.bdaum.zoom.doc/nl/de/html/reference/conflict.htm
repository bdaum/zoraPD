Aktualisieren-Dialog

Beim Importieren von Bildern kann es zu Konflikten kommen, wenn die gleichen Bilder schon vorher in den Katalog importiert werden. Es gibt verschiedene Methoden, solche Konflikte aufzulösen:

  • Bild überspringen. Das Bild wird nicht neu importiert, der Importvorgang wird mit dem nächsten Bild fortgesetzt..
  • Katalogeintrag überschreiben.  Der vorhandene Katalogeintrag wird komplett gelöscht und das Bild neu importiert..
  • Mit Katalogeintrag synchronisieren. Der vorhandene Katalogeintrag wird nur in Teilen mit den neuen Daten überschrieben. Welche Teile das sind, wird mit folgenden Optionen festgelegt:
    • Miniaturansicht aktualisieren. Aktivieren Sie diese Option, um die Miniaturansicht im Katalog zu aktualisieren - besonders nachdem Sie ein Bild mit einem Bildbearbeitungsprogramm modifiziert hatten oder wenn die Größe oder das Schärfen der Miniaturansichten in den Katalogeigenschaften verändert wurde. Ohne diese Option verläuft der Aktualisierungsvorgang allerdings erheblich schneller.
    • Bild-Status aktualisieren. Deaktivieren Sie diese Gruppe, wenn Sie manuell Status, Farbcode, Bewertung, etc. geändert haben und Sie nicht wollen, dass diese Informationen zurückgesetzt werden.
    • EXIF-Daten aktualisieren. Deaktivieren Sie diese Gruppe, wenn sich die EXIF-Daten der Bilder nicht verändert hatten. Beispielsweise möchten Sie nur die GPS-Daten aktualisieren.
    • GPS-Daten aktualisieren. Deaktivieren Sie diese Gruppe, wenn Sie das Bild mit Geotags versehen habem und Sie nicht wollen, dass diese Informationen zurückgesetzt werden.
    • IPTC-Daten aktualisieren. Deaktivieren Sie diese Gruppe, wenn Sie manuell Schlagworte, Kategorien und andere IPTC-Daten eingegeben haben und Sie nicht wollen, dass diese Informationen zurückgesetzt werden.
Sie können jeden dieser Konflikte Schritt für Schritt lösen, oder aber die ausgewählte Methode auf alle folgenden Konflikte innerhalb des Importvorgangs anwenden, indem Sie die Option Auf alle Konflikte anwenden markieren. Zusätzlich können Sie Nur neuere Elemente markieren, um nur solche Objekte zu importieren, die nach dem vorherigen Import noch modifiziert wurden.
