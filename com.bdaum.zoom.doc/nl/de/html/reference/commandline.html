Kommandozeilenparameter

Kommandozeilen-Parameter können verwendet werden, um das Verhalten des ZoRa Photo Director zu ändern und auf an spezifischen Bedürfnisse anzupassen.

Parameter Beschreibung Beispiele
-data Bestimmt die Lage der Speicherbereichs, in dem die Benutzer-Einstellungen gespeichert sind. Standardmäßig werden diese Werte in Unterordner workspace/ im Installationsordner abgelegt. Durch die Angabe eines anderen Weg hinter -data kann dieser Bereich an eine beliebige Stelle gelegt werden. Es gibt zwei vordefinierte Pfad-Abkürzungen:

  • @user.home steht für den Benutzerbereich des Betriebssystems wie C:\Dokumente und Einstellungen\benutzername. 
  • @user.dir steht für den Installationsordner wie C:\Programme\ZoRaPD. 
 -data @user.home/ZoRaPD
-nl Bestimmt die Sprache der Benutzeroberfläche. Geben Sie ein ISO-Sprachcode nach dem Parameter an.
Wenn nicht angegeben, wird die Sprache auf Grundlage der Spracheinstellung der Plattform gewählt.
(Derzeit werden nur Englisch und Deutsch unterstützt.)		 -nl en
-nl de
-noSplash ZoRa Photo Director wird ohne Anzeige des Splash-Schirms gestartet. -noSplash
-noBirdie Verhindert das gelegentlich erscheinende Vögelchen im Splash-Schirm
 -noBirdie
-vmPfad zur Java Virtual Machine (JVM). Dieser Parameter benutzt, wenn das Programm mit einer anderen als der Standard-JVM der Plattform verwendet werden soll.-vm C:\jre_pfad\bin\java.exe
