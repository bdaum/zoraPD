Massenweises Umbenennen

Diese Funktion benennt alle selektierten Bilder mit Hilfe einer ausgewählten Vorlage neu.
  • Wählen Sie zunächst welche Elemente eines Bildes verändert werden sollen:
    • Sowohl Datei- als auch Bildname (Standard)
    • Nur der Bildtitel (IPTC)
  • Dann wählen Sie eine Vorlage von der Liste. Es gibt mehrere vordefinierte Vorlagen, Sie können jedoch auch mit Hinzufügen... Ihre eigenen Vorlagen erstellen und mit Bearbeiten... Vorlagen verändern. Vorlagen bestehen aus konstanten Textteilen und Variablen, die in geschweifte Klammern eingeschlossen werden.
    • Im Feld Wink können Sie einen kurzen Text eingeben, der in den Vorlagen die Variable {cue} ersetzen soll.
    • Im Feld Startwert können Sie den Startwert für die Sequenznummer-Variable angeben.
