CSS-Stylesheets

Das Farbschema der grafischen Benutzeroberfläche werden mit Hilfe von CSS-Stylesheets kontrolliert. Fünf Farbschemata sind bereits vordefiniert:

p Natives Farbschema der Plattform
b Schwarz
d Dunkelgrau
g Grau
w Weiss

Zusätzliche Farbschemata können durch Hinzufügen von CSS-Stylesheets zum Unterordner dropins/ im ZorRa Installationsordner definiert werden. Auf diese Weise ist es auch möglich, die vordefinierten Farbschemata zu überschreiben: es werden einfach Stylesheets unter den Namen b.css, d.css, g.css or w.css im Unterordner dropins/ des Installationsordners abgelegt.

Das folgende Beispiel zeigt das Stylesheet d.css, wie es für Linux-Plattformen definiert ist. Die verschiedenen GUI-Elemente werden einfach anhand ihres Klassennamens (SWT- und Piccolo-SWT-Klassen werden verwendet) adressiert. "*" steht für "alle Klassen".
Spezielle Widgets werden durch einen angehängten Lattenzaun, gefolgt von der Widget-ID addressiert. Auf spezielle Stilkonfigurationen und Zustände eines Widgets kann mit einer in eckigen Klammern angegebenen Bedingung Rücksicht genommen werden, z.B. mit [editable=false].
