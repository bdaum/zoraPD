Webgalerie-Designer-Ansicht

Der Zweck des Webgalerie-Designers ist der Entwurf von Foto-Ausstellungen, die auf dem World Wide Web veröffentlicht werden. Bevor Webgalerien generiert werden, können die Bilder in eine geeignete Reihenfolge sortiert und in Gruppen (Storyboards) angeordnet werden. Sie können Bilder einfach aus einer Galerie in ein Storyboard mittels Drag&Drop ziehen. Drücken Sie Strg, wenn Sie die Bilder auf ein im Storyboard vorhandenes Bild ziehen, so ersetzen die neuen Bilder das schon im Storyboard vorhandene Bild. Andernfalls werden die neuen Bilder einfach zum Storyboard hinzugefügt.

Die Bilder können frei angeordnet werden - auch das Ziehen auf ein anderes Storyboard ist möglich. Um Bilder von einem Storyboard zu entfernen, ziehen Sie diese einfache über den oberen Rand der Ansicht.

Jedes Bild wird mit einer Beschriftung unterhalb des Bildes ausgestattet. Alle Beschriftungen sind editierbar. Klicken Sie einfach eine Beschriftung an, um sie zu bearbeiten. Sie können auch die Eigenschaften eines Bildes (einschließlich der Beschriftung) bearbeiten, indem Sie auf die grüne Schaltfläche "Eigenschaften" auf der linken Seite der Bildunterschrift klicken.

Aktionen und Werkzeuge des Sichtfensters

  • Storyboard hinzufügen (). Fügt ein neues Storyboard der Ausstellung hinzu und zeigt den Dialog mit den Storyboard-Eigenschaften. Jedes Storyboard enthält zwei Schaltflächen in der linken unteren Ecke: eine, um das Storyboard zu löschen, und eine, um die Eigenschaften des Storyboards zu bearbeiten.
  • Webgalerie-Eigenschaften (). Zeigt den Dialog mit den Webgalerie-Eigenschaften.
  • Synchronisieren (). Schaltet den Synchronisationsmodus ein oder aus. In diesem Modus wird das jeweils selektierte Bild in der aktuellen Kollektion angezeigt, sofern vorhanden.
  • Generieren (). Erzeugt die Webgalerie und zeigt sie im Webbrowser der Ablaufplattform.
  • Save as Design ... (). Erzeugt die Webgalerie und zeigt sie in einem Dialog. Die Einstellungen zur Darstellung und andere designbezogene Einstellungen können nun unter einem Namen abgespeichert werden. Ein solches Design kann später in einer bestehenden oder neuen Webgalerie geladen werden.

Navigation

Die Navigation verläuft ähnlich wie beim Hüllen-Sichtfenster. Sie können den gesamten Bereich durch Ziehen mit der Maus nach oben, unten, rechts oder links verschieben. Dabei darf der Cursor nicht über einem Bild stehen. Strg beschleunigt die Bewegung. Mit den anderen Modifikationstasten (jedoch nicht der Zoomtaste, siehe unten) können Sie das Verschieben des gesamten Bereichs erzwingen, auch wenn der Cursor gerade über einem Bild steht.

Sie können den Anzeigenabstand anpassen (zoomen), indem Sie bei gedrückter Zoomtaste (ALT- oder UMSCHALT- bzw. rechter Maustaste je nach den Benutzervorgaben) und gedrückter Maustaste die Maus nach links oder rechts bewegen. Alternativ können Sie das Mausrad verwenden, um zu zoomen.

Ein Doppelklick auf ein Galleriebild öffnet den Dialog mit den Bildeigenschaften.

Webgalerie-Eigenschaften

  • Allgemein
    • Name. Der Name der Galerie.
    • Beschreibung. Kurze Beschreibung des Webgalerie. Dieses Feld kann HTML-Markup enthalten.
    • Name der HTML-Seite. Dies ist der Name der Eingangswebseite in der generierte Webgalerie. Der Standardwert ist "index.html".
    • Schlagworte. Geben Sie hier die Schlagworte an, die in den Kopfeintrag der erzeugten Webseite eingefügt werden sollen.
    • Download-Link. Geben Sie hier einen Text wie z.B. "Download Original" ein. Dieser Link wird in den Bildbeschriftungen angezeigt werden und ermöglicht dem Benutzer, die Originaldatei herunterladen. Wenn Sie dieses Feld leer lassen, wird kein Download-Link generiert. Bitte denken Sie daran, dass nicht alle Benutzer in der Lage sind, das Format Ihrer Originale zu verarbeiten - besonders, wenn es sich um das RAW-oder das DNG-Format handelt. Beachten Sie auch, dass Ihre Originale Metadaten enthalten können, die nun der Öffentlichkeit zugänglich gemacht werden.
    • Namensschild. Dies kann ein GIF-, JPEG-oder PNG-Bild sein, das ganz oben auf der Seite platziert wird.
    • Wasserzeichen erstellen. Fügt jedem einzelnen Bild ein Copyright-Vermerk in Form eines Wasserzeichens hinzu. Dafür wird der Copyright-Eintrag des Bildes wird verwendet. Wenn für das Bild nicht ein eigenes Copyright angegeben wurde, wird das Copyright für die Galerie (siehe unten) verwendet.
    • Copyright. Geben Sie hier einen Copyright-Vermerk ein (das Copyright, das für die komplette Galerie gilt, nicht für die einzelnen Bilder). Die Copyright-Bemerkung wird in der Fußzeile der generierten Webseite eingefügt.
    • Kontakt. Ein Kontaktname, der in der Fußzeile der generierten Webseite angezeigt wird.
    • E-Mail. Eine E-Mail-Adresse für den Kontakt. Der Kontaktname wird mit einem E-Mail-Link an diese Adresse ausgestattet.
    • Web-URL. Ein Web-URL, der in der Fußzeile der Webseite angezeigt wird. Falls angegeben, wird auch das Namensschild mit einem Link zu dieser Seite ausgestattet.
    • Engine-Referenz. Generiert den spezifizierten Text, gefolgt vom Namen der verwendeten Web-Engine. Ein Doppelklick auf das Eingabefeld fügt den Text  "Powered by" ein.
  • Darstellung
    • Hintergrundbild. Dies kann ein GIF-, JPEG-oder PNG-Bild, das im Hintergrund der generierten Seite angezeigt wird.
    • Wiederholen. Aktivieren Sie dieses Kontrollkästchen, wenn Sie ein kleines Hintergrundbild wiederholen möchten, um den verfügbaren Platz zu füllen.
    • Farben
      • Hintergrund. Die Farbe des Hintergrundes, wenn kein Bild definiert ist.
      • Schattierung. Die Farbe von schattierten Bereichen, falls verwendet.
      • Rahmen. Die Farbe von Grenzlinien.
      • Link. Die Farbe von klickbaren Verweisen.
    • Größe der Miniaturansicht. Wählen Sie das gewünschte Größe für die Miniaturansichten. Wenn Sie ein quadratischen Format auwählen, wird das Zentrum des Bildes verwendet.
    • Deckkraft. Die Deckkraft der Miniaturansichten. Einige Web-Engines benutzeb diese Einstellung, um die aktuell ausgewählte Miniaturansicht hervorzuheben..
    • Zwischenraum. Der Raum zwischen Miniaturansichten in Pixeln.
    • Seitenränder. Freier Raum oben, rechts, unten und links in Pixeln.
    • Schriftarten
      • Titel. Schriftart und Farbe der Seitentitel.
      • Untertitel. Schriftart und Farbe der Untertitel. Wenn die Web-Engine mehrere Storyboards unterstützt, wird der Storyboard-Titel mit dieser Schriftart formatiert.
      • Bildtitel. Schriftart und Farbe des Bildtitels.
      • Navigation. Schriftart und Farbe für Navigations-Hyperlinks.
      • Beschreibung. Schriftart und Farbe von Standardtext.
      • Fußzeile. Schriftart und Farbe der Fußzeile.
  • HTML
    • Im <HEAD>-Element. HTML-Codeschnipsel, die in das <HEAD>-Element eingefügt werden sollen. Typische Kandidaten sind <SCRIPT>- oder <LINK>-Elemente.
    • HTML im Kopfbereich. HTML-Codeschnipsel, die direkt hinter dem <BODY>-Tag eingefügt werden sollen.
    • HTML im Fußbereich. HTML-Codeschnipsel, die direkt vor dem </BODY>-Ende-Tag eingefügt werden sollen.
  • Metadaten
    • Diese Gruppe zeigt wählbare EXIF-und IPTC-Eigenschaften, die in den erzeugten Bildern enthalten sein werden. Miniaturansichten enthalten niemals Metadaten. Originale sind von diesen Einstellungen nicht betroffen - sie werden immer mit ihren ursprünglichen Metadaten hochgeladen.
    • Metadaten in Beschriftungen anzeigen. Aktivieren Sie dieses Kontrollkästchen, um Metadaten nicht nur in die Bilddateien aufzunehmen, sondern um die ausgewählten Metadaten-Felder auch in den Bildunterschriften sichtbar zu machen.
  • Web-Engine
    • Engine. Verschiedene Web-Engines können ausgewählt werden. Jede Engine braucht eventuell zusätzliche spezielle Parameter, die im unteren Teil des Dialogs aufgeführt sind.
  • Output
    • Lokaler Ordner. Bitte wählen Sie einen Ordner für die Ausgabe der generierten Webgalerie, wenn Sie die generierten Dateien lokal abspeichern möchten.
    • FTP-Verzeichnis. Bitte wählen Sie eine FTP-Konto aus oder erzeugen Sie ein neues, wenn Sie die generierten Dateien auf einen Server hochladen möchten.
    • Sie können die Ausgabequalität mit diesen Einstellungen beeinflussen.

Storyboard-Eigenschaften

  • Titel. Wird als Untertitel angezeigt, wenn die Web-Engine mehrere Storyboards unterstützt. Kann leer bleiben.
  • Bildunterschriften anzeigen. Aktivieren Sie dieses Kontrollkästchen, wenn Sie Bildtitel für die Bilder des Storyboards anzeigen wollen.
  • Beschreibung. Kurze Beschreibung des Storyboards. Dieses Feld kann HTML-Markup enthalten.
  • Beschreibungen anzeigen. Aktivieren Sie dieses Kontrollkästchen, wenn Sie Bildbeschreibungen für die Bilder des Storyboards anzeigen wollen.
  • EXIF-Daten anzeigen. Aktivieren Sie dieses Kontrollkästchen, wenn Sie die EXIF-Daten für die Bilder des Storyboards anzeigen wollen.
  • Bildgröße. Wählen Sie eine geeignetes Bild-Größe für alle Bilder des Storyboards.
  • Kleine Bilder vergrößern. Aktivieren Sie dieses Kontrollkästchen, wenn kleine Bilder auf die ausgewählte Bildgröße vergrößert werden sollen. Große Bilder werden immer auf die ausgewählte Bildgröße verkleinert.

Bildeigenschaften

  • Beschriftung. Der Titel des Bildes, der unter dem Bild angezeigt werden soll.
  • Beschreibung. Kurze Beschreibung des Bildes. Wird mit in der Bildbeschriftung angezeigt, aber mit einer anderen Schriftart (Schriftart Beschreibung). Dieses Feld kann HTML-Markup enthalten.
  • HTML-Alt-Text. Text, der als <ALT>-Element in die Web-Seite eingefügt werden soll. Dieser Text wird während des Ladens des Bildes angezeigt, oder wenn es nicht geladen werden kann.
  • Herunterladbar. Aktivieren Sie dieses Kontrollkästchen, wenn Sie einen Download-Link für dieses Bild wollen. In diesem Fall wird auch das Originalbild in den Ausgabeordner kopiert.
  • Metadaten hinzufügen. Aktivieren Sie dieses Kontrollkästchen, wenn Sie Metadaten in das Bild einfügen möchten. Dies beeinflusst auch die Abbildung der Metadaten in der Bildbeschreibung.

Kontextmenü

Um das Kontextmenü zu erhalten, klicken Sie ein Bild mit der rechten Maustaste. Die folgenden Funktionen stehen zur Verfügung:
  • Zu Bild springen. Zu einem von einer Liste ausgewählten Bild springen.
  • Im Vollbildmodus zeigen. Zeigt das Bild in voller Auflösung.
  • Ändern. Das Bild kann bearbeitet oder rotiert werden.
  • Ordner zeigen. Bild im Systemordner zeigen.
  • In Zeitlinie anzeigen. Bild in seiner Zeitlinienkollektion anzeigen.
  • In Karte anzeigen. Aufnahmeort im Kartensichtfenster anzeigen (nur für geo-codierte Bilder).
  • Sprachnotiz abspielen. Spielt die dem Bild zugeordnete Sprachnotiz ab, falls vorhanden.
  • Storyboard-Eigenschaften bearbeiten. Erlaubt das editieren der Eigenschaften des ausgewählten Storyboards.
  • Storyboard löschen. Löscht das komplette Storyboard.
