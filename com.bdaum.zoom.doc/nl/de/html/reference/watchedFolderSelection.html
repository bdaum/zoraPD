Auswahl des überwachten Ordners

Hier können Sie neue Ordner in die Liste der überwachten Ordner hinzufügen oder den Pfad  eines definierten überwachten Ordner ändern. Sie müssen auch den Typ des Ordners auswählen:

  • Speicher. Wenn einem solchen Ordner neue Bilder hinzugefügt werden, werden die Bilder automatisch in den Katalog importiert. Wenn Bilder in einem solchen Ordner - zum Beispiel durch einen externen Editor - geändert werden, werden die Bilder automatisch erneut importiert. Jedoch wird nur die Miniaturansicht aktualisiert, während die Meta-Daten im Katalog intakt bleiben.
  • Transfer. Bilder in diesen Ordner werden auf die gleiche Weise importiert, wie Bilder aus einem Ordner in eine neue Ordnerstruktur importiert werden. Sie bleiben nicht an ihrem aktuellen Speicherort, sondern werden in einen Zielordner verschoben. Dabei können sie auch umbenannt werden. Ein typisches Szenario ist eine Kamera, die mit WLAN ausgestattet ist. Die Kamera überträgt einen Stapel von Bildern auf einen dedizierten Transferordner. Zora Photo Direktor beginnt dann automatisch mit der Verarbeitung dieser Bilder.

Sie können auch Unterordner überwachen oder von der Überwachung ausschließen. Diese Optiion arbeitet rekursiv auf Unterordner von Unterordnern und so weiter.
