Wortschatz-Einstellungen

Die IPTC-Felder Intellektuelles Genre, Szenen-Code und Thema-Code werden durch vordefinierte Vokabulare unterstützt, die von iptc.org zur Verfügung gestellt werden. Zusätzlich können benutzerdefinierte Vokabulare verwendet werden.

Geben Sie dazu einfach die Wörter jedes Vokabulars an, jedes Wort in einer separaten Zeile. Mit den Schaltflächen auf der rechten Seite können Sie alle Wörter aus dem benutzerdefinierten Vokabular löschen, das benutzerdefinierte Vokabular sortieren, in einer Datei speichern oder von einer Datei laden. Das Sortieren entfernt auch Duplikate.
