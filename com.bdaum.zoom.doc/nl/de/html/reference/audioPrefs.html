Audio-Einstellungen

  • Signale
    • Signalton bei Eingabeaufforderung. Aktivieren Sie diese Option, wenn eine Eingabeaufforderung durch ein akustisches Signal angezeigt werden soll.
    • Akustisches Signal nach Abschluss einer Operation. Wählen Sie diese Option, wenn der Abschluss einer Operation mit einem akustischen Signal angezeigt werden soll.
  • Sprachnotizen
    Diese Einstellungen bestimmt die Qualität der Sprachnotizen.
    • Abtastrate. Die Abtastrate ist standardmäßig auf eine mittlere Qualität (22 kHz) eingestellt. Falls Sie eine höhere Qualität wünschen, wählen Sie eine höhere Abtastrate. Ist Speicherplatz ein Thema, wählen Sie eine niedrigere Abtastrate.
    • Bits pro Sample. Die Bittiefe ist standardmäßig auf 8 Bit eingestellt. Falls Sie eine höhere Qualität benötigen, wählen Sie eine größere Bittiefe.
