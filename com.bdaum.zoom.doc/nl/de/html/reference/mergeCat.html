Kataloge zusammenführen

Mit dieser Funktion können Sie die Bilder eines anderen Katalogs in den aktuellen Katalog importieren.

Sie müssen die Datei des zu importierenden Katalogs angeben und ein Verfahren für den Umgang mit Dubletten auswählen:

  • Überspringen. Eine bereits bestehender Katalogeintrag bleibt erhalten - der neue Eintrag wird ignoriert.
  • Ersetzen. Der bereits bestehende Katalogeintrag wird gelöscht und der neue Eintrag wird importiert.
  • Zusammenführen. Der neue Eintrag wird mit dem bereits vorhandenen Eintrag zusammengeführt. Wie diese Zusammenführung durchgeführt wird, kann im Detail auf der nächsten Seite des Assistenten angegeben werden.
    • Überschreiben. Alle ausgewählten Metadaten-Felder werden überschrieben.
    • Nur leere Einträge füllen. Ausgewählte Metadaten-Einträge werden nur überschrieben, wenn sie derzeit keinen Wert enthalten.
    • Kombinieren. Ähnlich wie Nur leere Einträge füllen, nur dass die Werte multi-dimensionaler Felder (wie z.B. Schlagworte) miteinander kombiniert werden.
