Vorschau-Sichtfenster

Das Vorschau-Sichtfenster zeigt eine größere Version der Miniaturansicht (jeoch nicht das hochauflösende Bild).

Die folgenden Funktionen sind verfügbar:

  • Kontext- und Sichtfenster-Funktionen
    • Cue. Schaltet das Sichtfenster zwischen Cue- und Auswahlmodus. Im Cue-Modus wird das Bild unter dem Cursor gezeigt, im Auswahlmodus das gerade ausgewählte Bild.
    • als Graustufenbild. Schaltet die Vorschau zwischen Farb- und Schwarzweißmodus um.
    • Schwarz-Weiß-Farbfilter setzen. Setzt einen Filter für die Konvertierung von Farbe zu Graustufen, wenn der Schwarzweißmodus aktiv ist.
    • Vollbild anzeigen. Damit wird ein externer Bildbetrachter aufgerufen. Ist kein externer Bildbetrachter definiert, wird der interne Bildbetrachter aufgerufen. Durch das gleichzeitige Drücken der  Umschalt- oder einer anderen Steuertaste kann der interne Bildbetrachter erzwungen werden. Je nach Steuertaste sind dabei verschiedene Modi wählbar. 
    • Bearbeiten. Der Standard-Editor der Bilddatei wird aufrufen. Wenn mehrere Bilder ausgewählt sind, wird der Editor nur aufgerufen, wenn er alle beteiligten Dateitypen unterstützt.
    • Bearbeiten mit. Damit können Sie unter allen Editoren, die dem Bilddateityp zugeordnet sind, auswählen. Sind mehrere Bilder ausgewählt, werden nur Editoren, die alle beteiligten Dateitypen unterstützen, angezeigt.
    • Zeit-Suche. Sucht nach Bildern, die sich im gleichen Zeitfenster wie das ausgewählte Bild befinden. Die Größe des Zeitfensters kann angegeben werden.
    • Ähnlichkeitsrecherche. Sucht inhaltsbasiert nach Bildern, die dem ausgewählten Bild (oder einer Zeichnung) visuell ähnlich sind. 
    • Umkreissuche. Sucht nach weiteren Bildern, die im Umkreis der selektierten Bilder liegen. Eine maximale Entfernung kann angegeben werden.
    • Sprachnotizen>
      • Sprachnotiz hinzufügen Damit wird ein Dialogfeld angezeigt, das es Ihnen ermöglicht, sofort eine Sprachnotiz aufzeichnen oder eine vorhandene Audiodatei als Sprachnotiz anzufügen.
      • SprachnotizenSprachnotizen entfernen. Dadurch werden früher hinzugefügte Sprachnotizen entfernt. Wenn die Sprachnotiz beim Anlegen aufgenommen worden war, wird auch die Audio-Datei gelöscht.
      • Sprachnotizen>Sprachnotiz abspielen. Damit wird die Sprachnotiz des selektierten Bildes abgespielt.
    • Metadaten>
      • Metadaten kopieren. Kopiert die Metadaten des ausgewählten Bildes in die Zwischenablage.
      • Metadaten>Metadaten hinzufügen. Kopiert die Metadaten von der Zwischenablage in die selektierten Bilder. Ein Dialog ermöglicht die Auswahl spezifischer Metadaten.
    • Beziehungen>
      • Derivate zeigen. Öffnet das Hierarchie-Sichtfenster und zeigt alle Bilder, die vom ausgewählten Bild abgeleitet wurden.
      • Originale anzeigen. Öffnet das Hierarchie-Sichtfenster und zeigt das Bild, von dem das ausgewählte Bild abgeleitet wurde.
      • Composites zeigen. Öffnet das Hierarchie-Sichtfenster und zeigt die Bilder, die unter Verwendung des aktuell ausgewählten Bildes zusammengesetzt wurden (z.B. Panorama, Photomontage).
      • Komponenten zeigen.Öffnet das Hierarchie-Sichtfenster und zeigt die Bilder, die Bestandteile des aktuell ausgewählte Bilds sind.
    • Ordner anzeigen (). Öffnet den externen Ordner, in dem sich das Bild befindet, wählt diesen Ordner auch im Katalog-Sichtfenster aus und öffnet diese Kollektiion.
    • In Timeline anzeigen (). Selektiert die Timeline-Kollektion, der das Bild zugehört, im Katalog-Sichtfenster und öffnet diese Kollektiion.
    • Aktualisieren. Aktualisiert Kataloginformationen der selektierten Bilder oder einer selektierten Kollektion.
    • Löschen. Löscht die ausgewählten Bilder aus dem Katalog. Optional kann die Bilddatei auch vom Datenträger gelöscht werden.
    • Untertitel konfigurieren. Ruft einen Editor zum Konfigurieren der Zeile unter dem Bild auf.
  • Werkzeugtasten des Sichtfensters
    • Sprachnotiz abspielen (). Spielt eine angehängte Sprachnotiz.
    • als Graustufenbild (). Schaltet die Vorschau zwischen Farb- und Schwarzweißmodus um.
    • Schwarz-Weiß-Farbfilter setzen. Setzt einen Filter für die Konvertierung von Farbe zu Graustufen, wenn der Schwarzweißmodus aktiv ist.
    • Vollbild anzeigen (). Damit wird ein externer Bildbetrachter aufgerufen. Ist kein externer Bildbetrachter definiert, wird der interne Bildbetrachter aufgerufen. Durch das gleichzeitige Drücken der  Umschalt- oder einer anderen Steuertaste kann der interne Bildbetrachter erzwungen werden. Je nach Steuertaste sind dabei verschiedene Modi wählbar. 
    •  Bearbeiten (). Der Standard-Editor der Bilddatei wird aufrufen. Wenn mehrere Bilder ausgewählt sind, wird der Editor nur aufgerufen, wenn er alle beteiligten Bildtypen unterstützt.

  • Mausgesten
    • Doppelklick. Damit wird ein externer Bildbetrachter aufgerufen. Ist kein externer Bildbetrachter definiert, wird der interne Bildbetrachter aufgerufen. Durch die gleichzeitige Betätigung der  Umschalt- oder einer anderen Steuertaste kann der interne Bildbetrachter erzwungen werden. Je nach Steuertaste sind dabei verschiedene Modi wählbar.
    • Klick auf die Untertitelzeile. Ruft einen Editor zum Konfigurieren der Zeile unter dem Bild auf.
  • Tastatur-Aktionen
    • Eingabe. Identisch ist mit Kontext-Funktion Bearbeiten.
    • Umschalt + Eingabe. Identisch ist mit Kontext-Funktion Bearbeiten mit
    • Tab. Identisch mit Doppelklick
    • Tab + Umschalt. Identisch mit Doppelklick + Umschalt
    • Tab +Strg. Setzt den Fokus auf das nächste Sichtfenster
    • Tab +Strg + Umschalt. Setzt den Fokus auf das vorige Sichtfenster
    • Entf. Löscht die ausgewählten Bilder
    • 0,1,2,3,4,5. Bewertet die ausgewählten Bilder mit 0,1,2,3,4,5 Sternen
