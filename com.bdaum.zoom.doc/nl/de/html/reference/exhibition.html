Ausstellungsdesigner-Sichtfenster

Der Zweck des Ausstellungdesigners ist die Gestaltung von physischen fotografischen Ausstellungen. Die Reihenfolge und Anordnung der Bilder kann vor dem tatsächlichen Hängen ausprobiert werden. Bilder können auf virtuelle Wände platziert werden. Darüber hinaus erfolgt eine Prüfung der Auflösung für jedes Bild. Zu niedrige Auflösung führt zu einer Warnung.

Eine Ausstellung besteht aus einer oder mehreren Wänden. Platzieren Sie einfach Bilder auf die Wände, indem Sie sie aus einer Galerie auf eine Wand ziehen. Drücken Sie Strg, wenn Sie die Bilder auf ein in der Wand vorhandenes Bild ziehen, so ersetzen die neuen Bilder das schon auf der Wand vorhandene Bild. Andernfalls werden die neuen Bilder einfach zur Wand hinzugefügt.

Per Drag&Drop können Sie die Bilder frei zwischen Wänden verschieben. Bilder können gelöscht werden, indem Sie sie über den oberen Rand des Sichtfensters ziehen.

Jedes Bild besitzt in der oberen rechten Ecke ein Griffstück und ist mit einem Label an der rechten Seite des Bildes ausgestattet. Dieses zeigt die gleichen Daten, die normalerweise auf den Labeln in Ausstellungen verwendet werden. Alle Einträge eines Labels können bearbeitet werden. Klicken Sie einfach auf einen Eintrag, um ihn zu editieren.

Mit Hilfe des Griffstücks können Sie die Größe des Bildes anpassen. Eine Warnung wird auf dem Label angezeigt, wenn das Bild zu groß für die gegebene Auflösung wird.

Sichtfensterfunktionen
  • Ausstellungseigenschaften (). Zeigt den Dialog mit der Ausstellungeigenschaften.
  • Synchronisieren (). Schaltet den Synchronisationsmodus ein oder aus. In diesem Modus wird das jeweils selektierte Bild in der aktuellen Kollektion angezeigt, sofern vorhanden.
  • Wand hinzufügen (). Fügt der Ausstellung eine neue Wand hinzu und zeigt ein Dialogfeld mit den Wandeigenschaften. Jede Wand enthält zwei Schaltflächen in der unteren linken Ecke: eine, um die Wand zu löschen, und eine zum Bearbeiten der Wandeigenschaften.
  • Virtuelle Ausstellung generieren (). Generiert eine Website mit einer virtuellen 3D-Ausstellung, die auf jedem HTML5-fähigen Webbrowser angesehen werden kann (Internet Explorer 9 oder höher empfohlen). Stellen Sie sicher, dass der Grundriss der Ausstellung, die Wandhöhe und die Standardbetrachtungshöhe wohl definiert sind.

    Sie können mehrere Ausstellungen in das gleiche Verzeichnis (Website) generieren. Jede Ausstellung wird dann als ein separater Ausstellungsraum generiert, und der Benutzer erhält die Möglichkeit zwischen den Räumen hin- und herzuschalten.

Navigation

Die Navigation verläuft ähnlich wie beim Hüllen-Sichtfenster. Sie können den gesamten Bereich durch Ziehen mit der Maus nach oben, unten, rechts oder links verschieben. Dabei darf der Cursor nicht über einem Bild stehen. Strg beschleunigt die Bewegung. Mit den anderen Modifikationstasten (jedoch nicht der Zoomtaste, siehe unten) können Sie das Verschieben des gesamten Bereichs erzwingen, auch wenn der Cursor gerade über einem Bild steht.

Sie können den Anzeigenabstand anpassen (zoomen), indem Sie bei gedrückter Zoomtaste (ALT- oder UMSCHALT- bzw. rechter Maustaste je nach den Benutzervorgaben) und gedrückter Maustaste die Maus nach links oder rechts bewegen. Alternativ können Sie das Mausrad verwenden, um zu zoomen.

Ein Doppelklick auf einen Wandhintergrund zoomt heraus, so dass alle Wände sichtbar werden. Ein Doppelklick auf ein Bild-Label zoomt auf dieses Label, so dass es bequem editiert werden kann. Ein weiterer Doppelklick macht den ersten Doppelklick wieder rückgängig.

Ausstellungseigenschaften

  • Übersicht
    • Ausstellung
      • Name. Der Name der Ausstellung
      • Beschreibung. Kurze Beschreibung der Ausstellung (Ausstellungsort, Datum, Zeitplan, etc..). Dieses Feld kann HTML-Markup enthalten.
    • Infotafel
      • Text. Beschreibung des Ausstellungsinhalts, die auch in der Ausstellung angezeigt wird.
      • Position. Left fest, wo der Info-Text gezeigt wird: links oder rechts der Tür oder überhaupt nicht. Zusätzlich werden noch der Name der Ausstellung und die Namen der beteiligten Künster angezeigt.
      • Credits anzeigen. Markieren Sie dieses Feld, um die aus den Bildern ermittelten Credits auf der Infotafel anzuzeigen.
  • Layout. 
    Hier werden Vorgabewerte für die einzelnen Labels der Ausstellung definiert. Einige dieser Werte können individuell für jedes Exponat überschrieben werden.
    • Standard-Betrachtungshöhe. Die Höhe, in der sich das Bild befindet, wenn Sie es auf eine Wand ziehen.
    • Varianz. Ein Wert mit dem die Standard-Betrachtungshöhe variiert wird, wenn Bilder auf eine Wand gezogen werden.
    • Passepartout and Rahmen
      • Passepartoutfarbe. Die Farbe des Passepartouts (wenn vorhanden).
      • Passepartoutbreite. Die Breite des Passepartouts in mm. Für Bilder ohne Passepartout die Breite 0 angeben.
      • Rahmen. Die Farbe der Bilderrahmen (wenn vorhanden).
      • Rahmenbreite. Die Breite der Bilderrahmen in mm. Für rahmenlose Bilder die Breite 0 angeben.
    • Label
      • Schriftart. Die Label-Größe wird automatisch der gewählten Schriftart angepasst.
      • Reihenfolge. Wählen Sie die Reihenfolge von Titel, Beschreibung und Credits.
      • Textvorgabe. Dieser Text wird in das Label jedes neuen Ausstellungsstücks übernommen.
      • Ausblenden. Standardmäßig werden alle Labels ausgeblendet, wenn diese Option markiert ist.
      • Ausrichtung. Definiert die räumliche Anordnung des Labels in Bezug zum Bild. 
      • Abstand. Der Abstand des Labels vom Bildrand.
      • Versatz. Definiert, um wieviel das Label entlang des Bildrandes verschoben ist.
  • Raster. 
    • Raster anzeigen. Zeigt ein Raster in jeder Wand.
    • Rastergröße. Abstand der Gitterlinien voneinander.
    • magnetisch. Bilder werden automatisch am Raster ausgerichtet, wenn Sie in die Nähe von Rasterlinien gesetzt werden.
  • Details. In diesem Abschnitt werden die Bild- und Rahmendaten für jede Wand einschließlich Position, Größe und Auflösung angezeigt. Sie können auf Drucker oder in ein PDF ausgegeben werden.
  • Grundriss. Hier können Sie die räumliche Anordnung der Ausstellungswände und des Eingangs (Kreis) festlegen. Wählen Sie eine Wand oder den Eingang aus und geben Sie die Koordinaten - und im Fall einer Wand - den Winkel ein. Sie können auch auf die Endpunkte der Wänder klicken und diese mit der Maus oder den Pfeiltasten verschieben.
  • Web Design.
    Diese Sektion betrifft nur generierte virtuelle Ausstellungen. 
    • Farben
      • Boden. Die Farbe des Fußbodens.
      • Horizont. Die Farbe des Horizonts.
      • Decken. Die Farbe der Saaldecke.
    • Medien
      • Tonspur. Eine Tondatei, die während des Betrachtens der Ausstellung abgespielt wird.
        Hinweis: Aktuelle Firefox-Versionen können MP3 Tonspuren nicht abspielen!
      • Namensschild. Dies kann ein GIF-, JPEG-oder PNG-Bild sein, das ganz oben auf der Seite platziert wird.
  • Ausgabe.
    Diese Sektion betrifft nur generierte virtuelle Ausstellungen. 
    • Ausgabeziel
      • Lokaler Ordner. Bitte wählen Sie einen Ordner für die Ausgabe der generierten Ausstellung, wenn Sie die generierten Dateien lokal abspeichern möchten.
      • FTP-Verzeichnis. Bitte wählen Sie eine FTP-Konto aus oder erzeugen Sie ein neues, wenn Sie die generierten Dateien auf einen Server hochladen möchten.
    • Web-Link. Dies ist der Name der Eingangswebseite in der generierte Ausstellung. Der Standardwert ist "index.html".
    • Schlagworte. Geben Sie hier die Schlagworte an, die in den Kopfeintrag der erzeugten Webseite eingefügt werden sollen.
    • Ausgabequalität. Sie können die Ausgabequalität mit diesen Einstellungen beeinflussen.
    • Wasserzeichen erstellen. Fügt jedem einzelnen Bild ein Copyright-Vermerk in Form eines Wasserzeichens hinzu. Dafür wird der Copyright-Eintrag des Bildes wird verwendet. Wenn für das Bild nicht ein eigenes Copyright angegeben wurde, wird das Copyright für die Ausstellung (siehe unten) verwendet.
    • Copyright. Geben Sie hier einen Copyright-Vermerk ein (das Copyright, das für die komplette Ausstellung gilt, nicht für die einzelnen Bilder). Die Copyright-Bemerkung wird in der Fußzeile der generierten Webseite eingefügt.
    • Kontakt. Ein Kontaktname, der in der Fußzeile der generierten Webseite angezeigt wird.
    • E-Mail. Eine E-Mail-Adresse für den Kontakt. Der Kontaktname wird mit einem E-Mail-Link an diese Adresse ausgestattet.
    • Web-URL. Ein Web-URL, der in der Fußzeile der Webseite angezeigt wird. Falls angegeben, wird auch das Namensschild mit einem Link zu dieser Seite ausgestattet.

Wandeigenschaften

  • Ort. Aufstellungsort der Wand.
  • Dimensionen. Breite und Höhe der Wand.
  • Wandfarbe. Wählen Sie eine Hintergrundfarbe für die Wand.

Kontextmenü

Um das Kontextmenü zu erhalten, klicken Sie ein Bild oder den Wandhintergrund mit der rechten Maustaste. Die folgenden Funktionen stehen zur Verfügung:
  • Zu Bild springen. Zu einem von einer Liste ausgewählten Bild springen.
  • Zur letzten Selektion springen. Springt zum zuletzt ausgewählten Bild.
  • Im Vollbildmodus zeigen. Zeigt das Bild in voller Auflösung.
  • Als verkauft markieren / Verkauftmarke entfernen. Setzt bzw. entfernt einen visuellen Anzeiger, dass das Ausstellungsstück verkauft ist.
  • Ändern. Das Bild kann bearbeitet oder rotiert werden.
  • Löschen. Entfernt die selektierten Bilder aus der Austellung.
  • Ausschneiden Entfernt die selektierten Bilder aus der Austellung, hät sie aber im der Applikationszwischenablage vorrätig.
  • Einfügen Fügt die Bilder aus der Applikationszwischenablage an der Mausposition ein.
  • Ordner zeigen. Bild im Systemordner zeigen.
  • In Zeitlinie anzeigen. Bild in seiner Zeitlinienkollektion anzeigen.
  • In Karte anzeigen. Aufnahmeort im Kartensichtfenster anzeigen (nur für geo-codierte Bilder).
  • Sprachnotiz abspielen. Spielt die dem Bild zugeordnete Sprachnotiz ab, falls vorhanden.
  • Layout. Bildlayout bearbeiten.
  • Wandeigenschaften bearbeiten. Die Eigenschaften der ausgewählten Ausstellungswand können geändert werden.
  • Wand löschen. Komplette Ausstellungswand löschen.

Tipp

Sie erleichtern sich das Hängen von Bildern sehr, wenn Sie nebeneinander hängende Bilder nicht auf die gleiche Höhe hängen.
