Bildunterschriften konfigurieren


  • Standardmäßig werden der Bildtitel und zusätzliche Informationen in den Bildunterschriften angezeigt. Der Titel kann jedoch vollständig abgeschaltet oder über eine Vorlage konfiguriert werden.
  • Vorlagen werden aus konstanten Textteilen und Variablen aufgebaut. Sie können beides eingeben, aber es ist viel komfortabler und sicherer, die Variablen aus einer Liste auszuwählen, indem Sie auf die Schaltflächen Variable einfügen.... oder Metadaten einfügen.... klicken. Letzteres bezieht sich auf die verschiedenen Felder, die mit einem Bild wie EXIF- und IPTC-Daten definiert sind.

    Wenn eine Variable oder Metadaten nicht aufgelöst werden können (z.B. ein Feld ist undefiniert), wird die Variable einfach aus der Vorlage gelöscht. Dies ist manchmal nicht ausreichend: Sie möchten vielleicht einen zusätzlichen Text damit löschen. Zu diesem Zweck wird eine spezielle Syntax bereitgestellt:

    Muster
    		 Beschreibung
    (?...?)
    		 Bedingte Gruppe Typ 1
    Der vollständige Ausdruck zwischen den Klammern wird gelöscht, wenn alle Variablen und Untergruppen innerhalb der Klammern unaufgelöst bleiben.
    Beispiel:
    (?Shot in     {meta=iptc_location_created&city}?)
    (??...?) Bedingte Gruppe Typ  2
    Der vollständige Ausdruck zwischen den Klammern wird gelöscht, wenn mindestens eine Variable oder Untergruppe innerhalb der Klammern unaufgelöst bleibt.
    Beispiel:
    (??{meta=owner}, {yyyy}?)
    ...|(...)
    		 Fallback
    Wenn der Ausdruck (Variable, bedingte Gruppe oder ein anderer Fallback) vor dem | nicht aufgelöst ist, wird stattdessen der Ausdruck innerhalb der Klammern verwendet. Andernfalls wird er verworfen,
    Beispiel:
     {meta=iptc_location_created&city}|({title})

    Bedingte Gruppen und Fallbacks können verschachtelt werden.
    Die Steuerzeichen werden bei der Generierung des Hover-Textes aus der Vorlage entfernt.
  • Darüber hinaus können Sie eine Schriftgröße und eine Ausrichtung angeben, jedoch nur für benutzerdefinierte Vorlagen.
