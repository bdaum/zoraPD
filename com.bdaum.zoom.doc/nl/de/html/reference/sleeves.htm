Hüllen-Sichtfenster

Das Hüllen-Sichtfenster verwendet die Metapher der durchsichtigen Negativaufbewahrungshüllen.

Die Navigation unterscheidet sich völlig von der des Lichtkasten-Sichtfensters:
  • Es gibt keine Bildlaufleiste. Stattdessen navigieren Sie, indem Sie auf den Zwischenraum zwischen den Bildern klicken und die gesamte Hülle in die gewünschte Richtung ziehen. Strg beschleunigt die Bewegung. Abhängig davon, was als Zoomtaste (siehe unten) verwendet wird, können Sie mit der ALT-Taste oder der rechten Maustaste das Ziehen der Hülle auch dann erzwingen, wenn der Cursor über einem Bild steht.
    Alternativ können Sie die Pfeiltasten verwenden, um einen Bildlauf durchführen.
  • Abhängig von den Einstellungen in Fenster>Benutzervorgaben>Maus* können Sie den Anzeigeabstand durch Drücken einer Zoomtaste (ALT oder Umschalt ) in Kombination mit der linken Maustaste oder aber durch Drücken der rechten Maustaste und gleichzeitigem Ziehen der Maus nach rechts oder links anpassen. Alternativ können Sie für das Zoomen auch das Mausrad oder die Tasten + und - verwenden.
  • Ein Doppelklick stellt den anfänglichen Betrachtungsabstand wieder her. Alternativ kann die *-Taste gedrückt werden. Ein weiterer Doppelklick macht den ersten Doppelklick wieder rückgängig.
Auch die Präsentation ist unterschiedlich:
  • Wenn Sie eine Miniaturansicht auswählen, wird sie nach vorn gebracht und vergrößert. Alle anderen Miniaturansichten werden mit einer transparenten Schicht grau maskiert.
  • Durch Drücken der Leertaste wird eine Miniaturansicht nach der anderen ausgewählt: eine schnelle und einfache Mini-Diashow. Umschalt + Leertaste arbeitet in die entgegengesetzte Richtung.
  • Auch mit Umschalt + Pfeiltaste können Sie angrenzende Miniaturansichten auswählen.

Sichtfenster-Funktionen
  • n Spalten. Die Miniaturansichten werden in einem Raster von 4-24 Spalten neu angeordnet.

Für weitere Tastaturbefehle und Mausgesten konsultieren Sie bitte die allgemeine Galerien-Hilfeseite.
