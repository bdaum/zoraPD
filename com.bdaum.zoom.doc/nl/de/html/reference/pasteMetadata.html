Metadaten einfügen

Dieser Dialog erlaubt Ihnen die Auswahl von Metadaten, die in die selektierten Bilder eingefügt werden sollen. Die Metadaten müssen vorher mit Hilfe der Menüfunktion Metadaten>Metadaten kopieren in die Zwischenablage kopiert werden.

Wählen Sie die einzufügenden Metadaten einfach durch Ankreuzen der Kästchen aus. Zusätzlich wählen Sie auch noch einen Einfügemodus:

  • Überschreiben Existierende Einträge werden von den neuen Metadaten überschrieben.
  • Nur leere Einträge füllen. Metadaten werden nur eingefügt, wenn der Eintrag vorher leer war. Existierende Metadaten werden auf diese Weise nicht zerstört.
