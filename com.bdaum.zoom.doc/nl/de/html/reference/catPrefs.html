Katalogeigenschaften verwalten

Viele Benutzereinstellungen sind Katalogspezifisch und werden deshalb in den separaten Katalogeigenschaften (ALT+Eingabe) verwaltet.
Wird ein neuer Katalog angelegt, so werden Sie aufgefordert, diese Katalogeigenschaften durchzusehen. Normalerweise werden die Eigenschaften eines neuen Katalog auf Standardwerte gesetzt. Sie haben aber auch die Möglichkeit, die Eigenschaften des aktuellen Katalogs auf den neuen Katalog zu übertragen. Dazu Markieren Sie die Option Aktuelle Katalogeigenschaften für neue Katalog verwenden.
