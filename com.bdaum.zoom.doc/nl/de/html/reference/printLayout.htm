Layout-Dialog Drucken

Die Layout-Dialogfeld Drucken bestimmt das Layout einer gedruckten Seite.

Auf Seitenebene können Sie die folgenden Einstellungen festlegen:

  • Titel
  • Untertitel
  • Fußzeile
  • Linke und rechte Ränder in mm.
  • Obere und untere Ränder in mm.
  • Horizontaler und vertikaler Abstand in mm.

In alle diese Text-Einträge können Sie die folgenden Variablen einfügen:

Variable Beschreibung
{catalog} Katalogname
{today} Aktuelles Datum
{count} Anzahl der Bilder
{pageNo} Seitenzahl
{pageCount} Gesamtzahl der Seiten
{collection} Titel der aktuellen Kollektion
{user} Aktueller Computerbenutzer
{owner} Katalogeigentümer

Auf Bild-Ebene können Sie die folgenden Einstellungen festlegen:

  • Bildbeschriftung Zeile 1
  • Bildbeschriftung Zeile 2
  • Breite der Umrahmungslinie

In alle diese Text-Einträge können Sie die folgenden Variablen einfügen:

Variable Beschreibung
{title} Bildtitel oder Bildname, wenn kein Titel vorhanden
{name} Bildname
{creationDate} Erstellungsdatum des Bildes
{collection} Titel der aktuellen Kollektion
{sequenceNo} Bild-Sequenznummer
{pageItem} Bildnummer auf der Seite
{size} Bildgröße in Pixel
{format} Dateityp

Zusätzlich können sämtliche EXIF- und IPTC-Metadatenfelder in die Beschriftungen mit eingeschlossen werden.
