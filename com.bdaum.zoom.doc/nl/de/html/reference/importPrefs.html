Importeinstellungen

Allgemein

  • Herstellernotizen importieren. Falls Sie in den herstellerspezifischen Makernotes der EXIF-Metadaten interessiert sind, markieren Sie diese Option. Falls nicht, lassen Sie diese Option unmarkiert. Falls Sie später Ihre Meinung ändern sollten, können Sie diese Option ändern und die EXIF-Daten Ihrer Bilder im Katalog aktualisieren.
  • Nach Import die importierten Bilder anzeigen. Setzen Sie diese Option, um nach einem Import die importierten Bilder automatisch in der aktuellen Galerie anzuzeigen.
  • Max. Länge der Importliste. Einträge in der Importgruppe, welche diese Maximallänge überschreiten, werden gelöscht (wenn leere Systemkollektionen gelöscht werden).
  • Import automatisch aufrufen, wenn ein Gerät angeschlossen wird. Wenn diese Option gesetzt ist, wird die Funktion Von Gerät importieren automatisch aufgerufen, wenn eine Kamera oder ein ähnliches Gerät mit dem Computer verbunden wird.

RAW-Konvertierung

  • Raw-Konverter. Hier können Sie zwischen dcraw und RawTherapee als Raw-Konverter wählen

dcraw

Die folgenden Optionen gelten für dcraw:
  • Konverter
    • Externes dcraw-Modul. Hier können Sie den Pfad zu einem externen DCRAW-Modul spezifizieren, also einem DCRAW-Konverter außerhalb des Installationsordners.
      Tipp: Falls Sie einen auf DCRAW basierenden RAW-Konverter wie z.B. LightZone benutzen, ist es sinnvoll, auf das in diesem RAW-Konverter installierte DCRAW-Modul zu verweisen. So können Sie sicherstellen, dass sowohl der RAW-Konverter als auch ZoRa PhotoDirector die gleiche Version von DCRAW verwenden.
  • Optionen
    Diese Optionen werden nur beim Laden hochauflösender Bilder angewandt, nicht jedoch bei der Generierung von Miniaturansichten.
    • Weißabgleich. Wählt die Methode für den Weißabgleich aus:
      • D65. Eine Graukarte, die mit einer D65-Lichtquelle beleuchtet wird, erscheint grau.
      • Kamera. Der von der Kamera vorgeschlagene Weißabgleich wird durchgeführt.
      • dcraw. Der vom DCRAW-Konverter vorgeschlagene Weißabgleich wird durchgeführt.
    • Interpolation. Wählt den DCRAW-Interpolationsalgorithmus aus. Verfügbar sind::
      • bilinear (schnell)
      • Variable Number of Gradients (VNG)
      • Patterned Pixel Grouping (PPG) 
      • Adaptive Homogeneity-Directed (AHD) 
    • Iterationen. Zahl der Iterationen mit einem 3x3-Median-Filter. Vermindert Artefakte, verlangsamt allerdings die Konvertierung.
    • Spitzlichter wiederherstellen. Überbelichte Spitzlichter werden teilweise wieder hergestellt. Verlangsamt die Konvertierung.
  • Aktive Rezept-Detektoren. Markieren Sie alle RAW-Konverter, die Sie für die Entwicklung von RAW- und DNG-Bildern verwenden. Entfernen Sie die Markierungen von den anderen, um Import und Anzeige nicht unnötig zu verlangsamen. Im Fall, dass zu einem Bild mehrere Rezepte von unterschiedlichen RAW-Konvertern vorliegen, wird das aktuellste Rezept verwendet. In der Spalte Konfiguration können Sie angeben, wie Konflikte zwischen konkurrierenten Entwicklungsrezepten aufgelöst werden sollen.
  • Entwicklungsrezepte beim Import von RAW-Bildern berücksichtigen. Entfernen Sie die Markierung, wenn Sie die in RAW-Konvertern gemachten Korrekturen nicht anwenden wollen. Das beschleunigt einige Operationen - und Sie erhalten die ursprünglichen RAW-Dateien zur Ansicht.
  • Rezept von abgeleiteten Bildern archivieren. Ist diese Option ausgewählt, werden RAW-Rezepte, die zur Herstellung eines abgeleiteten Bildes dienten, im Katalog archiviert. So kann das Rezept später leicht wieder hergestellt werden, wenn es nach der Erstellung des abgeleiteten Bildes modifiziert worden sein sollte.
  • Alle betroffenen Katalogeinträge jetzt aktualisieren? Diese Option steht nur nach dem Hinzufügen eines neuen Rezeptdetektors zur Verfügung. Markieren Sie die Option, um vorhandene Rezepte des neuen Detektors auf existierende Katalogeinträge anzuwenden.

RawTherapee

Die folgenden Optionen gelten für RawTherapee
  • Konverter
    • RawTherapee-Modul. Hier müssen Sie den Pfad des RawTherapee-Ausführungemoduls angeben, das auf Ihrem System installiert sein muss
  • Optionen
    • .pp3-Sidecar-Rezepte berücksichtingen. Markieren Sie diese Option, wenn Sie die von RawTherapee erstellten Konvertierungsrezepte während dem Import, der Anzeige und dem Export von Bildern anwenden wollen. Die Ergebnisse sind 100% identisch zu den mit RawTherapee direkt erstellten Konvertierungsergebnissen.

DNG-Konvertierung

    (Nur unter Windows und dem MacOS)
  • Grundlagen
    • RAW-Dateien importieren. Hier wird angegeben, wie RAW-Dateien importiert werden: 
      • In DNG konvertieren und sowohl RAW- als auch  DNG-Dateien importieren. Dies ist der sicherste Import-Modus, da Bilder sowohl im ursprünglichen Format als auch im kameraneutralen DNG-Format gespeichert werden.
      • Nur RAW-Dateien importieren. Eine Konvertierung in das DNG-Format findet nicht statt.
      • In DNG konvertieren und nur die DNG-Dateien importieren.
      • In DNG mit eingebetteten RAW-Daten konvertieren und die DNG-Dateien importieren. Mit dieser Option ist es möglich, die ursprüngliche RAW-Datei aus der DNG-Datei mit Hilfe des Adobe-DNG-Konverters wiederzugewinnen.
    • Speicherort des Adobe-DNG-Konverters. Wenn Sie RAW-Dateien während des Importvorgangs in das DNG-Format konvertieren möchten, müssen Sie den Speicherort des Adobe-DNG-Konverters hier angeben. Sie können eine Kopie der kostenlosen Adobe DNG Converter von www.adobe.com/products/dng/  downloaden.
  • Optionen
    • Unkomprimierte DNG-Dateien erstellen. Erweiterte Einstellung. Lesen Sie bitte die Dokumentation des Adobe-DNG-Konverters. 
    • Ursprüngliches RAW-Format in lineares Bild konvertieren. Erweiterte Einstellung. Lesen Sie bitte die Dokumentation des Adobe-DNG-Konverters. 
    • DNG-Unterordner. Konvertierte DNG-Dateien werden in einem Unterordner des Ordners, der die RAW-Datei enthält, abgelegt. Sie können hier diesen Unterordner frei benennen oder aber das Feld leer lassen, um die DNG-Dateien Seite an Seite mit den RAW-Dateien zu speichern.
