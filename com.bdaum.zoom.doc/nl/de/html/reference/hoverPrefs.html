Hover-Präferenzen

Auf dieser Voreinstellungsseite können Sie das anwendungsweite Hover-Verhalten konfigurieren. Bitte beachten Sie, dass Sie den Hover über die Menüfunktion Hilfe>Hover anzeigen oder durch Drücken von F2 aus- und einschalten können.

Vorlagen

In diesem Abschnitt können Sie die Texte konfigurieren, die eingeblendet werden, wenn der Mauszeiger über ein Bild, ein Dia, eine Kollektion usw. bewegt wird. Sie können dies für jedes Element einzeln tun.

Die Hovertexte werden über Vorlagen gesteuert. Vorlagen werden aus konstanten Textteilen und Variablen erstellt. Sie können beides manuell eingeben, es ist jedoch viel komfortabler und sicherer, die Variablen aus einer Liste auszuwählen, indem Sie auf die Schaltflächen  Variable einfügen... oder Metadaten einfügen... klicken. Die letztere bezieht sich auf die verschiedenen Felder, die mit einem Bild definiert sind, z.B. EXIF- und IPTC-Daten.

Ein Sonderfall ist die Variable {metadata}. Diese Variable erstellt einen gesamten Metadatenblock, der alle Metadatenfelder enthält, die in den Metadaten-Einstellungen im Abschnitt Hover konfiguriert sind.

Wenn eine Variable oder Metadaten nicht aufgelöst werden kann (z.B. ein Feld ist nicht definiert), wird die Variable einfach aus der Vorlage gelöscht. Dies reicht manchmal nicht aus: Sie möchten eventuell zusätzlichen Text damit löschen. Zu diesem Zweck wird eine spezielle Syntax bereitgestellt:


?
 Ein Fragezeichen am Anfang einer Zeile. Die gesamte Zeile wird gelöscht, wenn alle Variablen in dieser Zeile nicht aufgelöst sind.
??
 Zwei Fragezeichen am Anfang einer Zeile. Die gesamte Zeile wird gelöscht, wenn mindestens eine Variable in dieser Zeile nicht aufgelöst ist.
(? ... ?)
 Der vollständige Ausdruck zwischen den Klammern wird gelöscht, wenn alle Variablen in der Klammer unaufgelöst bleiben. Dies funktioniert auch über Zeilenumbrüche hinweg.
(?? ... ?)
 Der vollständige Ausdruck zwischen den Klammern wird gelöscht, wenn mindestens eine Variable innerhalb der Klammer unaufgelöst bleibt. Dies funktioniert auch über Zeilenumbrüche hinweg.

Diese Markierungen werden abschließend aus der Vorlage entfernt, wenn der Hover-Text generiert wird.

Timing

In diesem Abschnitt können Sie das Hover-Timing konfigurieren:
  • Verzögerung. Dies ist die Zeit zwischen der Erkennung eines Maus-Hover-Ereignisses und dem tatsächlichen Popup des Hovertexts. Erhöhen Sie diese Zeit, wenn Hover-Popups regelmäßig mit der normalen Mausbedienung in Konflikt stehen.
  • Basiszeit Die minimale Zeit, in der die Hover-Texte angezeigt werden.
  • Zeit pro Zeichen. Dies ist die Zeit, die zur Basiszeit für jedes Zeichen im Hovertitel und Hovertext hinzugefügt wird. Damit können Sie das Hoververhalten an Ihre Lesegeschwindigkeit anpassen.
