Parameter für Transferordner

Für jeden Transferordner können Sie Parameter angeben, die den Import der Bilder im Ordner beeinflussen:
  • Metadaten. Die Werte dieser Felder werden den Metadaten aller Bilder hinzugefügt, die importiert werden:
    • Künstler. Der Künstler, der das Foto gemacht. Lassen Sie dieses Feld leer, wenn verschiedene Künstler den gleichen Transferordner verwenden. Es ist gute Praxis, dass jeder Künstler einen unterschiedlichen Transferordner verwendet.
    • Dateiquelle. Definieren Sie, wie das Bild entstand, z. B. mit einem Scanner oder einer Digitalkamera. Setzen Sie dieses Feld auf Unbekannt, wenn die Bilder in diesem Ordner aus verschiedenen Quellen stammen. Es ist gute Praxis, verschiedene Transferordner für jede Dateiquelle zu verwenden.
  • Keine Duplikate importieren. Setzen Sie diese Option, wenn Sie nicht zweimal die gleiche Datei importieren möchten. Wenn Sie diese Option nicht setzen, wird das Bild erneut importiert und unter einem leicht geänderten Namen gespeichert.
  • Übersprungene Dateitypen. Verwenden Sie diese Option, um bestimmte Dateitypen zu überspringen. Wenn Ihre Kamera z.B. sowohl RAW- als auch JPeG-Dateien erzeugt, möchten Sie möglicherweise nur die RAW-Dateien importieren.
