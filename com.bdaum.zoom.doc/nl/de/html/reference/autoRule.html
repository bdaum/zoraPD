Regeln für die automatische Erstellung von Kollektionen

Gruppe

Dieses Attribut dient nur für die Begrenzung der angezeigten Felder, um die Auswahl zu erleichtern. Wenn Sie im Zweifel sind, wählen Sie Alle.

Feld

Jede Regel muss auf genau ein Metadatenfeld wie das EXIF-Feld für Belichtungszeit oder das IPTC-Feld für das Modellalter verweisen. Nicht für alle vorhandenen Metadaten werden hier Felder angeboten, insbesondere nicht für Felder, für die es bereits Systemkollektionen gibt (z. B. das IPTC-Erstellungsdatum, welches sich in den Kollektionen der Zeitachse niederschlägt).

Typ

Der Typ einer Regel leitet sich aus dem ausgewählten Metadatenfeld ab:
  • diskret. Das Feld hat nur wenige mögliche Werte wie das EXIF-Feld blitz_ausgelöst.
  • linear. Die Feldwerte sind in einer linearen Weise verteilt. Hier ist es sinnvoll, Sammlungen mit gleich großen Wertebereichen zu erstellen. Auch negative Werte sind zulässig. Ein typisches Beispiel ist das Feld EXIF-Feld für Lichtwert.
  • exponentiell. Feldwerte sind exponentiell verteilt. Hier ist es sinnvoll, die Wertebereiche der Kollektionen stetig immer größer werden zu lassen, so dass sowohl sehr kleine als auch sehr große Werte abgedeckt werden können. Negative Werte sind hier nicht zulässig. Ein typisches Beispiel ist das EXIF-Feld für die Belichtungszeit.

Parameter

  • Regeln mit diskretem Typ haben keine Parameter.
  • Für die Regeln mit linearen Typ können Sie eine feste Intervallgröße angeben, die für all Kollektionoen gleich ist. Wenn Sie beispielsweise eine Intervall-Größe 2 für das EXIF-Feld Lichtwert angeben, werden die Lichtwerte 0 und 1 in Kollektion, die Werte 2 und 3 in eine andere Kollektion gruppiert und so weiter.
    Alternativ können Sie die Intervalle ausdrücklich als eine durch Semikolons getrennte Liste angeben. z.B. 0; 2; 5; 8; 16
  • Für Regeln mit exponentiellen Typ können Sie den Basiswert einer exponentiellen Reihe angeben. Geben Sie beispielsweise 2 als Basiswert an, würde dies zu einer Intervall-Serie wie ...; 0,25; 0,5; 1; 2; 4; 8; .... führen. Dementsprechend würde Kollektionen mit Wertbereichen von 0,25 - 0,5; 0,5 - 1; 1 - 2; 2-4; 4-8 usw. erstellt werden.
    Alternativ können Sie die Intervalle ausdrücklich als eine durch Semikolons getrennte Liste angeben z. B. 0; 2; 5; 8; 16.
