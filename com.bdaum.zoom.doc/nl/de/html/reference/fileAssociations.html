Externe Programme

Allgemein

  • Metadaten vor dem Aufrufen eines externen Programms exportieren. Aktivieren Sie diese Option, wenn Sie die im Katalog gespeicherten Metadaten exportieren möchten bevor ein externes Programm wie z. B. ein Bildeditor oder -betrachter aufgerufen wird. So kann das externe Programm die aktualisierten Metadaten nutzen. Die Metadaten werden als XMP-Begleitdatei (sidecar) gespeichert.
  • Externer Bildbetrachter. Wenn Sie einen externen Bildbetrachter wie FastPictureViewer, Fast Stone Image Viewer oder IrfanView zum Anzeigen der Bilder bei hoher Auflösung verwenden möchten, geben Sie hier dessen Pfad an. Wenn Sie diesen Eintrag leer lassen, wird der interne Bildbetrachter verwendet.

Dateizuordnungen

Dateizuordnungen bestimmen, welche externen Editoren für bestimmte Dateitypen verwendet werden sollen. 

  • Dateitypen. Dieser Abschnitt listet alle Dateitypen auf, die mit Editoren verbunden sein können. Sie können neue Dateitypen mit Hilfe der Schaltfläche Hinzufügen... hinzufügen. Zu jedem Dateityp können mehrere Datei-Suffixe angegeben werden. So sind zum Beispiel *.jpg, *.jpeg, *.jpe, *.jfif alles gültigen Suffixe für JPEG-Dateien.
  • Zugeordnete Editoren. Um  Editoren einem Dateityp zuzuordnen, wählen Sie im oberen Bereich den Dateityp aus. Dann drücken Sie im unteren Bereich die Schaltfläche Hinzufügen... und navigieren zum ausführbaren Programm des Editors. Um einen Editor als Standard-Editor festzulegen, wählen Sie im unteren Bereich den Editor aus und drücken dann die Schaltfläche Standard. Standard-Editoren können mit der Bearbeiten-Aktion () des Kontextmenüs einer Galerie gestartet werden. Die anderen Editoren können durch Aufrufen der Bearbeiten mit...-Aktion () und Auswahl des Editors von einer Liste gestartet werden.
    Bereits in der Ablaufplattform registrierte Dateizuordnungen können mit einem Druck auf die Taste Systemeinstellungen übernehmen übernommen werden.
  • Zuletzt aufgerufenen Editor verwenden. Wurde diese Option angekreuzt, so erinnert sich jedes Bild an den zuletzt verwendeten Editor. Dieser wird dann an Stelle des Standardeditors verwendet.
