Metadaten-Einstellungen

  • Wesentliche Metadaten. Metadaten-Sichtfenster wie z. B. das Eigenschaften-Sichtfenster, das EXIF-Sichtfenster, das IPTC-Sichtfenster oder das Metadaten-Sichtfenster können zwischen der Anzeige aller Metadaten und nur der wesentlichen Metadaten umgeschaltet werden. Hier können Sie festlegen, welche Felder als wesentlich betrachtet werden..
  • Hover. Hier können Sie konfigurieren, welche Metadaten-Felder in einem Popup-Fenster angezeigt werden sollen, wenn die Maus über einem Bild stehen bleibt.
  • Toleranzen. Hier können Sie für jedes einzelne Feld angeben, welche Toleranzen bei einer Abfrage mit der Relation ähnlich verwendet werden sollen.
  • Export. Hier können Sie konfigurieren, welche Metadatenfelder beim manuellen oder automatischen Metadatenexport geschrieben werden. Wahlweise können Sie auch erzwingen, dass die Metadaten in originale JPEG-Dateien zurückgeschrieben werden. Beachten Sie, dass damit die ursprünglichen XMP-Metadaten in der Datei zerstört werden.
  • Tuning. Hier können Sie konfigurieren, welche Metadatenfelder in der Katalogdatenbank indiziert werden sollen. Die Datenbankindexierung (nicht zu Verwecheln mit der Text- und Ähnlichkeitsindexierung) beschleunigt die Abfragen für diese Felder erheblich, während Importe und Löschvorgänge etwas verlangsamt und die Kataloggröße erhöht wird. Beachten Sie, dass die De-Markierung der standardmäßig indizierten Metadatenfelder nicht möglich ist.
    Änderungen in diesem Abschnitt werden erst nach der Neueröffnung des Katalogs wirksam.
