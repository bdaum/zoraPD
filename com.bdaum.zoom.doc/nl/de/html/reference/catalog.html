Das Katalog-Sichtfenster

Das Katalog-Sichtfenster zeigt die im Katalog gespeicherten bildbezogenen Informationen. Diese Informationen sind in verschiedene Kategorien unterteilt:

  • Kollektionen. Kollektionen werden von einer Abfrage definiert, welche bestimmte Bilder aus dem Katalog auswählt. Diese Abfragen können vom Benutzer definiert werden, andere sind durch das System bereits vordefiniert. Während benutzerdefinierte Kollektion frei editiert werden dürfen, erlauben systemdefinierte Kollektionen nur das Ändern ihrer Sortierkriterien.
    Kollektionen können geschachtelt werden: Sie können Unterselektionen () zu jeder gegebenen Kollektion hinzufügen, um die Abfrage weiter zu verfeinern. Zum Beispiel: Durch Hinzufügen der Unterabfrage EXIF Blendenzahl > 5.6 zu einer 5-Sterne-Kollektion, erstellen Sie eine Kollektion, welche alle 5-Sterne-Bilder mit einer Blendenzahl größer als 5.6 enthält
    Unterabfragen können sowohl zu benutzerdefinierten als auch zu systemdefinierten Kollektionen hinzugefügt werden. Wir unterscheiden zwischen den folgenden Arten von Kollektionen:
    • Importe (). Kollektionen dieser Art werden automatisch beim Import von Bildern erstellt. Sie zeigen die Bilder von früheren Import-Operationen. Kollektionen dieser Art dürfen aus dem Katalog entfernt werden, außer der Kollektion Letzer Import (), die auch besonders dargestellt wird.
    • Kategorien (). Kollektionen dieser Art werden automatisch erstellt - eine für jede definierte Kategorie. Kategorien können hinzugefügt und gelöscht werden, allerdings muss das in der Kategorienverwaltung der Katalogeigenschaften geschehen (Katalog>Eigenschaften...>Kategorien).
    • Bewertungen (). Kollektionen dieser Art werden beim Neuanlegen eines Katalogs automatisch erstellt: eine Kollektion Kollektion für nicht bewertete Bilder. Kollektionen dieser Art können nicht vom Benutzer entfernt werden.
    • Zeitlinie ().Kollektionen dieses Typs werden automatisch beim Import von Bildern erstellt und gruppieren die Bilder nach Erstellungsdatum. Die Zeitlinie kann in den Katalog-Eigenschaften konfiguriert werden (Katalog>Eigenschaften>Allgemeines). Kollektionen dieser Art können nicht vom Benutzer entfernt werden, sondern werden automatisch gelöscht, wenn sie leer werden.
    • Orte ().Kollektionen dieses Typs können automatisch beim Import von Bildern erstellt werden und gruppieren die Bilder nach Erstellungsdatum. Die Ortskollektionen können in den Katalog-Eigenschaften konfiguriert werden (Katalog>Eigenschaften>Allgemeines). Kollektionen dieser Art können nicht vom Benutzer entfernt werden, sondern werden automatisch gelöscht, wenn sie leer werden.
    • Verzeichnisse (). Kollektionen dieses Typs werden automatisch beim Import von Bildern erstellt und reflektieren den tatsächlichen Speicherort der Bilder im Dateisystem. Beachten Sie, dass der Inhalt von Wechselmedien selbst dann angezeigt wird, wenn diese nicht online sind. Um BIlder eindeutig identifizieren zu können wird deshalb der Laufwerksbuchstabe durch die Datenträgerbezeichnung ersetzt.Kollektionen dieser Art können nicht vom Benutzer entfernt werden, sondern werden automatisch gelöscht, wenn sie leer werden.
    • Benutzerdefinikerte Kollektionen ().Kollektionen dieser Art werden ausschließlich durch den Benutzer definiert - entweder direkt im Katalog-Sichtfenster mit der Kontextfunktion Neue Kollektion erstellen oder aber mit Hilfe der Galerie-Menüfunktion Abfrage als Kollektion speichern. Die meisten Metadatenfelder können zur Formulierung beliebig komplexer Abfragen verwendet werden. Kollektionen dieser Art dürfen jederzeit vom Benutzer gelöscht werden.
  • Alben (). Alben ähneln den Kollektionen, werden jedoch nicht durch Abfragen definiert. Stattdessen ziehen Sie einfach Bilder aus einer Galerie auf das Icon eines Albums, um es diesem hinzuzufügen. Neue Alben werden im Katalog-Sichtfenster mit der Kontextfunkton Neues Album anlegen angelegt. Um ein Unteralbum einem bestehenden Album hinzu zu fügen, verwenden Sie die Kontextfunktion Unteralbum hinzufügen.

  • Präsentationen
    • Ausstellungen (). Ausstellungen enthalten die Bilder von  Real-World-Ausstellungen. Sie erleichtern die Planung und das Layout solcher Ausstellungen.
    • Diashows (). Diashows enthalten Bilderserien, die als Vollbild-Diashow angezeigt werden sollen. Insbesondere erlauben Diashows das Festlegen der Reihenfolge der Bilder und die Definition von Zeitintervallen und Überblendeffekten.
    • Web-Galerien (). Web-Galerien unterstützen die Zusammenstellung und Konfiguration von Galerien, die auf dem Web präsentiert werden sollen. Anschließend können daraus fertige Web-Sites generiert werden, die mit Hilfe verschiedener Engines wie Galleriffic, Image Gallery oder Pirobox gezeigt werden können.

Alle Elemente solcher Kategorien (Kollektionen, Alben und Präsentationen)  müssen aus Übersichtlichkeitsgründen jeweils zu einer benannten Gruppe () gehören. Solche Gruppen könnnen mit der Katalog-Menüfunktion Neue Gruppe erstellen erzeugt werden. Jede Gruppe darf nur Elemente der gleichen Kategorie enthalten.

Menüs und Aktionen


Die folgenden Funktionen stehen im Katalog-Sichtfenster zur Verfügung::
  • Katalog-Kontext-Funktionen
    • Untergruppe hinzufügen. Fügt eine geschachtelte Untergruppe einer existierenden Gruppe hinzu. Dies ist nur innerhalb von benutzerdefinierten Gruppen möglich..
    • Kollektion erstellen. Erstellt eine neue benutzerdefinierte Kollektion in der ausgewählten Gruppe.
    • Unterselektion hinzufügen. Fügt eine Unterabfrage der ausgewählten Kollektion hinzu.
    • Neues Album anlegen.  Damit wird ein neues Album in der ausgewählten Gruppe erstellt.
    • Unteralbum hinzufügen. Fügt eine Unteralbum dem ausgewählten Album hinzu.
    • Neue Präsentation erstellen>
      • Ausstellung. Damit wird eine neue Ausstellung in der ausgewählten Gruppe erstellt.
      • Diashow. Damit wird eine neue Diashow in der ausgewählten Gruppe erstellt.
      • Web-Galerie. Damit wird eine neue Web-Galerie in der ausgewählten Gruppe erstellt.
    • Diashow wiedergeben. Startet die Wiedergabe einer Diashow-Präsentation. Die Abspieleinheit ist interaktiv und erlaubt die Änderung von Bildeigenschaften während der Wiedergabe.
    • Alle Bilder auswählen. Wählt in den aktuell geöffneten Galerien alle Bilder aus.
    • Bearbeiten  Ermöglicht, das ausgewählte Element zu bearbeiten.
    • Ausschneiden. Entfernt das ausgewählte Element und speichert es in der Zwischenablage. Bilder werden dabei nicht aus dem Katalog entfernt.
    • Kopieren. Das ausgewählte Element wird in die Zwischenablage kopiert.
    • Einfügen. Fügt ein Element aus der Zwischenablage in das aktuell ausgewählte Element ein, wenn möglich.
    • Löschen. Entfernt das ausgewählte Element. Bilder werden dabei nicht aus dem Katalog entfernt.
    • Gehe zu. Reduziert das Katalog-Sichtfenster auf den Inhalt des derzeit ausgewählten Elements.
    • Zurück. Kehrt zum zuvor dargestellten Inhalt zurück.
    • Grundzustand. Zeigt den vollständigen Katalog.
  • Katalog-Menüfunktionen
    • Neue Gruppe erstellen (). Definiert eine neue Gruppe für Kollektionen, Albums oder Präsentationen. Das erste Element, das der Gruppe zugefügt wird, entscheidet über die Art der Gruppe.
    • Bearbeiten (). Ermöglicht, das ausgewählte Element zu bearbeiten.
    • Löschen (). Entfernt das ausgewählte Element. Bilder werden dabei nicht aus dem Katalog entfernt.
    • Eigenschaften (). Zeigt die Katalog-Eigenschaften.
    • filtern... (). Ruft den Filter-Dialog auf, mit dem unbenutzte oder unerwünschte Katalogeinträge unterdrückt werden können und/oder nur Elemente mit dem ausgewählten Farbcode angezeigt werden können.
    • Alle kollabieren (). Kollabiert alle Einträge im Katalog-Sichtfenster, so dass nur noch die Gruppen sichtbar sind.
  • Mausgesten
    • Einfachklick. Wählt ein Element aus und zeigt seine Bilder in einer Galerie bzw. einem Präsentationseditor.
    • Strg+Einfachklick. Ruft die Bearbeitungsfunktion für das angeklickte Element auf.
    • Doppelklick. Expandiert oder kollabiert das angeklickte Element.
    • Ziehen. Elemente können von einer benutzerdefinierten Gruppe in eine andere benutzerdefinierte Gruppe gezogen werden.
