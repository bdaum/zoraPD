Fortschritts-Sichtfenster

Das Fortschritts-Sichtfenster informiert über Vorgänge, die im Hintergrund laufen. Insbesondere erlaubt es, gerade ausgeführte Aufträge abzubrechen. 

Das Fortschritts-Sichtfenster kann mit einem Doppelklick auf den Fortschrittsindikator im rechten Abschnitt der Statuszeile geöffnet werden. Jedoch.der Fortschrittsindikator nur sichtbar, wenn ein Einzelvorgang ausgeführt wird.

Alternativ kann das Sichtfenster mit Fenster>Sicht öffnen>Sonstiges...>Fortschritt geöffnet werden.
