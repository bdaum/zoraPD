Kabelgebundes Fotografieren

Das kabelgebunde Fotografieren ist ein Aufnahmemodus, in dem die Kamera mit einem Host-Computer (z. B. einem Laptop) verbunden ist. Wenn ein Bild aufgenommen wird, wird es automatisch zum Host-Computer übertragen und aus dem Kameraspeicher gelöscht.

Achtung. Nicht alle Kameras unterstützen kabelgebundenes Fotografieren. Die Kamera muss in der Lage sein, Bilder zu machen, während Sie über USB mit einem Host-Computer verbunden ist.

ZoRaPD unterstützt kabelgebundes Fotografieren für alle Kameras, die die Funktion Datei>Importieren>von Gerät unterstützen. Er kann über die Menüfunktion Datei>Kabelgebundes Fotografieren gestartet werden. Alternativ kann es gestartet werden, indem die Kamera mit dem Host-Computer verbunden wird, wenn die Option Beim Anschluss eines Geräts ... Kabelgebundes Fotografieren starten unter Fenster>Benutzervorgaben>Import>Allgemein eingestellt ist.

Während des kabelgebundes Fotografierens werden die von der Kamera kommenden Bilder zuerst in einen Transferordner abgelegt, bevor sie von ZoRaPD verarbeitet werden. Es ist daher notwendig, dass Sie einen solchen Transferordner (wie unter Katalog>Eigenschaften>Überwachte Ordner definiert) auswählen oder einen neuen erstellen.

Nach der Verarbeitung erscheinen die übertragenen Bilder in der Sammlung Importiert>Letzte Hintergrundimporte () und können von dort betrachtet werden.

Das kabelgebunde Fotografieren kann gestoppt werden, indem die Kamera vom Host getrennt wird oder indem die Funktion Datei>Kabelgebundes Fotografieren erneut aufgerufen wird.
