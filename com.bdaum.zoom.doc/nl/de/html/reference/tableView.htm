Tabellenansicht

Die Tabellenansicht organisiert die Bilder einer Sammlung oder einer Abfrage in eine senkrechte Spalte. Zusätzliche Spalten zeigen ausgewählte Metadaten an.

Die Navigation erfolgt mit Hilfe der Bildlaufleisten.

 Die Breite der Spalten können Sie durch Ziehen der Trennstriche zwischen den individuellen Spaltenüberschriften festgelegen.


Die Sortierung kann durch einen Klick auf eine der Spaltenüberschriften geändert werden. Klicken Sie ein zweites Mal auf die gleiche Spaltenüberschrift, wird zwischen aufsteigender und absteigender Sortierreihenfolge gewechselt. Ein kleiner Pfeil im Spaltenkopf zeigt die Sortierrichtung an.

Durch Klicken auf Konfigurieren>>> in der ersten Spaltenüberschrift (Bildspalte) können Sie die Tabelle konfigurieren, d.h. festlegen, wie viele zusätzliche Spalten angezeigt und was ist in diesen Spalten angezeigt werden soll

Weitere wichtige Befehle und Mausgesten finden Sie in der allgemeinen Hilfeseite für Galerien.
