Tastenkürzel

Diese Seite der Voreinstellungen erlaubt Ihnen die Konfiguration derTastenkürzel. Wählen Sie einfach ein Kommando in der Tabelle (benutzerdefinierte Einträge sind fett gedruckt) aus und geben Sie im Feld Tastenfolge die Tastenkombination in der gleichen Weise an, wie Sie sie später benutzen würden. Spezialtasten wie Löschen, Zurück und Tab lassen sich über die Pfeiltaste rechts neben dem Feld einfügen.
Ist einer Tastenfolge mehr als ein Kommando zugeordnet, so werden die betreffenden Kommandos im Bereich Konflikte angezeigt. Mit einem Doppelklick auf einen solchen Eintrag können Sie zum betreffenden Kommando navigieren, um dem Kommando dann eine andere Tastenfolge zuzuweisen.
