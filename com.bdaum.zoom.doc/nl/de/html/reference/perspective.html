Perspektiven

Perspektiven bestimmen die zunächst sichtbaren Sichtfenster und das Layout der Sichtfenster im Workbench-Fenster. Sie bieten eine Reihe von Funktionen, die darauf abzielen, bestimmte Arbeitsvorgänge zu unterstützen.

Perspektiven öffnen

Um eine neue Perspektive zu öffnen:

  1. Klicken Sie auf Perspektive öffnen auf der Perspektivenleiste rechts oben in der Workbench. (Das gleiche erreichen Sie mit der Menüfunktion Fenster>Perspektive...Perspektive öffnen...)
 
  1. Wählen Sie die Perspektive, die Sie öffnen möchten.

Wechseln zwischen Perspektiven

Offene Perspektiven werden durch Symbole auf der Perspektivenleiste dargestellt. Wenn Sie mehr als eine Perspektive geöffnet haben, können Sie zwischen ihnen wechseln, indem Sie die Symbole auf der Perspektivenleiste klicken.

Ändern des Layouts von Perspektiven

Sie können einfach das Layout der Perspektiven ändern:

  • Ändern Sie die Größe der Sichtfenstern durch Ziehen der Trennleisten zwischen den Sichtfenstern.
  • Verschieben Sie Sichtfenster an einen anderen Ort, indem Sie sie am Reiter "anfassen" und ziehen..
  • Minimieren Sie ein Sichtfenster, indem Sie auf die Minimieren-Schaltfläche in der Kopfzeile des Sichtfensters klicken. Das minimierte Symbol können Sie mit einem Rechtsklick und Auswahl von "Wiederherstellen" wieder am alten Platz auferstehen lassen.
  • Schließen Sie eine Sicht mit einem Klick auf die Schließen-Schaltfläche im Reiter.
  • Öffnen Sie neue Sichtfenster mit der Menüfunktion Fenster>Sicht anzeigen.
Zusätzlich können Sie die in einer Perspektive gezeigten Menüfunktionen und Werkzeugsymbole mit Hilfe der Menüfunktion Fenster>Perspektive...>Perspektive anpassen ... konfigurieren.

Wenn Sie zum anfänglichen Layout einer Perspektive zurückkehren möchten, rufen Sie die Menüfunktion Fenster>Perspektive...>Perspektive zurücksetzen auf.

Speichern einer benutzerdefinierten Perspektive

Wenn Sie eine Perspektive geändert haben, können Sie Ihre Änderungen für die zukünftige Verwendung abspeichern:
  1. Wechseln Sie in die Perspektive, die Sie speichern möchten.
  2. Klicken Sie auf Fenster>Perspektive...>Perspektive speichern unter....
  3. Geben Sie einen neuen Namen für die Perspektive im Feld Name ein.
  4. Klicken Sie auf OK .

Der Name der neuen Perspektive wird zum Menü Fenster>Perspektive...>Perspektive öffnen... hinzugefügt.

Löschen einer benutzerdefinierten Perspektive

Selbstdefinierte Perspektiven können auch wieder gelöscht werden:

  1. Öffnen Sie die Perspektiven-Seite der Benutzervorgaben (Fenster>Benutzervorgaben>Perspektiven*).
  2. Wählen Sie diePerspektive aus, die Sie löschen möchten und klicken Sie auf Löschen.
  3. Klicken Sie auf OK.

*) Auf dem Mac: ZoRa Photo Director>Einstellungen>...
