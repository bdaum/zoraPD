FTP-Konto-Einstellungen

Dieser Dialog wird verwendet, um ein neues FTP-Konto anzulegen oder die Definition eines vorhandenen Kontos zu ändern:

  • Allgemein
    • Kontoname. Der Name des Kontos, wie er in der Benutzeroberfläche erscheint.
    • Host. Der Host-Name des FTP-Konto wie z.B. xyz.de oder abc.com.
    • Zielverzeichnis. Das Verzeichnis auf dem Host, in dem die hochgeladenen Elemente gespeichert werden sollen.
    • Export aufzeichnen. Der FTP-Export wird in den Metadaten des hochgeladenen Bildes vermerkt. Später kann die Aufzeichnung für jedes Bild im Sichtfenster Eigenschaften unter  Datei>Track eingesehen werden.
    • Login. Der Benutzername für den FTP-Account.
    • Passwort. Das Passwort für diesen Benutzernamen
    • Konto. Ein Unterkonto falls verfügbar.
    • Anonym. Markieren Sie dieses Feld für den anonymen Zugang. Login, Passwort und Konto sind in diesem Fall nicht erforderlich.
  • Verbindung
    • Passiv-Modus. Standardmäßig eingeschaltet. Der Client ist dafür verantwortlich, die Verbindung herzustellen. Wird von den meisten Servern unterstützt. Andernfalls, im aktiven Modus, ist der Server für den Verbindungsaufbau verantwortlich.
    • Port. Der für FTP verwendet Kommunikations-Port. Normalerweise ist dies Port 21 (Standard).
  • Web
    • Host. Die Angabe eines Web-URL für den Zugriff auf die Webpräsenz des FTP-Kontos. Dies erlaubt, nach dem Upload sofort einen Web-Browser aufrufen, um die hochgeladenen Inhalte anzuzeigen.
    • Präfix. Ein bestimmtes Verzeichnis auf dem FTP-Server, wenn die Webpräsenz nicht das ganze FTP-Konto umfasst sondern nur das im Präfix angegebene Verzeichnis. Wenn angegeben, muss das Zielverzeichnis (siehe oben) mit diesem Präfix beginnen.
    • Notizen. Beliebige Notizen.
