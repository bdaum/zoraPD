Einstellungen für automatische Aktionen

Beziehungen zwischen Bildern

    • Verknüpfungen zwischen RAW- und DNG-Dateien automatisch erstellen. Wenn Sie sowohl RAW- als auch konvertierte DNG-Dateien importieren, ist es möglich, beide als (Original <-> Derivative) miteinander zu verknüpfen, was dann später im Hierarchie-Sichtfenster angezeigt werden kann. Voraussetzung ist, dass die Kurznamen der Dateien (ohne Erweiterung) identisch sind. Sie können diese Erkennung auf Dateien im selben Ordner beschränken, oder aber über Ordnergrenzen hinweg erlauben.

    • Abgeleitete Bilder automatisch aus EXIF-Daten erkennen. Ist diese Option eingeschaltet, so können Beziehungen zwischen Bildern automatisch erkannt werden, auch wenn diese Bilder nicht im RAW- oder DNG-Format vorliegen. Zur Erkennung werden das Aufnahmedatum und andere wichtige Bilddaten herangezogen.

    • Original-XMP-Daten auch auf abgeleitete Bilder anwenden. Ist diese Option eingeschaltet, so werden XMP-Daten des Originalbildes auch auf abgeleitete Bilder angewandt, wenn für diese keine speziellen XMP-Dateien vorhanden sind.

    Automatische Erstellung von Kollektionen

    Durch Hinzufügen von Regeln zu dieser Seite können Sie Kollektionen im Katalog automatisch erstellen, wenn Sie Bilder importieren. Für jedes Bild werden die Metadaten analysiert. Wenn eine Regel zutrifft, wird eine Kollektion, welche das Bild (und andere mit entsprechenden Metadaten) enthalten wird, erstellt - sofern eine solche Kollektion noch nicht vorhanden ist.

    Diese Seite zeigt eine Übersicht über alle definierten Regeln. Sie können die Regeln bearbeiten oder entfernen oder neue Regeln hinzufügen. Details zur Regel-Definition finden Sie hier.
