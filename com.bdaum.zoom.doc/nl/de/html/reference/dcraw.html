Externes DCRAW-Modul einbinden

DCRAW ist die in ZoRa PhotoDirector verwendete Raw-Engine. DCRAW wird kostenlos von Dave Coffin als Quelltext zur Verfügung gestellt. Mehrere Websites stellen binäre Module zur Verfügung. Da immer neue Digitalkameras immer neue RAW-Formate in die Welt setzen, erscheinen auch immer wieder neue Versionen von DCRAW.

Windows

Die Windows-Version von ZoRa PhotoDirector enthält eingebaute DCRAW-Module sowohl für 32- als auch für 64-Bit Plattformen. Mit jeder neuen Version von ZoRa PhotoDirector werden auch diese DCRAW-Module aktualisiert.

Neuere DCRAW-Module können von folgenden Web-Adressen heruntergeladen werden:
  • www.fastpictureviewer.com/downloads/#links
  • http://www.centrostudiprogressofotografico.it/en/dcraw/

Um ein eigenes DCRAW-Modul in ZoRa Photo Director zu installieren, bringen Sie es einfach in den Unterordner dropins/ im ZoRa Photo Director-Installationsordner. Eine andere Möglichkeit ist, den Speicherort des DCRAW-Moduls unter Fenster>Benutzervorgaben>Import>RAW-Konvertierung anzugeben.

Linux

Unter Linux verwendet ZoRa PhotoDirector das im Linux-System installierte DCRAW-Modul. Wenn Sie Ihre Plattform immer up-to-date halten, sollten Sie auch immer über eine aktuelle Version von DCRAW verfügen.

Mac OSX

Die Mac-OSX-Version von ZoRa PhotoDirector enthält ein eingebautes DCRAW-Modul. Dies ist allerdings nicht mehr aktuell (Version 9.16). Neuere Version von DCRAW-Binaries für Mac OSX findet man auf dem Web. Jedoch laufen diese Versionen in der Regel nicht unter allen OSX-Versionen. Da die OSX-Bibliothekskonventionen geändert wurden, können Module, die auf einem neuen System kompiliert wurden, auf älteren Systemen oft nicht ausgeführt werden. Man muss also probieren, was passt.

Die beste Wahl ist wahrscheinlich, von Dave Coffin's Website den aktuellen Quelltext von DCRAW herunterzuladen und selbst zu kompilieren. Dazu sind die folgenden Schritte notwendig:

  • Das XCode IDE von Apple herunterladen und installieren (kostenlos)
  • Die Command line tools von innerhalb des XCode IDE installieren.
  • Das folgende Kommand ausführen:  
    gcc -o dcraw -O4 dcraw.c -lm -DNODEPS

Neue DCRAW-Module können einfach in den Unterordner dropins/ im ZoRa Photo Director- Installationsordner gebracht werden. Alternativ kann auch der Speicherort des Moduls unter ZoRa Photo Director>Einstellungen>Import>RAW-Konvertierung angegeben werden.
