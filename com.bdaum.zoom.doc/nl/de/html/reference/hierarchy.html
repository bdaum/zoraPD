Hierarchie-Sichtfenster

Das Hierarchie-Sichtfenster wird in vier verschiedenen Spielarten verwendet (Originale-Sichtfenster, Derivate-Sichtfenster, Composites-Sichtfenster und Komponenten-Sichtfenster) und zeigt die Beziehung zwischen Bildern. Folgende Beziehungen können dargestellt werden:

  • Das Derivate-Sichtfenster zeigt Bilder, die aus einem anderen Bild abgeleitet wurden. Beispielsweise kann ein TIFF-Bild von einem RAW-Bild abgeleitet werden, oder ein zugeschnittenes Bild wird von einem Bild voller Größe abgeleitet. Aus Galerien wird diese Funktion mit Kontext-Funktion Derivate zeigen aufgerufen.
  • Das Originale-Sichtfenster zeigt Originale, von denen ein bestimmtes Bild abgeleitet wurde. Aus Galerien wird diese Funktion mit der Kontext-Funktion Originale zeigen aufgerufen.
  • Das Composites-Sichtfenster zeigt Composites, für die bestimmte Bilder als Komponente verwendet wurden, z. B. Panoramen, HDR-Bilder, Focus-Stacking, Fotomontagen, etc. Aus Galerien wird diese Funktion mit Kontext-Funktion Composites zeigen aufgerufen.
  • Das Komponenten-Sichtfenster Komponenten, die zu einem bestimmten Composite gehören. Aus Galerien wird diese Funktion mit der Kontext-Funktion Komponenten zeigen aufgerufen.

Die folgenden Funktionen stehen in diesen Sichtfenstern zur Verfügung:

  • Drag & Drop
    • Das Ziehen einer Miniaturansicht aus einer Galerie auf eine Miniaturansicht in Hierarchie-Sichtfenster fügt dieses Bild in die aktuellen Hierarchie ein - das Bild ist mit dem Zielbild nun "verbunden".
  • Kontext-Funktionen
    • Beschreibung bearbeiten. Hier können Sie Details zu der Beziehung zwischen zwei Bildern eingeben, beispielsweise mit welchen Transformationen ein Bild von einem anderen abgeleitet wurde.
    • Aufheben der Verknüpfung. Das ausgewählte Bild wird aus der Hierarchie entfernt.
    • Kandidaten für Originale/Derivate anzeigen. Zeigt eine Kollektion von Bildern mit dem gleichen Originaldateinamen und einem Modifikationsdatum, dass älter / jünger als das des aktuell ausgewählten Bildes ist. 
    • Vollbild anzeigen. Damit wird ein externer Bildbetrachter aufgerufen. Ist kein externer Bildbetrachter definiert, wird der interne Bildbetrachter aufgerufen. Durch das gleichzeitige Drücken der  Umschalt- oder einer anderen Steuertaste kann der interne Bildbetrachter erzwungen werden. Je nach Steuertaste sind dabei verschiedene Modi wählbar. 
    • Bearbeiten. Der Standard-Editor der Bilddatei wird aufrufen. Wenn mehrere Bilder ausgewählt sind, wird der Editor nur aufgerufen, wenn er alle beteiligten Dateitypen unterstützt.
    • Bearbeiten mit. Damit können Sie unter allen Editoren, die dem Bilddateityp zugeordnet sind, auswählen. Sind mehrere Bilder ausgewählt, werden nur Editoren, die alle beteiligten Dateitypen unterstützen, angezeigt.
    • Zeit-Suche. Sucht nach Bildern, die sich im gleichen Zeitfenster wie das ausgewählte Bild befinden. Die Größe des Zeitfensters kann angegeben werden.
    • Ähnlichkeitsrecherche. Sucht inhaltsbasiert nach Bildern, die dem ausgewählten Bild (oder einer Zeichnung) visuell ähnlich sind. 
    • Umkreissuche. Sucht nach weiteren Bildern, die im Umkreis der selektierten Bilder liegen. Eine maximale Entfernung kann angegeben werden.
    • Sprachnotizen>
      • Sprachnotiz hinzufügen Damit wird ein Dialogfeld angezeigt, das es Ihnen ermöglicht, sofort eine Sprachnotiz aufzeichnen oder eine vorhandene Audiodatei als Sprachnotiz anzufügen.
      • Sprachnotizen entfernen. Dadurch werden früher hinzugefügte Sprachnotizen entfernt. Wenn die Sprachnotiz beim Anlegen aufgenommen worden war, wird auch die Audio-Datei gelöscht.
      • Sprachnotiz abspielen. Damit wird die Sprachnotiz des selektierten Bildes abgespielt.
    • Metadaten>
      • Metadaten kopieren. Kopiert die Metadaten des ausgewählten Bildes in die Zwischenablage.
      • Metadaten hinzufügen. Kopiert die Metadaten von der Zwischenablage in die selektierten Bilder. Ein Dialog ermöglicht die Auswahl spezifischer Metadaten.
    • Farbcode. Ändert die Farbkodierung der ausgewählten Bilder.
    • Bewertung. Ändert die Bewertung der ausgewählten Bilder.
    • Drehen im und gegen den Uhrzeigersinn. Dreht die Miniaturansicht von 90 nach rechts oder links..
    • Ordner anzeigen. Öffnet den externen Ordner, in dem sich das Bild befindet, und wählt diesen Ordner auch im Katalog-Sichtfenster aus.
    • Aktualisieren. Aktualisiert Kataloginformationen der selektierten Bilder oder einer selektierten Kollektion.
    • Löschen. Löscht die ausgewählten Bilder aus dem Katalog. Optional kann die Bilddatei auch vom Datenträger gelöscht werden.
  • Menüfunktionen des Sichtfensters
    • Vollbild anzeigen. Damit wird ein externer Bildbetrachter aufgerufen. Ist kein externer Bildbetrachter definiert, wird der interne Bildbetrachter aufgerufen. Durch das gleichzeitige Drücken der  Umschalt- oder einer anderen Steuertaste kann der interne Bildbetrachter erzwungen werden. Je nach Steuertaste sind dabei verschiedene Modi wählbar. 
    • Bearbeiten. Der Standard-Editor der Bilddatei wird aufrufen. Wenn mehrere Bilder ausgewählt sind, wird der Editor nur aufgerufen, wenn er alle beteiligten Dateitypen unterstützt.
    • Bearbeiten mit. Damit können Sie unter allen Editoren, die dem Bilddateityp zugeordnet sind, auswählen. Sind mehrere Bilder ausgewählt, werden nur Editoren, die alle beteiligten Dateitypen unterstützen, angezeigt.
    • Originale/Derivative/Komponenten/Composites zeigen... Diese Funktion schaltet zwischen dem Originale-Sichtfenster und dem Derivate-Sichtfenster, oder zwischen dem Composites-Sichtfenster und dem Komponenten-Sichtfenster. Dabei wird das gerade selektierte Bild an die Spitze der betreffenden Hierarchie gestellt.
    • Zeige Hierarchie für selektiertes Bild. Zeigt die aktuelle Hierarchie (Originale, Derivative, Composites oder Komponenten) für das aktuell in einer Galerie ausgewählte Bild.
  • Werkzeugtasten des Sichtfensters
    • Vollbild anzeigen (). Damit wird ein externer Bildbetrachter aufgerufen. Ist kein externer Bildbetrachter definiert, wird der interne Bildbetrachter aufgerufen. Durch das gleichzeitige Drücken der  Umschalt- oder einer anderen Steuertaste kann der interne Bildbetrachter erzwungen werden. Je nach Steuertaste sind dabei verschiedene Modi wählbar. 
    • Bearbeiten (). Der Standard-Editor der Bilddatei wird aufrufen. Wenn mehrere Bilder ausgewählt sind, wird der Editor nur aufgerufen, wenn er alle beteiligten Bildtypen unterstützt.
    • Bearbeiten mit ().. Damit können Sie unter allen Editoren, die mit dem Bilddateityp zugeordnet sind, auswählen. Sind mehrere Bilder ausgewählt, werden nur Editoren, die alle beteiligten Bildtypen unterstützen, angezeigt.
    • Originale () / Derivative () / Komponenten () / Composites () zeigen... Diese Funktion schaltet zwischen dem Originale-Sichtfenster und dem Derivate-Sichtfenster, oder zwischen dem Composites-Sichtfenster und dem Komponenten-Sichtfenster.   • Mausgesten
    • Doppelklick. Expandiert oder kollabiert einen Baumknoten.
    • Strg + Klick. Die Funktion Beschreibung bearbeiten wird aufgerufen.
  • Tastatur-Aktionen
    • Eingabe. Identisch ist mit Kontext-Funktion Bearbeiten.
    • Umschalt + Eingabe. Identisch ist mit Kontext-Funktion Bearbeiten mit
    • Tab. Damit wird ein externen Bildbetrachter aufgerufen (siehe Allgemeine Einstellungen). Ist kein externer Bildbetrachter definiert, wird der interne Bildbetrachter aufgerufen.
    • Tab + Umschalt. Erzwingt den internen Bildbetrachter.
    • Entf. Löscht die ausgewählten Bilder
    • 0,1,2,3,4,5.. Bewertet die ausgewählten Bilder mit 0,1,2,3,4,5 Sternen.
