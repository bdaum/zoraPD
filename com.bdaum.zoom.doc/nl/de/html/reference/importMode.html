Importmodus

Beim Importieren von Bildern haben Sie die Wahl zwischen zwei Strategien:
  • Import vom ursprünglichen Ort. Mit dieser Option bleiben alle importierten Dateien am ursprünglichen Speicherort. Für jede dieser Dateien wird ein Katalogeintrag erstellt, der auf diesen Speicherort verweist.
    Dies ist sogar für Wechselmedien möglich. Wenn das Medium entfernt wird, wird der Katalogeintrag als Offlineeintrag angezeigt, aber alle Vorgänge, für die das Bild in hoher Auflösung erforderlich ist, schlagen fehl. Durch erneutes Montieren des Mediums wird dieses Problem behoben.
    Ein weiteres Problem in diesem Modus ist die (unwahrscheinliche) Erstellung von DNG-Dateien während des RAW-Dateiimports aus schreibgeschützten Medien. Dies ist nicht möglich und wird eine Fehlermeldung erzeugen.
  • Import in neue Ordnerstruktur. Wenn Sie diese Option auswählen, werden Sie aufgefordert, den Zielordner anzugeben. Alle importierten Dateien werden in diesen Zielordner kopiert. Die Dateikopien können je nach Ihrer Wahl in Unterordner des Zielordners organisiert werden.
    Darüber hinaus können alle Dateien umbenannt werden, indem eine geeignete Umbenennungsvorlage ausgewählt wird. Wenn keine Umbenennung gewünscht wird, wählen Sie "Originaldateiname".

Bitte beachten Sie, dass beim Import von einem Gerät der Import immer in eine neue Ordnerstruktur erfolgt.
