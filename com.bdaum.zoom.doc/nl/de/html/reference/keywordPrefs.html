Schlagwort-Einstellungen

Die Liste der Filterausdrücke auf dieser Seite bestimmt, welche Schlagworte zur Liste der definierten Schlagworte hinzugefügt werden (in den Katalogeigenschaften), wenn Bilder in den Katalog importiert werden. Einige fotografishce Tools fügen beispielsweise den Schlagworten GPS-Koordinaten hinzu. Hinzufügen von aller dieser Schlagworte würde zu einer riesigen und unüberschaubaren Sammlung von Schlagworten führen.

Wenn ein neues Schlagwort der Schlagwortliste hinzugefügt werden soll, werden die Filterausdrücke einer nach der anderen mit dem neuen Schlagwort verglichen. Bei einer Übereinstimmungen wird das Stichword entweder abgelehnt oder angenommen, je nach der Definition des Filterausdrucks. Besteht keine Übereinstimmung, so wird das Schlagwort mit dem nächsten Filterausdrucks verglichen usw. Trifft überhaupt kein Schlagworteintrag zu, wird das Schlagwort akzeptiert.
Filterausdrücke können Platzhaltersymboler (wildcards) enthalten:
  • ' * ' entspricht einer Zeichenfolge beliebiger Länge 
  • '? ' entspricht genau einem Zeichen
