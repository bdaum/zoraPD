Wartung

Die Hauptmenüleiste enthält mehrere Funktion für die Katalogwartung:

  • Katalog>Wartung...>Katalog aufteilen...  kann eine Selektion von Bildern in einen neuen, separaten Katalog exportieren und diese Bilder aus dem aktuellen Katalog entfernen.
    Beispiel: Der aktuelle Katalog enthält Bilder von mehreren Fotografen. Sie möchten den Katalog je nach Fotografen aufspalten. Dazu starten Sie eine Adhoc-Suche (Metadata>Adhoc search...) mit IPTC>Künstler. Dann rufen Sie Katalog>Wartung...>Katalog aufteilen.. auf. Sie können auch eine Untermenge der angezeigten BIlder ausgliedern, indem Sie die fraglichen Bilder selektieren.
  • Katalog>Wartung...>Kataloge zusammenführen... kann mehrere Kataloge zu einem kombinieren. Für die Behandlung von Duplikaten stehen mehrere Optionen zur Verfügung.
  • Katalog>Wartung...>Index neu erstellen erzeugt den Lucene-Index komplett neu. Dieser Index wird für die Ähnlichkeitssuche und die Textrecherche benutzt. Verwenden Sie diese Funktion nur, wenn Sie Grund zur Annahme haben, dassdie Ähnlichkeitssuche und die Textrecherche falsche Suchergebnisse liefern.
  • Katalog>Wartung...>Wiederherstellen... kann verwendet werden, um einen Katalog aus Sicherungsdaten wieder herzustellen. Sie können sogar einen anderen als den aktuellen Katalog wiederherstellen. Im Anschluss wird der wiederhergestellte Katalog geöffnet.
  • Katalog>Wartung...>Migrieren...kann verwendet werden, um einen Katalog auf eine andere Plattform (z.B. Windows nach Linux) zu portieren. Dabei können auch die Ordnerstrukturen geändert werden. Sie benötigen allerdings Kenntnisse in regulären Ausdrücken, um diese Funktion anwenden zu können.
