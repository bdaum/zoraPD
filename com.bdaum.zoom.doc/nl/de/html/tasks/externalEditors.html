Der Aufruf von externen Editoren und Betrachtern


ZoRa Photo Director konzentriert sich ausschließlich auf die Bildverwaltung und -suche und überlässt das Bearbeiten von Bilder spezialisierten Bildeditoren wie Photoshop, Paint Shop Pro, Picture Window und ähnlichen Programmen..

Mit Hilfe der Tastenkombination Strg+E können Sie jedoch ausgewählte Bilder schnell in einem Editor öffnen. Vorher müssen Sie jedoch die Editoren den entsprechenden Dateitypen zuordnen (Fenster>Benutzervorgaben>Dateizuordnungen*). Jeder Dateityp kann mit einem anderen Editor assoziiert werden. Zum Beispiel können RAW-Dateien automatisch mit einem RAW-Konverter geöffnet werden, TIFF-Dateien dagegen mit einem klassischen Bild-Editor. Sie können auch mehrere Editoren dem gleichen Dateityp zuweisen und dann durch Betätigen von Strg+Umschalt+E einen dieser Editoren aus einer Liste auswählen.

Beim Doppelklick auf ein Bild wird dieses  in einem Vollbild-Bildbetrachter geöffnet. Wenn Sie keinen externen Bildbetrachter definiert haben, wird der interne Viewer verwendet, der alle unterstützten Dateitypen öffnen kann. Sie können jedoch auch unter Fenster>Benutzervorgaben>Allgemeines* ihren Lieblingsbildbetrachter eintragen. Trotzdem ist es noch möglich, auch den internen Betrachter aufzurufen, indem Sie beim Doppelklick die Umschalt-Taste gedrückt halten.


*) Auf dem Mac: ZoRa Photo Director>Einstellungen>...
