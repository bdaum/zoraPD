Web-Galerien

Web-Galerien sind eine weitere Option, um Bilder auf dem Internet zu veröffentlichen - vorausgesetzt Sie haben Zugang zu einer Website. 


Sie beginnen mit dem Anlegen einer lokalen Web-Galerie mit einem oder mehreren Storyboards. Dazu legen zunächst eine neue Gruppe and (oder benutzen eine existierende Gruppe, welche bereits Web-Galerien enthält) mit Hilfe der Funktion Neue Gruppe erstellen () vom Menü des Katalog-Sichtfensters. Mit einem Rechtsklick auf diese Gruppe rufen Sie die Kontextfunktion Neue Präsentation erstellen>Web-Galerie... () auf. Sie erhalten dann einen Dialog zur Eingabe der Eigenschaften der neuen Web-Galerie.

Nachdem Sie alle Bilder auf die Storyboards platziert und die verschiedenen Parameter für Galerie, Storyboards, und einzelne Bilder eingegeben und eine Web-Engine ausgewählt haben, können Sie die Web-Galerie einsetzen. Dazu kann die Web-Galerie entweder zunächst in einen lokalen Ordner exportiert werden (um sie später zur Website hochzuladen) oder aber direkt zu einem FTP-Konto hochgeladen werden.
