Erste Hilfe

  • Sollten Sie Probleme mit der Grafik haben - besonders bei sehr großen Bildern - versuchen Sie die Option Hochwertige Interpolation verwenden im Fenster>Benutzervorgaben>Allgemeines* auszuschalten.

  • Bei Performanzproblemen ziehen Sie bitte das Kapitel Tuning zu Rate.

  • Wenn Sie Nachrichten erhalten, die auf Probleme mit der Lucene-Index hindeuten, oder wenn Sie glauben, dass die Ähnlichkeitssuche, die Textsuche oder die Dublettensuche nicht funktionsfähig ist, gehen Sie wie folgt vor:
 Rufen Sie Katalog>Wartung>Index neu erstellen auf.
  • Ihre Kamera wird beim RAW-Import nicht unterstützt? Möglicherweise müssen Sie die neueste Version von DCRAW installieren. Sie erhalten diese unter http://www.insflug.org/raw/Downloads/. Nach dem Download kopieren Sie dcraw.exe in den Unterordner dropins/ des Installationsordners, zum Beispiel nach C:\Programme\ZoRaPD\dropins.

  • Wenn das Überblenden von Bildern in Diavorführungen und im Bildbetrachter unter Linux nicht funktioniert, stellen Sie sicher das ein Compositing Manager (z.B. compiz-fusion) installiert ist, und dass die Desktop-Effekte eingeschaltet sind.

  • Unter Windows 8 schlägt der Start von ZoRaPD gelegentlich fehl, wobei das Galerie-Sichtfenster die Fehlermeldung "Illegal thread access..." zeigt. Starten Sie in diesem Fall Ihren Computer neu.


*) Auf dem Mac: ZoRa Photo Director>Einstellungen>...
