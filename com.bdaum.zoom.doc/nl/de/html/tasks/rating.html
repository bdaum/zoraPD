Bewerten von Bildern

Mit der Bildbewertung oder dem Ranking können Sie Ihre Bilder nach ihrer Qualität klassifizieren. Wie Qualität definiert wird, liegt bei Ihnen: technische Qualität, ästhetische Qualität, kommerzielle Qualität usw. Branchenstandard sind sechs Bewertungsstufen von 0 bis 5.

Bewertete Bilder importieren

Wenn Sie Bilder in einen Katalog importieren, wird auch deren Bewertung importiert. Die Bewertung kann in der Bilddatei oder in einer XMP-Sidecar-Datei definiert sein.

Ein einzelnes Bild bewerten

Die naheliegendste Möglichkeit, ein einzelnes Bild zu bewerten, ist das Klicken auf das Bewertungssymbol in der oberen rechten Ecke des Miniaturbilds. Eine andere Möglichkeit besteht darin, das Bild auszuwählen und das Menü "Bewerten ..." oder die Kontextfunktion aufzurufen. Oder ziehen Sie das Bild auf eine Bewertungskollektion im Katalog-Sichtfenster. Schließlich können Sie auch eine Tastenkombination drücken, standardmäßig Strg + Umschalt + T.
Alle diese Aktionen rufen das Bewertungsdialogfeld auf, in dem Sie entweder auf ein Bewertungssymbol klicken oder eine der Tasten 0, 1, 2, 3, 4, 5 oder * drücken (* für automatisierte Bewertung).

Viele Bilder zusammen bewerten

Das Anwenden desselben Ranges auf mehrere Bilder kann fast auf die gleiche Weise erfolgen, mit Ausnahme des Symbols für die Miniaturansicht. Wählen Sie einfach mehrere Bilder aus, rufen Sie dann die Menü- oder Kontextfunktion "Bewerten ..." auf oder drücken Sie die Tastenkombination für die Bewertung. Oder ziehen Sie die Bilder auf eine Bewertungskollektion im Katalog-Sichtfenster.

Viele Bilder einzeln bewerten

Am bequemsten ist es, wenn Sie eine Diashow mit den ausgewählten Bildern ausführen. Drücken Sie gleichzeitig eine der Tasten 0, 1, 2, 3, 4 oder 5. Die Diaschau wird nach kurzer Zeit automatisch fortgesetzt. Alternativ können Sie auch die Maus bewegen, um das Diashow-Kontrollfeld aufzurufen und das Bewertungsdialogfeld von dort aus aufzurufen.
