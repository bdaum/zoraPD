Suchen und Finden von Bildern

Es gibt viele Wege nach Bildern zu suchen:
  • Kollektionen können Bilder unter verschiedenen Aspekten gruppieren. Kollektionen werden im Katalog aufgelistet. Einige Kollektionen werden vom System definiert, während andere Kollektionen vom Benutzer definiert werden. Zum Beispiel könnten Sie eine Kollektion anlegen, die alle Bilder im Hochformat mit dem Schlagwort "Wolkenkratzer" enthält. In der Regel werden Kollektionen für die Bildabfragen, die Sie häufig verwenden, angelegt. Die Bildauswahl durch Kollektionen kann durch eingeengt werden. Dies kann sowohl innerhalb von Systemkollektionen als auch innerhalb von benutzerdefinierten Kollektionen erfolgen. Zum Beispiel könnten Sie innerhalb der 5-Sterne-Kollektion eine Unterkollektion anlegen, die alle Bilder mit Farbcode "rot" enthält.

  • Suchabfragen werden in der Regel dann verwendet, wenn die betreffende Bildauswahl nur einmal verwendet werden soll:
    • Bei der Adhoc-Suche () werden die Suchkriteria in der gleichen Weise wie bei den Kollektionen definiert, jedoch wird diese Definition normalerweise nicht in den Katalog übernommen.
    • Die Schlagwortsuche () ist ein schneller Weg für die Suche nach einzelnen Schlagwörtern oder Schlagwortkombinationen. 
    • Die Zeitsuche () ermöglicht die Suche nach allen Bilder innerhalb der gleichen Zeitspanne wie die aktuell ausgewählten Bilder. Dies ist eine hervorragende Möglichkeit, um alle Bilder aus einer Serie schnell zu finden.. 
    • Die Umkreissuche () ermöglicht die Suche nach allen Bilder in der Nähe der vorgegebenen Bilder - vorausgesetzt, dass diese Bilder mit Geo-Tags versehen wurden. 
    • Die Ähnlichkeitssuche () wird für die Suche nach visuell ähnlichen Bildern verwendet. Dies kann auf dem einem vorgegebenen Bild basieren, jedoch es ist auch möglich, ein Beispiel-Bild zu malen. Verschiedene Ähnlichkeitsalgorithmen können für die Ausführung der Suche ausgewählt werden.
    • Die Dublettensuche () wird für die Suche nach Duplikaten verwendet. Dies kann  auf Grund des Bildnamens, der EXIF-Daten oder der visuellen Ähnlichkeit (siehe oben) geschehen. 
    • Die Suche nach Bildserien () erfolgt auf Grund von Erstellungsdatum und -zeit eines Bildes. Weitere Optionen erlauben eine zusätzliche Eingrenzung des Suchbereichs.
    • Die Suche verwaisten Einträgen () ermittelt Katalogeinträge, deren Bilddatei abhanden gekommen ist.
    • Die Volltextsuche () sucht in allen mit einem Bild verbundenen Textelementen: Schlagworte, Titel, Beschreibung, Standorte, Kontakte und Kunstwerke. Auch die Titel und die Beschreibung abgeleiteter Dias, Ausstellungs- und Web-Exponate werden berücksichtigt. Diese Suche erlaubt komplexe Suchausdrücke und Platzhaltersymbole (wildcards).
Suchabfragen können im Gesamtkatalog ausgeführt werden, oder aber auf die aktuell ausgewählte Kollektion begrenzt werden. Durch Betätigung der Gallerie-Menüfunktion Abfrage als Kollektion speichern () kann die Abfrage auch nachträglich in eine Kollektion umgewandelt werden. Auf diese Weise können auch Kollektionen definiert werden, die auf der Ähnlichkeitssuche oder der Volltextsuche beruhen.
  • In den Eigenschaften-, EXIF-, IPTC- und Metadatensichtfenstern können durch Betätigung des Schaltfelds Ähnliche Bilder suchen () weitere Bilder mit übereinstimmenden Attributen gesucht werden. Wird zusätzlich die Umschalt-Taste betätigt, wird die Suche auf die akutelle Kollektion begrenzt.
