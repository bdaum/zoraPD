Verschlagwortung

Ein häufiges Problem beim Definieren von Schlagworten für eine Reihe von Bildern ist die Schlagwortinflation verursacht durch verschiedene Wortformen, Rechtschreibfehler oder Verwendung von Synonymen. Zum Beispiel können die Schlagworte "Kinder", "Kind", "Kindre" oder "Gör" alle das Gleiche bedeuten.

Zur Vermeidung der Schlagwortinflation bietet der Dialog zur Schlagwortdefinition ein paar Hilfsmittel:

  • Eine Liste der zuletzt verwendeten Schlagworte. Ein Schlagwort aus dieser Liste erspart Ihnen Einiges an Tippen oder Blättern, und Sie können sicher sein, dass das ausgewählte Schlagwort bereits vorhanden ist.
  • Eine Liste der verfügbaren Schlagwort. Gehen Sie durch diese Liste und wählen Sie die gewünschten Schlagworte von dort. Auch hier können Sie sicher sein, dass das ausgewählte Schlagwort bereits vorhanden ist.
  • Die Vorschlagen...-Schaltfläche. Durch Klicken auf diese Schaltfläche wird eine Liste mit vorgeschlagenen Schlagworten basierend auf visueller Ähnlichkeit zusammengestellt. In dieser Liste finden Sie möglicherweise das richtige Schlagwort für Ihren Zweck.
  • Nur wenn Sie das richtige Schlagwort in einer dieser drei Listen nicht finden, geben Sie ein neues Schlagwort in dem dafür vorgesehenen Feld ein. Wir empfehlen die Installation eines Wörterbuchs, um die Rechtschreibprüfung zu aktivieren.
  • Schauen Sie sich dann und wann die Liste der im Katalog definierten Schlagworte an. Gegebenenfalls möchten diese Liste bereinigen und Schlagworte löschen, bearbeiten oder ersetzen.

Einige Tipps für die Schlagwortauswahl (inspiriert von Robert Kneschke):

  • Verwenden Sie sowohl abstrakte als auch konkrete Schlagworte. Zum Beispiel wenn ein Bild ein Tempolimit-Schild zeigt, fügen Sie die Schlagworte "Verkehrsschild" und "Tempolimit" ein, vielleicht auch noch "Tempo 30"..
  • Anstelle eines Adjektivs verwenden Sie ein Substantiv, z. B. anstelle von "schön" verwenden "Schönheit".
  • Wenn Sie Ihre Bilder über einen Agenten verkaufen möchten, fordern Sie von diesem eine Liste der Schlagworte, die verkaufen, und importieren diese Liste in Ihren Katalog.
  • Auf dem Web gibt es einige Hilfsmittel, mit denen man herausfinden kann, wie andere ähnliche Bilder verschlagworten und welche Schlagworte gut oder schlecht sind:
    • Stocktagger schlägt ähnliche Schlagworte basierend auf dem Vokabular von Getty Images und Istockphoto vor.
    • Photokeywords und Findphotokeywords suchen im Web nach ähnlichen Bildern und zeigt deren Schlagworte.
    • PicNiche analysiert die Qualität von Schlagworten und schlägt ähnliche Schlagworte vor.
