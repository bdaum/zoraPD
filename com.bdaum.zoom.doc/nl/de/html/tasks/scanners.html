Mit Scannern arbeiten

ZoRa Photo Director besitzt keine Funktion zum Import von TWAIN-Scannern. Viele Fotografen verwenden sowieso nicht den Standard-TWAIN-Treiber, sondern qualitativ bessere Scan-Werkzeuge wie SilverFast oder VueScan.

Falls Sie den Import von gescannten Bildern automatisieren möchten, können Sie so vorgehen, dass Sie immer in das gleiche Verzeichnis scannen, gleichzeitig dieses Verzeichnis als überwachten Ordner konfigurieren (Katalog>Eigenschaften>Überwachte Ordner).
