Vergleichen von Bildern

Um zwei Bilder nebeneinander zu vergleichen, selektieren Sie beide Bilder in einer Galerie. Drücken Sie dann TAB oder F4, oder verwenden Sie das Kontextmenü, um den internen Bildbetrachter aufzurufen (externe Bildbetrachter werden nicht zum Vergleichen von Bildern verwendet). Die Bilder werden nebeneinander angezeigt. Auf jedem Bild können Sie  Mausbewegungen oder Tastaturaktionen zum Schwenken und Zoomen anwenden, wie für den internen Bildbetrachter beschrieben. Durch Drücken von F4 können Sie in den synchronisierten Modus wechseln, in dem jede Zoom- oder Schwenkaktion gleichzeitig auf beide Bilder angewendet wird. Um in den unsynchronisierten Modus zurückzukehren, drücken Sie erneut F4.
