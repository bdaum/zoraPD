Tuning


Es gibt mehrere Stellen, an denen man das System für eine höhere Geschwindigkeit optimieren kann:

Anzeige

Fenster>Einstellungen>Allgemeines*
  • Anzeige
    • Ausgabe-ICC-Profil. Auf CMS deaktivieren schalten, um die Darstellung von Bildern und Miniaturansichten zu beschleunigen.
    • Hochwertige Interpolation verwenden. Schalten Sie diese Option aus, um die Darstellung von Bildern und Miniaturansichten zu beschleunigen. Es kann allerdings zu einer leichten Pixelierung der Bilder kommen.
    • Im internen Betrachter wenn möglich vor jedem hochauflösenden Bild ein Vorschaubild zeigen.  Schalten Sie diese Option aus, um Bilder schneller im Vollbildmodus anzuzeigen.
    • JPeG-Artefakte der Miniaturansichten im internen Bildbetrachter kaschieren. Schalten Sie diese Option aus, um Bilder schneller im Vollbildmodus anzuzeigen.

Fenster>Einstellungen>Darstellung*
  • Farbschema. Wählen Sie Plattform, um Sichtfenster und Dialoge schneller aufzubauen.

Import

Katalog>Eigenschaften...

  • Allgemeines
    • Zeitlinie automatisch erstellen.  Wenn Sie nicht an einer Zeitlinie interessiert sind, schalten Sie diese Option aus, um den Importprozess zu beschleunigen.
    • Ortskollektionen automatisch erstellen.  Wenn Sie nicht an Ortskollektionen interessiert sind, schalten Sie diese Option aus, um den Importprozess zu beschleunigen.
  • Miniaturansichten
    • Auflösung. Wählen Sie kleine oder mittelgroße Miniaturansichten, um den Importprozess zu beschleunigen. 
    • Miniaturansichten vom Vorschau-Bild erstellen. Wählen Sie diese Option, um den Import von RAW-Dateien wesentlich zu beschleunigen. Allerdings kommt es dabei zu visuellen Abweichungen zwischen Miniaturansicht und hochauflösendem Bild.
    • Während des Importvorgangs scharfzeichnen. Für maximale Importgeschwindigkeit schalten Sie die Scharfzeichnung ab.


Fenster>Einstellungen>Import*.

Allgemein

  • Beziehungen
    • Abgeleitete Bilder automatisch aus EXIF-Daten erkennen. Schalten Sie diese Option ab, wenn Sie nicht an den Beziehungen zwischen Bildern interessiert sind.

RAW-Konvertierung

Die Konvertierung von RAW-Bildern erfolgt mit dem Modul DCRAW. Sie können diesen Prozess mit Hilfe der unten angegeben Parameter beschleunigen. Außerdem können Sie eine optimierte Version von DCRAW einsetzen. Auf Windows z.B. unterstützt das eingebaute DCRAW-Modul alle Betriebssystemversion vom Windows XP an. Eine für Vista optimierte Version wird etwas schneller laufen.  Lokalisieren Sie eine solche für Ihr System optimierte Version auf dem Internet, laden sie herunter, installieren sie auf der Harddisk, und spezifizieren den Pfad unter Fenster>Einstellungen>Import>RAW-Konvertierung*).
  • Optionen

    • Interpolation. Wählen Sie die bilineare Interpolation, um den RAW-Import zu beschleunigen.
    • Iterationen. Setzen Sie die Zahl der Iterationen auf 0, um den RAW-Import zu beschleunigen.
    • Spitzlichter wiederherstellen. Schalten Sie diese Option aus, um den RAW-Import zu beschleunigen.
  • Rezept-Detektoren. Selektieren Sie nur die RAW-Konverter, die Sie auch tatsächlich verwenden.
Raw Therapee 2.4.x kann beträchtliche Performanzprobleme verursachen, da es sämtliche Rezeptdateien im Bildordner neu schreibt und so ZoRa Photo Director dazu veranlasst, die zugehörigen Bilder neu zu importieren. Raw Therapee 3.0.x verhält sich hier wesentlich besser. Wir empfehlen deshalb, Raw Therapee 3.0.x zu benutzen.
    • Entwicklungsrezepte beim Import von RAW-Bildern berücksichtigen. Entfernen Sie die Markierung, um die Performanz zu verbessern - insbesondere beim Bearbeiten von Bildern, wenn die Bilder in überwachten Ordnern enthalten sind.

DNG-Konvertierung

  • Beziehungen
    • Verknüpfungen zwischen RAW- und DNG-Dateien automatisch erstellen. Schalten Sie diese Option ab, wenn Sie nicht an den Beziehungen zwischen Bildern interessiert sind.

Indizierung

Es gibt zwei Bereiche, in denen indexiert wird:
  • Metadaten-Indexierung. Einige Metadatenfelder werden in der Katalogdatenbank (Db4o) indiziert. Die Indexierung beschleunigt den Abfrageprozess beträchtlich: Eine Abfrage für ein nicht indiziertes Metadatenfeld durchsucht alle Bilder, während eine Abfrage für ein indiziertes Metadatenfeld relativ schnell übereinstimmende Bilder findet. Natürlich dauert die Pflege eines Index einige Zeit, daher werden Importe und Löschungen langsamer, wenn viele Metadatenfelder indiziert werden. Auch der Katalog wird größer. Sie können die Metadaten-Indizierung unter enster>Einstellungen>Import>RAW-Konvertierung*) konfigurieren.
  • Textsuche und Ähnlichkeit. Für Text- und Ähnlichkeitssuchen wird ein separater Indexierungsprozess verwendet (Lucene). Diese Indexierung wird immer im Hintergrund ausgeführt. Durch Beschleunigung dieses Prozesses wird die allgemeine Leistung verbessert. Sie können diesen Indizierungsvorgang unter Katalog>Eigenschaften>Indizierung konfigurieren. Möglicherweise möchten Sie die Anzahl der ausgewählten Ähnlichkeitsalgorithmen auf ein Minimum reduzieren, da jeder Algorithmus beim Erstellen des Index zusätzliche Schritte erfordert. Auf Plattformen mit geringer Leistung möchten Sie möglicherweise die Indizierung insgesamt deaktivieren oder sogar das Indizierungsmodul während der Installation abwählen.

Generelle Performanz

Katalog>Eigenschaften...

  • Überwachte Ordner. Halten Sie die Zahl der überwachten Ordner auf einem Minimum. So reduzieren Sie die Hintergrundaktivitäten und verbessern die Gesamtperformanz.

  • Defragmentierung. Das gelegentlich Defragmentiieren des Katalogs kann die Abfragegeschwindigkeit erhöhen. Gehen Sie zur Statistik-Seite und klicken Sie auf den Verweis Jetzt defragmentieren. Die Defragmentierung kann einige Minuten daueren.
