Tipps und Tricks


Metadaten

  • Nicht editierbare Metadaten ändern
Die meisten von einer Digitalkamera gelieferten Daten lassen sich normalerweise nicht bearbeiten. Um sie doch bearbeiten zu können, gehen Sie so vor: Setzen Sie das Field Exif>Kameraeinstellungen>Dateiquelle auf ein analoges Gerät, z.B. einen Filmscanner. Nun können Sie die meisten EXIF-Felder bearbeiten. Nachdem Sie Ihre Änderungen angebracht haben, vergessen Sie nicht, das Feld Exif>Kameraeinstellungen>Dateiquelle auf Digitalkamera zurückzusetzen.

  • Ungültiges oder fehlendes EXIF-Bild-Erstellungsdatum
Das kann leicht durch Eintragen eines gültigen Datums im Feld IPTC>Herkunft>Erstellungsdatum behoben werden.

  • Katalog aus einer Sicherung wieder herstellen
Kataloge bestehen aus einer .zdb-Datei und einem .zix-Ordner. Sicherungen sind in Ordnern mit der Dateinamenerweiterung .backup enthalten. Um eine solche Sicherung zu aktivieren
  1. schließen Sie ZoraPD
  2. warten Sie bis das ZoraPD-Icon aus der System-Taskleiste verschwunden ist
  3. ersetzen Sie Ihren Katalog durch die Version aus dem .backup-Ordner (sowohl die .zdb-Datei als auch den .zix-Ordner)
  4. starten Sie ZoraPD erneut.
