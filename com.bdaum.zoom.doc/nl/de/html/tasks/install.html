Installation

Installation unter Windows, Linux and Mac OSX

Laden Sie die Installationsdatei (JAR-Datei) herunter. Dann doppelklicken Sie diese Datei, um sie auszuführen*. Alternativ können Sie die Datei auch mit einem Rechtsklick und Auswahl von Öffnen mit... > Java ausführen. Eine dritte Möglichkeit ist, den Installationsprozess von der Kommandzeile aus zu starten:

java -jar {Installationsdatei}

Folgen Sie dann den Anweisungen des Installationsprogramms. Unter Windows und Linux* können Shortcuts für Desktop und Menü generiert werden. Auf dem Macintosh müssten Sie stattdessen Aliasse manuell erzeugen.

Installation auf Wechseldatenträger

Es ist möglich, ZoRa Photo Director auf Wechseldatenträger (z.B. Memory Stick) zu installieren. Um portabel zu bleiben, installieren Sie am Besten auch eine JRE in the Unterordner jre/.
Auch Kataloge können leicht auf Wechselmedien untergebracht werden. Bei Verwendung von Vorschaubilder mit mittlerer Größe nimmt jeder Katalogeintrag weniger als 20 KB Speicherplatz ein. Auch große Kataloge mit mehreren 10.000 Bildern finden deshalb auf einem Memory Stick Platz.

Deinstallation

Um ZoRa Photo Director zu deinstallieren,  gehen Sie zum Ordner {Installationsverzeichnis}/Uninstaller/ und Doppelklicken die Datei ZoRaPDUninstaller.jar. Alternativ können Sie die Datei auch mit einem Rechtsklick und Auswahl von Öffnen mit... > Java ausführen. Eine dritte Möglichkeit ist, den Installationsprozess von der Kommandzeile aus zu starten:

java -jar {Installationsverzeichnis}/Uninstaller/ZoRaPDUninstaller.jar

Anschließend haben Sie die Option, die benutzerdefinierten Anwendungsdaten im Installationsverzeichnis mit zu löschen - oder auch nicht, falls Sie die Anwendung später noch einmal installieren wollen.

*) Auf Ubuntu müssen die Installationsdatei und generierte Shortcuts zunächst manuell als ausführbares Programm markiert werden (Eigenschaften).
