Setting up your environment

The next step after creating a first catalog and before importing any images is to set-up essential preferences (Window>Preferences*). These preference are valid for the whole workbench independently from the current catalog.

At least you should have a close look at the following preference pages:

  • General. 
    • In group General set a suitable interval for automatic catalog backups.
    • In group Display switch Output ICC-profile to Adobe RGB if you work with a Wide Gamut Monitor.

  • Import. Check all entries on this page before you import images. You find a detailed explanation of the entries here.

  • Keywords. Setup a keyword filter to avoid that your central keyword list gets polluted with unwanted keywords. A detailed explanation is found here.
