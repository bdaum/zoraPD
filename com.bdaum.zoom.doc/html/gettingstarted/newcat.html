Creating a new Catalog

During the very first start of the application, or when a previously used catalog doesn't exist any more, you are prompted to enter the location of an existing catalog or to specify the location of a new catalog. During the creation of the catalog you are prompted with the dialog for the Catalog Properties.  At this stage you should complete or check the following entries before importing any images.

  • Header area
    • Backup File
  • General
    • Owner information
    • Automatically create a time line
    • User defined fields in image meta data
  • Thumbnails
    • Resolution
    • Derive thumbnails from preview image
    • Sharpening during import

For a detailed explanation please see Catalog Properties.

You can invoke the Catalog Properties later via Catalog>Properties or  Alt+Enter.
