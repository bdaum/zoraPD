Export to folder wizard

The Export to Folder function is invoked via menu function File>Export...>Others>Local Folders or FTP. It is used to copy the selected images into the chosen folder, which can be a local or a folder on an FTP account.

You may either export images selected in a gallery or export all images of a slide show, exhibition, or web gallery selected in the Catalog View.

You have the choice to send either originals (which may be rather large) or to downsize the images and compress them into the popular JPEG format. 

The size of these JPEGs can be adjusted using the scale. Alternatively, you may enter a maximum size (either width for landscape orientation or height for portrait orientation).

You can influence the Output Quality with these setttings.

The following options are available for the export operation:

  • Add watermarks for the protection of your intellectual property (not when uploading originals).
  • If you want to include meta data (EXIF and IPTC) into the generated JPEG files, check the option "Include meta data". On the following wizard page you can then select the meta data fields that you want to export.
  • You can filter images and send only public or public+medium images.
Optionally, the exported image can be included into the catalog, too. In this case a "derived" relation is created in order to connect the exported image with the original image.
