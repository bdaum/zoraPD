Exhibit layout


Use this dialog for defining the size of an exhibit, the width and color of the mat and the frame, and some label properties. These values will override the default values defined for the whole exhibition.

  • Image
    Width and height of the image. These values are coupled through the aspect ratio of the image. If you modify one value, the other value will change likewise.
  • Mat
    • Color. The color of the mat (if present)
    • Width. The width of the mat in millimeters. Specify 0 for images without a mat.
  • Frame
    • Color. The color of the frame (if present)
    • Width. The width of the frame in millimeters. Specify 0 for images without a frame.
  • Total
    Width and height of the complete exhibit including mat and frame. If you change one of this values the values for the image will change likewise.
  • Label
    • Hide. Check this option for not displaying a label with this exhibit.
    • Alignment. Defines the spatial relationship of the label to the image. 
    • Distance. The distance of the label from the border of the image.
    • Indent. How much the label is shifted along the border of the image.
