Refresh dialog

The menu function File>Refresh... re-imports selected files. In the Refresh dialog, you can specify what to refresh:

  • Re-import existing images. This will allow you to update images that are already in the catalog. This is typically used after an image had been modified by another program. Another option is to use Watched Folders to detect changes automatically.
  • Include up-to-date images. This option is normally not used. It should only be used after a new program version has introduced new attributes or corrected attributes, or when thumbnail size or sharpening was changed in the Catalog Properties. Using this option, all catalog entries can be updated with new or corrected attributes.
  • Remove missing images from catalog. This is used to remove catalog entries for which no image file exists (orphans).
  • Include remote images. Check this option to images that are stored remotely on some server on the Internet, too. This option can slow down the Refresh operation considerably because images must be downloaded to the local computer.

You can determine, too, which information groups should be refreshed. You may not want to refresh groups where you already have edited image meta data manually.

  • Refresh Thumbnail. This option will update the catalog thumbnail and is typically used after images have been updated with an image editor or after the thumbnail size or sharpening had been changed. However, refreshing will be much faster if this option is switched off.
  • Refresh image status. De-select this group if you have manually modified status, color code, rating, content type, privacy settings, or user fields and don't want to reset that information.
  • Refresh EXIF data. De-select this group if the EXIF data have not changed. For example, you may only want to refresh the GPS data.
  • Refresh GPS data. De-select this group if you have geo-tagged the image and you do not want to reset that information.
  • Refresh IPTC data. De-select this group if you have manually entered keywords, categories and other IPTC data and you do not want to reset that information.
