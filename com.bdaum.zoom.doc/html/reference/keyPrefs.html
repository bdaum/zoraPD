Keys preferences

The Keys preference page allows you configuring keyboard shortcuts. Simply select a command in the table at the top (user defined entries are printed in bold) and enter the key shortcut (in the same way as you would use it) in field Key Sequence. Special keys such as Delete, Backspace, or Tab can be inserted via the arrow button at the right hand side of the Key Sequence field.
If a key sequence is assigned to more than one command, those commands will appear in the Conflicts area. Double click an entry there to navigate to the command in question and change its key sequence.
