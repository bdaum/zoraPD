Raw Developer Recipe Detector

Identifier:
com.bdaum.zoom.core.recipeDetector

Since:
0.9.6

Description:
Allows the definition of detectors for raw development recipes. Because such recipes are highly proprietary, each raw processor must be covered by a dedicated recipeDetector extension

Configuration Markup:

<!ELEMENT extension (detector+)>

<!ATTLIST extension

point CDATA #REQUIRED

id    CDATA #IMPLIED

name  CDATA #IMPLIED

>


<!ELEMENT detector (parameter?)>

<!ATTLIST detector

id          IDREF #REQUIRED

name        CDATA #REQUIRED

class       CDATA #REQUIRED

description CDATA #IMPLIED

>

A recipe detector detects proprietary development recipes produced by RAW converters and translates them into a common Recipe object.


  • id - A unique ID
  • name - Name of detector
  • class - The detector implementation. It is more convenient to extend AbstractRecipeDetector instead of implementing IRecipeDetector from scratch.
  • description - describes to which types of recipes the detector applies

<!ELEMENT parameter (parameterValue+)>

<!ATTLIST parameter

name    CDATA #REQUIRED

id      CDATA #REQUIRED

default CDATA #IMPLIED

>

Parameters are used to control the execution of recipe detectors. For example, you might want to assign a priority to a specific type of recipe detectors if that type competes with other recipe detectors for the same recipe files. Or, you might want the user to specify if recipes in a cache take priority above sidecar recipes, or vice versa.


  • name - Display name of parameter
  • id - Unique parameter ID. Predefined IDs: com.bdaum.zoom.recipe.xmpEmbedded is used for recipes stored in XMP sidecars and specifies the priority of the detector in case of conflicts as an integer
  • default - Default value of parameter: specify id of parameter value

<!ELEMENT parameterValue EMPTY>

<!ATTLIST parameterValue

label CDATA #IMPLIED

id    CDATA #REQUIRED

>

Description of a parameter value


  • label - Display label. If omitted, the label defaults to the id value
  • id - Unique ID for value

Examples:
see com.bdaum.zoom.recipes.acr, com.bdaum.zoom.recipes.bibble, com.bdaum.zoom.recipes.c1, com.bdaum.zoom.recipes.rt.
