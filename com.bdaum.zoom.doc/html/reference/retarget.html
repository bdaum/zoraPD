Retarget operation

Retargeting is only necessary when image files were physically moved outside of ZoRa Photo Director, so that catalog entries had become orphaned. The retargeting operation is therefore only offered in the context menu of orphaned thumbnails.
The retargeting operation allows you to update the selected image, or all images in one of the parent directories of that image - including subfolders. In the process it may create a new directory structure in the Catalog View.
Optionally, you can also retarget voicenotes if you had moved voicenote files together with their images.
Hint: You can easily find orphaned catalog entries via menu function Catalog>Search...>Orphaned Entries.
Hint 2: You can also retarget an orphaned catalog entry by dragging the target file onto the orphaned entry and keeping the ALT-key pressed.
