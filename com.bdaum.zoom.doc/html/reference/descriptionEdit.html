Edit description dialog


This dialog allows you to describe how an image was derived from another image or how an image was composed from other images:

KindSelect the type of composition such as Panorama, HDR, Focus stacking, etc.
ToolSpecify the name of the tool used for the image transformation, e.g. Adobe DNG converter, Hugin, picturenaut, etc.
Parameter fileSpecify the file that controls the image transformation. For example, a .pto-file controls the panorama composition with Hugin. Press the Browse... button to locate the file. Press the Open... button, to open the associated application. For example, if you open a .pto-file, Hugin will start and open the associated panorama.
RecipeDescribe how the transformation was achieved.
