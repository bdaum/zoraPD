Contributing help pages

Help pages can be embedded into the system in three ways:

  1. Specifying a table of contents as an XML file and registering it at extension point org.eclipse.help.toc. You may create an own primary help book that shows up as a root element in the help tree or you can embed your table of contents into an existing tree. ZoRa Photo Director defines the following anchor points for embedding sub-trees:


    com.bdaum.zoom.doc/tocconcepts.xml#concepts
    		 Additional sections in help book Concepts
    		 
    com.bdaum.zoom.doc/tocgettingstarted.xml#gettingstarted
    		 Additional sections in help book Getting started
    		 
    com.bdaum.zoom.doc/tocreference.xml#reference
    		 Additional sections in help book Reference 
    com.bdaum.zoom.doc/tocreference.xml#views
    		 Additional sections in chapter Views in help book Reference 
    com.bdaum.zoom.doc/tocreference.xml#webEngines
    		 Additional sections in chapter Views/Webengines in help book Reference 
    com.bdaum.zoom.doc/tocreference.xml#preferences
    		 Additional sections in chapter Preferences in help book Reference 
    com.bdaum.zoom.doc/tocreference.xml#export
    		Additional sections in chapter Export in help book Reference 
    com.bdaum.zoom.doc/tocreference.xml#exportOthers
    		Additional sections in chapter Export/Others in help book Reference 
    com.bdaum.zoom.doc/tocreference.xml#extensions
    		 Additional sections in chapter Extensions in help book Reference 
    com.bdaum.zoom.doc/tocreference.xml#import
    		 Additional sections in chapter Import in help book Reference 
    com.bdaum.zoom.doc/tocreference.xml#internet
    		Additional sections in chapter Internet in help book Reference 
    com.bdaum.zoom.doc/tocreference.xml#viewers
    		 Additional sections in chapter User interface in help book Reference 
    com.bdaum.zoom.doc/toctasks.xml#tasks
    		 Additional sections in help book Tasks 
    com.bdaum.zoom.doc/toctasks.xml#export
    		Additional sections in chapter Export in help book Tasks 
    com.bdaum.zoom.doc/toctasks.xml#import
    		 Additional sections in chapter Importing images in help book Tasks 
    com.bdaum.zoom.doc/toctasks.xml#exportOthers
    		 Additional sections in chapter Export/Others in help book Tasks 
    com.bdaum.zoom.doc/toctasks.xml#internet
    		Additional sections in chapter Internet in help book Tasks 
    com.bdaum.zoom.doc/toctasks.xml#viewers
    		 Additional sections in chapter Invoking editors and viewers in help book Tasks 
    com.bdaum.zoom.gps/toc.xml#mappingSystems
    		 General information about add-on mapping systems 
    com.bdaum.zoom.gps/tocprefs.xml#mappingSystemPrefs
    		 Information about preference settings introduced by add-on mapping systems

    Please see the Eclipse documentation for the extension point org.eclipse.help.toc for further details.

  2. You can associated help pages with widgets as context help, so that the help page is called when F1 is pressed while the widget has the focus. For doing so, you must register a context description at extension point org.eclipse.help.contexts. Please see the Eclipse documentation for the extension point org.eclipse.help.contexts for further details.

  3. Contribute to the help index. Index entries are added by registering an index description file at extension point org.eclipse.help.index. Please refer to the Eclipse documentation for the extension point org.eclipse.help.index. for further details.

When authoring help pages for ZoRa Photo Director plug-ins, please use the ZoRa Photo Director style sheet. This is stored in the plug-in com.bdaum.zoom.rcp. From your own help pages you can, for example, refer to it like this (assuming that your help page is stored in a direct child folder of the plug-in folder).

  <link rel="stylesheet" type="text/css" 
        href="../../com.bdaum.zoom.rcp/book.css">

and/or

  <link rel="stylesheet" type="text/css" 
        href="PLUGINS_ROOT/PRODUCT_PLUGIN/book.css">
