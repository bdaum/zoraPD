Merge Catalogs


This function can import the images of another catalog into the current catalog.

You must specify the catalog file to be imported and a policy for handling duplicates:
  • Skip. An already existing catalog entry remains as is - the new entry is ignored.
  • Replace. The already existing catalog entry is deleted, then the new entry is imported.
  • Merge. The new entry is merged into the already existing entry. How this merge will be performed can be specified in detail on the next wizard page:
    • Overwrite. All selected meta data entries are overwritten.
    • Fill-in blanks. Selected meta data entries are only overwritten when they currently do not contain any value.
    • Combine. Similar to Fill-in blanks but the values of multi-dimensional entries (such as the Keywords entry) are combined.
