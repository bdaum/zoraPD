Table view

The Table view organizes the thumbnails of a collection or a query into a vertical column. Additional columns display selected meta data.

Navigation is performed through the scroll bars.

The width of the columns can be set by dragging the separators between the individual column headers.



The sort order can be changed by clicking on one of the column headers. Clicking a second time on the same column header will toggle between ascending and descending sort order. A little arrow in the column header indicates the sort order.

By clicking on Configure>>> in the first column header (thumbnail column) you can configure the table, i.e. determine how many additional columns are displayed and what is shown in those columns.

For more key commands and mouse gestures please see the general Galleries help page.
