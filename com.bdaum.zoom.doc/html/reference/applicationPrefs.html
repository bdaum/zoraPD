Application


  • The Fullscreen group allows configuring the fullscreen mode: whether menu bar and/or trim bars are shown in fullscreen mode.
  • The option Run program in tray mode minimizes the application into the system tray at program start. A double click on the tray icon makes the application visible. A click on the close button does not exit the program but minimizes it again into the system tray.
  • The option Delete trashcan when sent to tray deletes the trashcan when the program runs in tray mode and is sent to the tray. Otherwise the trashcan is only deleted when the program is exitted.
