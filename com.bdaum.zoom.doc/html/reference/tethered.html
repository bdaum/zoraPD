Tethered shooting

Tethered shooting is a shooting mode where the camera is connected to a host computer (e.g. laptop). When an image is shot it is automatically transferred to the host computer and deleted from the cameras memory.

Note: Not all cameras support tethered shooting. The camera must be able to take pictures while it is connected to a host computer via USB.

ZoRaPD supports tethered shooting for all cameras that do support the File>Import>from device function. It can be started via the menu function File>Tethered shooting. Alternatively, it can be started by connecting the camera to the host computer when the option On device insertion ... start tethered shooting is set in Window>Preferences>Import>General.

During tethered shooting the images coming from the camera are first placed into a transfer folder before they are processed by ZoRaPD. It is therefore necessary that you select a transfer folder as defined under Catalog>Properties>Watched Folders or that you create a new one.

After processing the transferred images appear in the collection Imported>Last background imports () and can be viewed from there.

Tethered shooting can be stopped by disconnecting the camera from the host or by invoking the function File>Tethered shooting again.
