Output quality

The controls of the Output Quality group influence the output quality of exported images:

  • Sharpen. Check this button to enable output sharpening. An Unsharp Mask filter is used for that purpose:
    • Amount. The amount of sharpening
    • Radius. The radius of the Unsharp Mask in pixel.
    • Threshhold. Increase this value for inhibiting sharpening in low contrast area, so that noise is kept low.
When downsizing images you should apply some sharpening because interpolation causes smear between pixels. When outputting for printing, you should sharpen even a bit more.
  • JPeG quality. You can set the JPeG quality between 1 for low visual quality and small storage size and 100 for best visual quality and large storage size. The default value of 75 combines good compression with virtually no visible artifacts.
  • Subsampling algorithm. By default, bilinear interpolation is used when downscaling images. For best quality select Lancos - which is also the slowest.
