Lighbox view

The Lightbox view uses the metaphor of the classical lightbox, where you put your trannies on a translucent surface. Navigation in the Lightbox view is performed in the classic way using the vertical scrollbar.

The size of the thumbnails can be adjusted using the Zoom slider in the views title bar. Alternatively, the size of the thumbnails can be adjusted with the mouse wheel or with the keyboards  + and - keys.

The Lightbox view features a special "Collapsed" mode which can be switched on with the Collapsed tool button  () or the local Collapsed menu function. In this mode, consecutive images with similar names can be stacked together to one single thumbnail. Such stacks can be individually expanded by clicking the  +-button shown on top of the thumbnail, or by using the Toggle collapsed state context function. The criteria for name similarity can be configured by pressing CTRL and clicking on the Collapsed tool button. In addition is it possible to declare stacks explicitely.

The collapsed mode forces the images to be sorted by URI. Returning to the non-collapsed mode will return to the original sort sequence, too.

For more key commands and mouse gestures please refer to the general Galleries help page.
