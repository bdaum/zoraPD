Import into new folder structure


Usually, when importing files and folders, all image files stay in their original locations. However, by using the menu function File>Import...>folder and selecting "into new folder structure" files can be renamed and moved into a new folder structure.

The import options are specified with the help of a wizard:

Step 1, File selection

The top section of this wizard page contains a tree organizing the files based on their modification date. This allows selecting images by their creation date. Only supported file types are considered, however.
  • Don't import duplicates. Check this option if you want to avoid that duplicates are imported into the catalog. Detection is performed by comparing the original filename and the modification date of the file. A warning is written into the Log file if a duplicate was detected.

Step 2, Output target selection

  • In the Transfer to entry specify the target folder where you want the images to be stored. The images selected in step 1 will be copied to the target folder during the import process. 
  • In the Create subfolder entry you may opt to have sub folders created, either by year, by year and month, by year, month and day, or not at all.

Step 3, File renaming

Imported files can and should be renamed during the import. The renaming process is controlled by a template. There are several predefined templates but you can also add (Add...) your own and edit (Edit...) templates. In the Cue field you may want to supply a short string that will replace {cue} variables within templates.

Step 4, Metadata options

In the last step, the metadata of the imported images may be completed by adding additional entries.
  • Add metadata
    • Artist. Photographer or artist who created the imported pictures.
    • Event. The event documented by the imported images.
    • Keywords. Keywords to be added to the imported photos.
    • Add to album. This option only appears when you drag files or folders into an album gallery. If you do not want to add the images to the album remove the check mark from the check box.
  • Automated rating. Check this option if you want to apply automated rating to all imported images.
