Import Preferences

General

  • Content
    • Import makernotes. If you are interested in the manufacturer specific maker notes embedded in the EXIF data of your images, mark this option. Otherwise, leave this entry unmarked. If you should later change your opinion, you can mark it then and refresh the EXIF-data of your images.
    • Import face data from camera. Mark this option if you want to import face data such as face regions from your camera.
  • Display
    • Max. length of import list. Entries that exceed the maximum length will be deleted (when empty system collections are removed).
  • Events
    • Show imported image after import. Check the option to automatically show the imported images in the current gallery when an import is finished.
    • On device insertion. Set this option to Import from Device if you want this function to be invoked when a device or a removable medium is connected to the computer. Set this opiton to Start tethered shooting if you want to start tethered shooting mode as soon as a device is inserted.
    • Show new tethered shot in. Select how a new image obtained via tethered shooting should be shown:
      • gallery. The image appears as a thumbnail in the Last import collection in the current gallery. It can be opened from there with a double click.
      • internal image viewer. The image is immediately shown by the internal image viewer.
      • external image viewer. The image is immediately shown by the configured external image viewer.

RAW conversion

  • Raw converter. Here you have the option to select between dcraw, RawTherapee, and darktable.

dcraw

The following options are availabe for dcraw:
  • Converter
    • External DCRAW module. Here you can specify a path to an external DCRAW module, i.e. an DCRAW converter located outside of the installation folder.
      Tip: If you use an Raw converter based on DCRAW, e.g. LightZone, it makes sense to point to the DCRAW module of that converter. By doing so you can make sure that both the raw converter and ZoRa PhotoDirector use the same version of DCRAW.
  • Options
    These options only apply when loading high-resolution images, not for generating thumbnails.
    • White balance. Selects different white balance methods:
      • D65 A gray card illumated with a D65 light source appears gray.
      • Camera. The white balance determined by the camera is used
      • dcraw. The white balance determined by the DCRAW converter is used.
    • Interpolation. Selects the DCRAW interpolation algorithm. Available are:
      • bilinear (fast)
      • Variable Number of Gradients (VNG)
      • Patterned Pixel Grouping (PPG) 
      • Adaptive Homogeneity-Directed (AHD) 
    • Iterations. Number of iterations for applying a 3x3 median filter. Reduces artifacts but slows down conversion.
    • Highlight recovery. Blown-out Highlights are recovered (slower).
  • Active Recipe Detectors. Check all RAW converters that you are using for developing RAW files. Uncheck all others for not slowing down import and display functions. If you use several RAW converters and there are multiple recipes for an image, the most recent recipe will be used. In the Configuration column you can specify how to resolve conflicts between competing recipes.
  • Process recipes when importing RAW images. Uncheck this option if you do not want to apply the changes made in RAW converters. This will speed up operation - and you will be looking onto the original RAW files.
  • Archive recipes of derived images. Check this option to store the complete raw recipe used for creating a derived image into the catalog. This allows to later restore the recipe in case it was modified after the creation of the derived image.
  • Update all affected catalog entries now? This option is available after adding a new recipe detector. Check this option to apply already existing recipes of this recipe detector to the existing catalog entries.

RawTherapee

The following options are availabe for RawTherapee
  • Converter
    • RawTherapee module. Here you must specify a path to the RawTherapee executable installed elsewhere on your system.
  • Options
    • Consider .pp3 sidecar recipes. Check this option if you want to apply existing conversion recipes created with RawTherapee when importing, displaying, or exporting images. The results will be 100% identical to conversion results created with RawTherapee alone.

darktable

The following options are availabe for darktable
  • Converter
    • darktable module. Here you must specify a path to the darktable executable installed elsewhere on your system.

DNG conversion  (Only under Window and MacOS)

  • Basics
    • On Import of raw files. Specify how RAW files are imported: 
      • Convert to DNG and import both RAW and DNG files. This is the safest import mode because images are kept in both original format and in the camera-neutral DNG format.
      • Import RAW files only. No conversion to DNG is performed.
      • Convert to DNG and import DNG files only. 
      • Convert to DNG with embedded RAW file and import DNG. With this option it is possible to retrieve the original RAW file from the DNG file with the help of the Adobe DNG converter.
    • Location of Adobe DNG converter. If you want to convert RAW files to the DNG format during import you must specify the location of the DNG converter here. You can download a copy of the free Adobe DNG Converter from www.adobe.com/products/dng/.
  • Options
    • Create uncompressed DNG files. Advanced setting. Please consult the documentation of the Adobe DNG converter. 
    • Convert original RAW file into linear image. Advanced setting. Please consult the documentation of the Adobe DNG converter.
    • DNG subfolder. Converted DNG files are placed into a sub folder of the folder containing the RAW image. You may specify a sub folder name here or leave the entry empty for placing the DNG files side-by-side with the RAW files.
