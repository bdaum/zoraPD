Group dialog


The group dialog is used for defining the properties of groups and subgroups.

General

  • The group name as it appears in the Catalog view.

Rules and collection filters

  • Catalog specific rules for automatic collection creation that overwrite the global rules defined in the preferences. These specific rules are defined in group "Automatic".
  • Filters for the groups "Rating", "Recent imports", "Locations", "Timeline". By defining an appropriate filter it is possible to hide certain elements in those groups.

Appearance

  • Thumbnail labels. Here you can define a policy how thumbnail labels of images in this group shall be displayed. If you choose "inherited", the settings of the parent group or of the global preferences are used.
    Select "custom" to define specific templates, font sizes, and alignments for that group.  Note, that subgroups or collections may overwrite those policies.
