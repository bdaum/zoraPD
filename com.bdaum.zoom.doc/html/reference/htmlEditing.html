HTML editing



Some text fields may contain HTML mark-up to style the entered text. In this case you will be able to switch between plain text mode (with spell checking) and HTML text mode (no spell checking).
When working in HTML mode you must enter all text styling (including line breaks) in form of HTML tags. Special characters such as national characters must be encoded as HTML entities. We therefore recommend strongly to
  • first enter all text in plain text mode. The spell checker is active in this mode.
  • then switch to HTML mode. You can now easily add extra styling by selecting text parts, pressing CTRL+Spacebar and selecting a style from the list.

Keyboard shortcuts

Ctrl+Spacebar Select HTML style from a list
Ctrl+B Make selected text bold
Ctrl+F Make selected text bold
Ctrl+I Make selected text italic
Ctrl+K Make selected text italic
Ctrl+U Make selected text underlined
