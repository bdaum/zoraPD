Vocabulary preferences

The IPTC fields Intellectual Genre, Scene Code and Subject code are supported by predefined vocabularies provided by iptc.org. In addition, user defined vocabularies can be used, too.

To do so, simply specify the words of each vocabulary, each word on a separate line. The buttons at the right hand side allow you clearing all words from the user defined vocabulary, sorting the user defined vocabulary, storing it to a file or loading from file. Sorting removes duplicates, too.
