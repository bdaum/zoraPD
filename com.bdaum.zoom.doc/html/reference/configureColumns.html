Configure Columns

The Configure Columns dialog allows configuring the selection and sequence of columns in the Table view:

  • To add a column to the table, select an item in the tree on the left side, and press the right arrow button.
  • To remove a column from the table, select the column in the list on the right side and press the left arrow button.
  • To move a column towards the thumbnail column, select the column in the list on the right side and press the up arrow button.
  • To move a column away from the thumbnail column, select the column in the list on the right side and press the down arrow button.
