Keyword Search

Not surprisingly, the keyword search searches only in the keywords defined with an image - opposed to the text search which will search in all text entries. It is possible to use Boolean operators such as AND, OR. and NOT to search for keyword combinations:
  • Use the AND operator for AND conditions: Only images containing both keywords will be returned.
  • Use the OR operator to find all images containing one, several, or all of the keywords.
  • Use the NOT operator to find images not containing a keyword.

Examples: 

cats OR birds
 find all images tagged with cats or birds 
dogs AND cats
 find all images tagged with both dogs and cats 
animals NOT cats
 find all images tagged with animals but not with cats


As you type, the content assistant of the keyword search dialog will make proposals based on the catalogs keyword list. Because the catalog contains all available keywords other entries don't make sense.
  • Press Arrow-Up, Arrow-Down, Page-Up, Page-Down to scroll through the entries.
  • Press the Arrow-Right or End to accept a proposal and continue typing. Enter starts the search.
  • Type on to reject a proposal
