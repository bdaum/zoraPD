GPS Parser

Identifier:
com.bdaum.zoom.ui.gpsParser

Since:
0.9.0

Description:
Extension point for registering different parsers for GPS tracker file formats

Configuration Markup:

<!ELEMENT extension (parser*)>

<!ATTLIST extension

point CDATA #REQUIRED

id    CDATA #IMPLIED

name  CDATA #IMPLIED

>

Extension for registering parsers for geo data file formats



<!ELEMENT parser EMPTY>

<!ATTLIST parser

name       CDATA #REQUIRED

extensions CDATA #REQUIRED

class      CDATA #REQUIRED

waypoints  CDATA #IMPLIED

>

Parser


  • name - Name of the Parser
  • extensions - File name extensions separated by white space
  • class - Parser implementation. Must implement com.bdaum.zoom.ui.gps.IGpsParser.
  • waypoints - Optional implementation for a waypoint collector. Must implement com.bdaum.zoom.ui.gps.IWaypointCollector.


Examples:
see com.bdaum.zoom.gps/plugin.xml
 

API Information:
see com.bdaum.zoom.ui/com.bdaum.zoom.ui.gps.IGpsParser and com.bdaum.zoom.ui/com.bdaum.zoom.ui.gps.IWaypointCollector
