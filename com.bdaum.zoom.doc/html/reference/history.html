History view

The History view shows a timeline of the past queries. This includes not only permanent collections shown in the Catalog view but also Adhoc queries, Time queries, Similarity queries, Text queries, and so on. 

Menus and actions


The following operations can be executed::
  • Context functions
    • Play slideshow. Start to play a slide show presentation. The player is interactive and allows changing image properties while playing.
    • Select all images. Selects all images in the currently opened galleries.
    • Edit. Allows to edit the selected item.
  • View actions
    • Today (). Positions the view to todays queries.
    • Yesterday (). Positions the view to yesterdays queries.
    • Week (). Positions the view to the queries from the week start or the previous week.
    • Month (). Positions the view to the queries from the month start or the previous month.
    • Older (). Positions the view to older queries.
    • Show three month. Shows only queries from the last three month.
    • Show six month. Shows only queries from the last six month.
    • Show one year. Shows only queries from the last year.
  • Mouse gestures
    • Single click. Selects an item and show its images in a gallery or a presentation editor.
    • Double click and Ctrl+click. Invokes the edit function on the clicked item.
