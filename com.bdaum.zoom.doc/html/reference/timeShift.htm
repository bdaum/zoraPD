Apply time shift

This function is typically used to correct EXIF time stamps when the camera was set to a wrong time or time zone. The specified shift value is applied to all selected images. It is applied to the following fields:
  • EXIF image creation date
  • EXIF file modification date but only if equal to the image creation date
The modifications are applied to the ZoRa Photo Director catalog. Image files are not modified but the new values are used when XMP files are created.
