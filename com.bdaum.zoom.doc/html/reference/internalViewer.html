Internal image viewer

The internal image viewer can be invoked by pressing a modifier key and double-clicking an image (or via the context menu). The following modifier keys are possible:


Shift Forces the internal viewer.
Alt Forces the internal viewer being invoked in black&white mode*.
Ctrl
Forces the internal viewer in crop mode. The cropped area of a RAW image is shown without its surroundings.
Ctrl + Alt Forces the internal viewer in crop mode and in black&white mode*.
Ctrl + Shift Forces the internal viewer in raw original mode. Development recipes of RAW images are not applied.
Ctrl + Shift+ Alt Forces the internal viewer in raw original and black&white mode*.

*) If the image viewer is called while black&white mode is already active, it switches to color mode, instead.

On invocation the image viewer will immediately show an enlarged thumbnail while loading the original image in full resolution. For RAW images the loading process can take a few seconds because the RAW file needs to be developed first.

If you are impatient, you can force the image viewer to show the embedded preview image while loading and developing the raw data - provided such a preview image is available. You can set this option under Window>Preferences>External programs*.

The following mouse and key actions are available:

  • Depending on the settings made under Window>Preferences>Mouse* you can adjust the magnification by pressing the ALT or Shift key or using the right mouse button and dragging left or right. 
    Alternatively you can zoom in and out with the mouse wheel or by using the + and - keys.
  • You can pan by by dragging the mouse using the left mouse button. Ctrl accelerates the panning operation.Alternatively you may use the arrow keys for panning.
  • A double click toogles between the initial image scale and the 1:1 image scale. Alternatively, you can use the * key or the Spacebar.
  • A numeric key 1, 2, ... 9 adjust the image scale to 1:1, 1:2, .... 1:9, depending on the key.
  • Shift plus a numeric key 1, 2, ... 4 adjust the image scale to 1:1, 2:1, ..., 4:1.
  • h to zoom the image to the height of the monitor
  • w to zoom the image to the width of the monitor
  • F1 displays help information.
  • F2 displays metadata of the current image.
  • Press ESC to leave the image viewer. Alternatively, you can move the mouse to the top right or left screen corners. A red close button will appear. Click this button to leave the image viewer.

*) On Mac use ZoRa Photo Director>Preferences>...
