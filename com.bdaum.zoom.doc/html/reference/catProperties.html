Catalog Properties

The Catalog Properties can be reached via menu function Catalog>Properties... or via the key shortcut Alt+Enter. They contain all non-image related data of a catalog. Other configurations can be made under Window>Preferences...* but they affect all catalogs.

The header data of the Catalog Properties include:
  • File name. File name and path of the catalog.
    Optionally, you can write protect the catalog.
  • Language. Here you can select an installed dictionary for spell checking
  • Creation date. Date and time when the catalog was created.
  • Last import. Date and time when the last import was performed.
  • Sequence no. The number of imported images since catalog creation.
  • Catalog theme. The general catalog theme such as General, Food, Wedding, or Travel. This has influence on the initial set of categories.
  • Yearly sequence no. The number of imported images in the current calendar year.
  • Backup File. File name and path of the back-up file. This entry can be modified. The back-up interval can be set by clicking Configure Interval. 
    The backup path may contain the variable {catalogLocation}. This will be replaced by the path of the folder containing the current catalog.
    If the file name includes the variable {date}, each back-up will go into a different file. Otherwise, a new back-up will overwrite a previous back-up. 
The rest of the data is ordered into groups which can be reached via the tabs of the tabbed notebook or by using the Next and Previous buttons.
  • General
    • Owner information. The owner of the catalog - usually the person who created it.
    • Description. A short description of the catalog.
    • Automatically create a timeline. If you have not already organized your images in a folder structure representing a time line: this a great possibility to order your images by date. During the import of images the necessary collections for organizing a time line will be created automatically. You may organize the timeline in years, months, or days or switch off timeline creation. If you later change your mind you can recreate or remove the timeline by pressing the button Create now/Remove now.
    • Automatically create location collections. This option will organize all geo-named images into appropriately named collections. You can organize your locations by country, by province/state, or by city, or switch off location collection creation altogether. If you later change your mind you can recreate or remove the location folders by pressing the button Create now/Remove now. 
      Because large location collections can be slow, the option is switched off by default.
      Note: The location where the images was created (not the location shown) is used for this function.
    • User field label 1 and 2. There are two fields in the catalogs meta data for user defined purposes. Here you can label those fields appropriately.
  • Thumbnails
    • Resolution. Select the thumbnail resolution that suits you best. Large thumbnails look better in the preview but will slow down operation and increase catalog size. 
    • Derive thumbnails from preview images. Check this option if you want to utilize the preview images embedded in RAW files for thumbnail creation. Otherwise, the RAW file is converted (using DCRAW) to obtain the catalog thumbnails. This may take longer but will provide a better consistency between thumbnail and highres image because developing recipes from some raw converters are applied.
    • Sharpen during import. Select a degree for sharpening thumbnails during import. Sharpening can improve the appearance of the thumbnails in the catalog. After changing this setting you may want to refresh  existing thumbnails.
    • Use WebP instead of JPeG compression. By using the WebP compression method you can save about 20 percent catalog space. The disadvantage, however, is that WebP compressed thumbnails deteriorate during rotation.
    • Compression quality. Choose between 1 for low visual quality and small storage size and 100 for best visual quality and large storage size. The default value of 75 combines good compression with virtually no visible artifacts.
  • Categories
    • Images can be associated to categories. A hierarchy of available categories can be created and maintained here. This hierarchy can be exported to a file with the Save ... button. Category files can be imported with the Load ... button. To create a new category, use the Add... button, to create a sub-category, use the Refine... button. To remove categories, press the Remove... button.
  • Keywords
    • Image meta data can contain an unlimited number of keywords. These keywords are collected during the import process. (Click Configure keyword filter for not importing unwanted keywords) On the left hand side of the page you can manage the keywords contained in the catalog.
      • New keywords can be added to this collection by pressing the Add... button. To correct spelling errors in an existing keyword press the Edit... button. All catalog entries containing that keyword will be updated. To replace keywords by one or several other keywords click the Replace... button. To remove keywords, press the Remove... button.
      • The Collect... button collects keywords from images that are not in the list, for example keywords that were previously deleted or keywords that were filtered by a previously active keyword filter. The Collect-dialog also shows unused keywords.
      • The keywords can be exported to a file with the Save ... button. Keyword files can be imported with the Load ... button. The Show images button shows all images containing the selected keyword.
      • Add person names to keywords. Mark this option if you want to add person names to the IPTC keywords.
      • Exclude geographical keywords. This will exclude from the list all keywords found in location records (see Locations).
    • On the right hand side you can define Controlled Vocabularies. When such vocabularies are defined, all keywords that are not contained in a vocabulary are marked with a warning icon.
      • Click Add... to add a vocabulary to the list.
      • Click View... to view the content of the selected vocabularies. If more than one vocabulary is selected, the combined content is shown.
      • Click Remove to remove the selected vocabularies from the list.
      • Click Enforce... to remove any keywords that are not contained in a vocabulary from the catalog. Keywords that match a vocabulary synonym are replaced by the synonyms owner.
  • Indexing (optional)
    • Here you can configure the indexing process. On the left side you may select or deselect algorithms for content based similarity search. The number of selected algorithms will determine the size of the index and the speed of indexing. Note, that only the selected algorithms will later be available for searching.
      Algorithms marked with  are AI algorithms and require an account with a service provider (User Preferences>AI-Services).
    • On the right side select all the text fields evaluated for building the text index. Selecting the right fields will help you to get rid of "noise" result images when you search for text expressions.
    • Note that the above choices only affect future imports. However, you may want to re-index the whole catalog via Catalog>Maintenance>Refresh index.
    • If you are not interested in similarity search and free text search, you can switch off indexing altogether. This can make sense on low performance platforms.
  • Locations
    • Locations are used in IPTC data to describe the location where an image was taken (Location created) and what the image shows (Location shown). Here, the collection of all locations is presented. You can add new locations, edit locations, and remove locations. You can also show all images of a selected location or display a location in the Map view.
  • Artwork/Objects
    • Artwork/Object entries are used in IPTC data to describe artwork or copyrighted objects depicted in images. Here, the collection of all artwork/object entries is presented. You can add new artworks or objects, edit  or remove artworks and objects. You can also show all images depicting a selected artwork.
  • Contacts
    • Contacts entries are used in IPTC data to provide detail information for contacting the creator or owner of the image. Here, the collection of all contacts entries is presented. You can add new contacts, edit  or remove contacts. You can also show all images containing a selected contact. 
  • Statistics
    • The statistics page shows statistic data about the catalog such as used and free space, number of collections and images, etc. It offers an option to defragment the catalog. This can improve performance considerably.
  • Watched folders
    • Here you can add folders to the list of watched folders. When new images are added to such a folder, the images are imported automatically into the catalog. Depending on the type of the watched folder the images are either transferred to a target folder or are process in situ. 
    • The latency setting specifies a delay between the actual file change and the triggered import process. Such a delay is necessary to avoid duplicate imports caused by incremental file updates. If you experience frequent messages such as "File ... has already been imported" increase the latency time.
    • You may also pause folder watching by setting the paused check box.
    • Check Automatically add imported folders to the list of watched folders for automatic maintenance of the list of watched folders. This option will not apply to File>Import...>From Device... or to the File>Import...>Into new Folder Structure....
    • Show background imports. If this option is set, import collections will be created, too, for images silently imported in the background such as images from watched folders.
