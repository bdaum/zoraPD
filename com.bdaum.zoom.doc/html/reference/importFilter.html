Import Filter

Identifier:
com.bdaum.zoom.image.importFilter


Since:
0.8


Description:
Import filters for supporting additional image file formats.
Configuration Markup:

<!ELEMENT extension (importFilter+)>

<!ATTLIST extension

point CDATA #REQUIRED

id    CDATA #IMPLIED

name  CDATA #IMPLIED

>

Extension point for adding additional image import filters



<!ELEMENT importFilter EMPTY>

<!ATTLIST importFilter

extensions CDATA #REQUIRED

class      CDATA #REQUIRED

name       CDATA #REQUIRED

>

An import filter for a specific image content type. The import filter associates one or multiple file extensions with a import filter factory.


  • extensions - A list of file name extensions separated by blanks
  • class - The import filter factory. Must implement com.bdaum.zoom.image.IImportFilter.
  • name - The name of the filter

Examples:
See com.bdaum.zomm.exr/plugin.xml and com.bdaum.zoom.hdr/plugin.xml.


API Information:
Import filter factories must implement the interface com.bdaum.zoom.image.IImportFilterFactory.


Supplied Implementation:
com.bdaum.zomm.exr and com.bdaum.zoom.hdr.
