Hover preferences

The preference page allows you configuring the application wide hover behaviour. Please note that you can switch hover off and on using the menu function Help>Show hover or by pressing F2.

Templates

In this section you can configure the texts that pop-up when the mouse hovers over an image, a slide, a collection, etc. You can do this for each item individually.

The hover texts are controlled via templates. Templates are constructed from constant text parts and variables. You can type in both, but it is much more comfortable and safe to select the variables from a list by clicking the button Insert variable... or Insert metadata.... The later refers to the different fields defined with an image such as EXIF and IPTC data.

A special case is the variable {metadata}. This variable will create a whole metadata block, containing all metadata fields as configured in the Metadata preferences in section Hover.

When a variable or metadata cannot be resolved (e.g. a field is undefined) the variable is simply deleted from the template. This is sometimes not sufficient: you may want to delete some extra text with it. For this purpose a special syntax is provided:

?
 Conditional line 1
A question mark at the beginning of a line. The whole line will be deleted when all variables and groups within that line remain unresolved.
Example:
?Shot in {meta=iptc_location_created&city}
??
 Conditional line 2
Two question marks at the beginning of a line. The whole line will be deleted when at least one variable or group within that line remains unresolved.
Example:
??{meta=owner}, {yyyy}
(? ... ?)
 Conditional group 1
The complete expression between the parantheses will be deleted if all variables and subgroups within the paranthesis remain unresolved. This works across line breaks, too.
Example:
(?Shot in {meta=iptc_location_created&city}?)
(?? ... ?)
 Conditional group 2
The complete expression between the parantheses will be deleted if at least one variable or subgroup within the paranthesis remains unresolved. This works across line breaks, too.
Example:
(??{meta=owner}, {yyyy}?)
...|(...) Fallback
If the expression (variable, conditional group, or another fallback) in front of the | is unresolved, the expression within the parantheses is used instead. Otherwise it is discarded, This works across line breaks, too.
Example:
 {meta=iptc_location_created&city}|(Not geotagged)

Conditional groups and fallbacks can be nested.
This mark-up is finally removed from the template when the hover text is generated.

Timing

In this section you can configure the hover timing:

  • Delay. This is the time between the detection of a mouse hover event and the actual pop-up of the hover text. Increase this time if hover pop-ups regularly conflict with normal mouse operation.
  • Base time. The minimum time the hover texts are shown.
  • Time per character. This is the time added to the base time for each character in the hover title and hover text. This allows you to adaapt the hover behaviour to your reading speed.
