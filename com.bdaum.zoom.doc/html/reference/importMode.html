Import Mode

When importing images you have the choice between two strategies:

  • Import in situ. WIth this option all imported files stay at their original location. A catalog entry pointing to the file location is created for each of these files.
    This even possible for removable media. When the medium is removed the catalog entry appears as an offline entry but all operations that require the high-resolution image will fail. Mounting the medium again will fix this problem.
    Another problem in this mode is the (unlikely) creation of DNG files during RAW file import from read-only media. This is not possible and will create an error message.

  • Import into new folder structure. When choosing this option you are prompted to specify the target folder. All imported files are copied into that target folder. The file copies can be organised into subfolders of the target folder, depending on your choice.
    In addition, all files can be renamed by choosing an appropriate renaming template. If no renaming is wanted choose "Original file name".
Please note the when importing from a device the import is always performed into a new folder structure.
