Paste metadata

This dialog allows you selecting meta data to be pasted into selected images. The meta data must previously have been copied into the clipboard using the menu function Metadata>Copy Metadata.

Select the meta data items that you want to paste by marking them. In addition select as paste mode:

  • Overwrite. This will overwrite existing entries into the selected images.
  • Fill in blanks. This will paste meta data only into entries that are still empty. Existing meta data of images is not destroyed.
