Split Catalog


If images are selected, this function will export the selected images into a separate catalog. If a collection is selected in the Catalog view, the function will do the same for all images in the selected collection.

After specifying the target file, a catalog description and a new optional time line, the export process will take place. All other properties of the new catalog are taken from the current catalog.

After finishing the export process, the Properties dialog of the new catalog is shown and can be edited.

Optionally, you can delete the exported catalog entries from the current catalog. Catalog images that are used in slide shows, exhibitions, or web galleries are not deleted.
