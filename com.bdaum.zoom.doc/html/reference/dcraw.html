Obtaining a DCRAW module


DCRAW is the raw engine used in ZoRa PhotoDirector. DCRAW is provided as a free raw converter by Dave Coffin in source form. Several web sites provide binary modules for the different platforms. Because new cameras introduce new raw file formats, DCRAW is updated frequently.

Windows

The Windows version of ZoRa PhotoDirector come with a built-in DCRAW module for both 32- and 64-bit platforms. This DCRAW module is updated with each new release of ZoRa PhotoDirector.

Newer DCRAW modules can be downloaded from
  • http://www.fastpictureviewer.com/downloads/#links
To install a custom DCRAW module, simply drop it into the sub folder dropins/ in the ZoRa Photo Director installation folder. Alternatively, you may specify the location of the DCRAW module under Window>Preferences>Import>Raw conversion.

Linux

Under Linux ZoRa PhotoDirector relies on the DCRAW module installed in the Linux platform. If you keep your platform up-to-date, you should always have the latest version of DCRAW.

Mac OSX

The Mac OSX version of ZoRa PhotoDirector comes with a built-in DCRAW module. However, this module is somewhat outdated (Version 9.16). Newer versions of DCRAW for Mac OSX are found on the web but usually they do not run on all OSX platforms. Between OSX versions, library conventions were changed, so that modules compiled on a newer platform do not run on an older platform. So, it is a bit trial and error.

The best choice is probably downloading the latest source code from Dave Coffin's web site and compiling it yourself. This can be done by the following steps:

  • Install the XCode IDE from Apple (free)
  • Install the Command line tools from within the XCode IDE.
  • Run  
    gcc -o dcraw -O4 dcraw.c -lm -DNODEPS

New DCRAW modules can simply be dropped into the sub folder dropins/ in the ZoRa Photo Director installation folder. Alternatively, you may specify the location of a DCRAW module under ZoRa Photo Director>Preferences>Import>Raw conversion.
