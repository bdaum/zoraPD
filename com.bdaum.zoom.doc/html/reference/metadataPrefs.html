Metadata preferences

  • Essential Metadata. Views displaying meta data such as the Properties view, the Exif view, the IPTC view and the Metadata view can be switched between displaying all meta data fields or only essential meta data fields. Here you can configure which fields are considered as essential.
  • Hover. Here you can configure which meta data fields are displayed in a pop-up window when the mouse hovers over an image.
  • Tolerances. Here you can configure which tolerances are applied to the individual fields when a search with the relation similar is performed.
  • Export. Here you can configure which metadata fields are written when manually or automatically exporting metadata. Optionally, you can force metadata to be exported into original JPEG files. Note, that this will destroy the original XMP metadata contained in the file.
  • Tuning. Here you can configure which metadata fields shall be indexed in the catalog database. Database indexing (not to be mistaken for text and similarity indexing) speeds up queries for these fields considerably while slightly slowing down imports and deletions, and increasing the catalog size. Please note that it is not possible unmarking the metadata fields indexed by default. Changes made in this section become effective after reopening of the catalog.
