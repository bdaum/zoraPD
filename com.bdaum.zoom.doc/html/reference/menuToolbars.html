Menus and Toolbars

Items for the main menu and the main tool bar can be added through the Eclipse extension point org.eclipse.ui.actionSets. Constants for menu and tool bar path specifications are defined in interface com.bdaum.zoom.ui.IZoomActionConstants.

Items for view menus and context menus are added through the Eclipse extension points org.eclipse.ui.popupMenus and org.eclipse.ui.viewActions. The separator IDs for adding new items to those menus are defined in interface com.bdaum.zoom.ui.IZoomActionConstants.
