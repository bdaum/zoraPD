Categorize


Categorizing images is regulated by the ITPC standard. This standard allows at most one primary category per image but additionally an arbitrary number of supplemental categories.

Categories are defined in ZoRa PhotoDirector through the Catalog properties. These definitions result in Category collections in the Catalog view. Each of these collections contains images with matching primary categories.

The Categorize command allows for quick categorization of a large number of images. To invoke it, first select some images, then call the context function Categorize ... . In the following dialog you see the current image on the left side and the category tree in the middle. You can now assign main and supplemental categories by clicking the tree nodes. Tree nodes can have five states:

Not set
 Node has background color
 Click to change the state to Supplemental
Alt-click to change the state to Primary
Supplemental
 Node is filled yellow
 Click to change the state to Primary
Alt-clickto change the state to Not set
Primary
 Node is filled red
 Click to change the state to Not set
Alt-click to change the state to Supplemental
Proposal
 Node has yellow frame
 Click to change the state to Supplemental
Alt-click to change the state to Not set
Supplemental
Proposal
 Node has red frame
 Click to change the state to Primary
Alt-click to change the state to Not set

At the right hand side you see a list of automatically derived category proposals with their confidence scores. Matching categories are already marked as proposals in the category tree in the middle.You may either drag non-matching proposals onto a category for adding it as a synonym to this category, or Ctrl+drag the proposal to add it as a category or subcategory to the catalog.
Alternatively, you may mark a proposal in the proposal list to add it to the current image as a keyword.
Note, that this area is only visible if the AI component is installed.

Below the category tree you find a Configure... link. This invokes the Catalog properties for configuring the categories. On the right hand side a Accept link is shown when categories are marked as proposals. Clicking this link will convert all proposals into primary or supplemental categories.
Another Configure... link is below the proposal list. This invokes the User preferences for configuring the AI services.

In addition to defining categories and keywords it is also possible to enter image descriptions and to set the images privacy level.
    The buttons in the button bar allow navigating between images:

    • Back. Go back to previous image. Alternatively you can use the CURSOR-LEFT key.
    • Next. Go forward to next image. Alternatively you can use the CURSOR-RIGHT key.
    • Clone. Press this button for quickly assigning the same categories to a series of images. It will copy the category definitions from the previous image, apply it to the current image and the go forward to the next image. Alternatively you can use the CTRL+CURSOR-RIGHT key combination.
    • Done. Commit all changes and close the dialog. (You may still Undo the changes later on).
    • Cancel. Discard all changes and close the dialog.
