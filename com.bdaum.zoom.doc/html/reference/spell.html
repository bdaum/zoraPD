Spell checking

Spell checking is automatically applied when entering longer texts or new keywords.

Invalid words are underlined in red color. With a right mouse click you can accept a suggested replacement or - depending on the dictionary type - add the invalid word to the dictionary or ignore the word for the rest of the session.

A precondition for spell checking is that the necessary dictionaries are installed. On Windows and Linux platforms ZoRa PhotoDirector will try to find pre-installed dictionaries.

You may, however, install additional dictionaries into the dictionaries/ subfolder of the installation folder. Suitable dictionary files are all HunSpell or MySpell dictionary files as used in OpenOffice, LibreOffice, Firefox, and other products. You can download such dictionary files from the OpenOffice extension page (http://extensions.services.openoffice.org/en/dictionaries). You may either drop the whole .oxt file into the dictionaries/ subfolder or unpack the .oxt file (.oxt files are zip-files) and drop the obtained .dic and .aff files into the dictionaries/ subfolder.

The dictionary to be used can be set for each catalog separately. To do so, please open the Catalog Properties and select a langage.
