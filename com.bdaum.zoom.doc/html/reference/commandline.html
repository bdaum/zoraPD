Command line parameters

Command line parameters can be used to modify the behavior of the ZoRa Photo Director and to adapt it to specific needs.

ParameterDescriptionExamples
-dataDetermines the location of the area where user preferences are stored. By default, these values are stored in the workspace/ subfolder of the installation folder. By specifying a different path after -data, this area can be placed anywhere. There are a couple of predefined path abbreviations:
  • @user.home stands for the user area of the operating system such as C:\Documents\username. 
  • @user.dir stands for the installation folder such as C:\Program files\ZoRaPD. 
-data @user.home/ZoRaPD
-nlDetermines the language of the user interface. Specify an ISO language code after the parameter.
If omitted, the language will be determined based on the platform language.
(Currently, only English and German are supported).		-nl en
-nl de
-noSplashZoRa Photo Director will start without displaying the splash screen.-noSplash
-noBirdie
 Inhibts the occasionally birdie in the splash screen.
 -noBirdie
-vmPath to the Java Virtual Machine (JVM). Used when you want to run the program with a different JVM than the standard JVM installed on the platform.-vm C:\jre_path\bin\java.exe
