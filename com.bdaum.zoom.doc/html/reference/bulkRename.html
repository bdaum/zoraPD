Bulk renaming

This function will rename all selected images with the help of the chosen template.
  • First choose what should be affected by the renaming operation:
    • Both file name and image name (default)
    • Image title (IPTC)
  • Second select a template from the list. There are several predefined templates but you can also add (Add...) your own and edit (Edit...) templates. Templates consist of constant text parts and variables enclused in curly brackets.
    • In the Cue field you may want to supply a short string that will replace {cue} variables within templates.
    • In the Start at field you may want to supply a start value for the variable representing the sequence number.
