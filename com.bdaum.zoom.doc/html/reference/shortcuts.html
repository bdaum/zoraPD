Contributing keyboard shortcuts

You may want to contribute keyboard shortcuts because

  • you have created a ZoRa Photo Director extension with own commands that require own keyboard shortcuts
  • you want to contribute keyboard shortcuts for a platform and locale with different shortcuts
New keyboard shortcuts can be defined by registering them at the extension point org.eclipse.ui.bindings. Please refer to the Eclipse documentation of extension point org.eclipse.ui.bindings for detailed information.
Examples for existing shortcut definitions are found in file com.bdaum.zoom.ui/plugin.xml. You should analyze this file in order to avoid conflicting definitions of shortcuts.
