Parameters for transfer folders

For each transfer folder you may various parameters influencing the import process when images in these folders are transferred to their target destination:

  • Metadata. The values in these fields are added the the metadata of each image that is imported.
    • Artist. The artist who took the photo. Leave this field empty if various artists use the same transfer folder. It is good practise that each artist uses a different transfer folder.
    • File source. Define how the image was created, e.g. a scanner or a digital camera. Set this field to Unknown when the images in this transfer folder are generated by different sources. It is good practise to use different transfer folders for each file source.
  • Skip duplicates. Set this option if you do not want to import the same file twice. If you do not check this option, the image will be imported again but stored under a slightly modified name.
  • Skipped file types. Use this option to skip certain file types. For example if your camera produces both raw and JPeG files you may want to only import the raw files.
