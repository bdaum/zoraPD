File renaming templates

File renaming templates control the renaming of files when images are imported from a device or if images are imported into a new folder structure. There are several predefined templates but you may create your own templates as well.

Each template consists of constant characters (except '{' and '}') and variables enclosed in {}. Templates with only constant characters are rejected, because each image would receive the same name.
