Image Viewer

Identifier:
com.bdaum.zoom.ui.imageViewer

Since:
0.9.4 Updated with 2.0.0

Description:
Viewers for specific media types

Configuration Markup:

<!ELEMENT extension (imageViewer+)>

<!ATTLIST extension

point CDATA #REQUIRED

id    CDATA #IMPLIED

name  CDATA #IMPLIED

>


<!ELEMENT imageViewer EMPTY>

<!ATTLIST imageViewer

formats CDATA #REQUIRED

class   CDATA #REQUIRED

name    CDATA #REQUIRED

id      IDREF #REQUIRED

>

Image viewer extension. Image viewer extension can integrate internal and external viewers for additional media types such as video or audio.


  • formats - Supported file formats in uppercase. Separated by blanks. These are the file types as provided by EXIFTOOL. The file extensions should be given, too, without the period.
  • class - IImageViewer implementation
  • name - Name if viewer (e.g. Video viewer)
  • id - Unique ID of image viewer extension

Examples:
VideoViewer
