Mouse preferences

The Mouse preference page allows you configuring the behavior of the mouse wheel and mouse movements.

  • Speed
    • Zoom and Pan. Changes the mouse sensitivity in the presentation views (slide, exhibition and web gallery editor), the image viewer, and the Sleeves view.
    • Wheel zoom inertia. Determines the lag of mouse wheel movements when zooming with the mouse wheel.
  • Control keys
    • Zoom. Determines the modifier key that switches from pan to zoom when dragging the mouse in the Sleeves view, Exhibition editor, Slideshow editor, Webgallery editor and Image viewer:
      • Alt. Keep ALT pressed while zooming with the left mouse button. release ALT to pan instead.
      • Shift. Keep Shift pressed while zooming with the left mouse button. release Shift to pan instead.
      • Right mouse key. Use the right mouse button to zoom, the left mouse button to pan.
      • No key. When using a touch sensitive display you can zoom by using the typical two-finger-gestures, or pan using one finger only. Otherwise you can zoom with the scrollwheel and pan with the left mouse button.
