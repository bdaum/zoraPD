General preferences

  • Display
    • Output ICC-profile. With standard monitors you should leave this setting at sRGB. If you use a Wide Gamut Monitor, switch to Adobe RGB. Otherwise colors could look exaggerated. For cinematographic monitors the color profiles Rec. 709, Rec. 2020, DCI-P3, P3-D60,and P3-65 are available.
      If correct colors are not an issue, switch to Disable CMS which will speed up the display of images. See also Color Management.
    • Use quality interpolation. Check this option to utilize quality interpolation when viewing high-res images. Otherwise, the faster nearest-neighbour algorithm will be used, resulting in slight pixelation. Depending on the graphics memory available, very large images might not display at all when quality interpolation is used. In such a case switch back to normal interpolation.
    • Processors used for image processing. Specify how many cores of you CPU are used simultanously for image processing. High number lead to faster image processing but could temporarily slow down other tasks.
  • Internal viewers
    • Precede high-res image with preview image in internal viewer where possible. If this option is switched on, you will see an image of good quality (the preview image) earlier, but the final high-res image will be shown later due to the extra processing. Also, the preview image may differ in color and brightness from the high-res image.
    • Hide thumbnail JPeG artifacts in internal image viewer. When this option is checked, noise is added to the enlarged thumbnail in the internal image viewer in order to hide the ugly JPeG artifacts.
    • Enlarge small images to full screen. Mark this option if you want to small images in a larger size than the original. The quality, however, may be bad.
    • Use secondary monitor for image viewer. This option can be chosen on dual monitor systems when the main workbench is run on the primary monitor. The internal image viewer will then open on the primary monitor, the secondary monitor, or alternatingly on both the primary and secondary monitor depending on your choice.
    • Slideshow inactivity timeout. Specify a maximum interval for pausing the slide show. If this interval is exceeded, the slide show is aborted.
  • Backup and Program Update
    • Backup interval. Specify the number of days between each backup. The backup will be performed silently. The backup location can be specified for each catalog separately in the Catalog Properties.
    • Maximum backup generations. Please specify how many generations of backups you want to keep. Older backups will be deleted after a new backup was written.
    • Check for updates. Determines when to check for new program versions. This can be done manually (Help>Check for Updates) or automatically with each regular back-up (recommended) or on each program start.
  • Miscellaneous
    • Undo levels. Determines how many steps can be undone. A high value will need more memory. The default value is 9.
    • Supress progress indicator for background activities. Mark this option to make background activities invisible.
