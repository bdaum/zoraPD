Keyword preferences

The list of filter expressions on this page determines which keywords will be added to the list of defined keywords (as displayed in the Catalog Properties) in the catalog when images are imported. Some photographic tools, for example, include the GPS coordinates into the images keywords. Adding all of these to the list of defined keywords would lead to a huge and noisy keyword list. 

When a new keyword is about to be added to the list of defined keywords, the filter expressions are compared to the new keyword one by one beginning at the top of the list. When an expression matches with the keyword, the keyword is either rejected or accepted, depending on the definition of the filter expression. If the filter expression does not match, the keyword will be compared with the next filter expression in the list, and so on. If none of the filter expressions match, the keyword is accepted.
Filter expressions may contain wildcards:
  • '*' matches with any string of arbitrary length
  • '?' matches with one and only one character
