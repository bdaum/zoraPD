Section break properties

The section break dialog allows you editing or defining several properties for individual section breaks, aka title slides. It typically describes the following section of slides until the next section break or the end of the slideshow. The following properties can be defined in excess of the standard slide properties:
  • Title. A title for the following section.
  • Description Some text describing the following section.
  • Layout:
    • Text only. Only the title and the description are shown.
    • Thumbnails left(right/top/bottom. This will show an overview of the following slides within the title slide.
    • Map left/right. This will show a map of image locations of the following slides, provided that these images are geotagged. This feature is only available when the geo component was installed. Note, that an internet connection is required when playing the slideshow.

The dialog also features a Compile until field. Select a number of slides to be analysed. Information such as the location, the creation date, model information, and artworks or objects shown is collected and placed into the Description field. In addition, a layout is proposed. You should use this function before entering other information as the Description field and the Layout field are overwritten.
