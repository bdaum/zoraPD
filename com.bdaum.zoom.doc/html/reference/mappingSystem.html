Mapping System

Identifier:
com.bdaum.zoom.gps.mappingSystem


Since:
0.9.4


Description:
Extension point for declaring mapping systems


Configuration Markup:


<!ELEMENT extension (mappingSystem+)>

<!ATTLIST extension

point CDATA #REQUIRED

id    CDATA #IMPLIED

name  CDATA #IMPLIED

>

<!ELEMENT mappingSystem EMPTY>

<!ATTLIST mappingSystem

mapComponent CDATA #REQUIRED

id           IDREF #REQUIRED

name         CDATA #REQUIRED

>

A mapping system is able to display location in a map and to define new locations


  • mapComponent - Map component that appear in the Map View
  • id - Unique ID of mapping system
  • name - Name of mapping system

Examples:
Google Maps (default implementation)


API Information:
IMapComponent


Supplied Implementation:
Google Maps (default implementation)
