Audio preferences

  • Signals
    • Acoustic alarm on user prompt. Check this option if you want to be prompted with an acoustic signal.
    • Acoustic signal on operation completion. Check this option if the completion of an operation should be indicated with an acoustic signal.
  • Voice notes
    These preferences affect the quality of voice notes.
    • Sampling rate.  By default the sampling rate is set at a medium quality setting (22 kHz). If you need higher quality, choose a higher sampling rate. If disc space is an issue, choose a lower sampling rate.
    • Bit depth. By default the bit depth is set at 8 bit. If you need higher quality, choose a higher bit depth.
