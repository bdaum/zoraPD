Print layout dialog

The Print layout dialog determines the layout of a printed page.

On page level you can set the following items:

  • Title
  • Subtitle
  • Footer
  • Left and right margins in mm.
  • Top and bottom margins in mm.
  • Horizontal and vertical gap in mm.

You may insert variables into all text entries :

VariableDescription
{catalog}Catalog name
{today}Current date
{count}Number of images
{pageNo}Page number
{pageCount}Total number of pages
{collection}Title of current collection
{user}Current computer user
{owner}Catalog owner

On image level you can set the following items:

  • Caption line 1
  • Caption line 2
  • Key line width

Into all text entries you may insert the following variables:

VariableDescription
{title} Image title or image name if no title is defined
{name}Image name
{creationDate}Image creation date
{collection} Title of current collection
{sequenceNo}Image sequence number
{pageItem}Image number on page
{size}Image size in pixel
{format}File type
