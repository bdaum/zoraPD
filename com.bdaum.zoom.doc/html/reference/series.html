Series view

The Series view is used to show the result from the function Find Series Images belonging to the same series are organized in groups. The title bar of each group shows the thumbnail and the name of the first image in the series. Groups can be expanded and viewed by clicking on the + button.

Navigation in the Series view is performed in the classic way using the vertical scrollbar.

The size of the thumbnails can be adjusted using the Zoom slider in the views title bar. Alternatively, the size of the thumbnails can be adjusted with the mouse wheel or with the + and - keys.

For more key commands and mouse gestures please see the general Galleries help page.
