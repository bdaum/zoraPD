FTP Account Configuration

This dialog is used to create a new FTP account or to modify an existing account definition.

  • General
    • Account Name. The name of the account as it will appear in the user interface.
    • Host. The host name of the the FTP account such as xyz.de or abc.com.
    • Target Directory. The directory on the host where the uploaded items are stored.
    • Track exports. A log entry of the export is kept in the images metadata. Later, you may view the track in the Properties view under File>Track.
    • Log-in. The user name for the FTP account.
    • Password. The password for this user name.
    • Account. A subaccount if available.
    • Anonymous. Mark this field for anonymous access. Log-in, Password and Account are not required in this case.
  • Connection
    • Passive Mode. By default turned on. The client is responsible to establish the connection. This is supported by most servers. Otherwiese, in active mode, the server is responsible for establishing the connection.
    • Port. The communication port used for FTP. Typically, this is port 21 (default).
  • Web
    • Host. Specifying a Web URL for accessing the web presence of the FTP account. This allows to immediately invoke a web browser after transmission and to view the uploaded content. 
    • Prefix. A specific directory on the FTP host if the web presence does not represent the whole FTP account but only a subdirectory as defined in the prefix. If specified, the Target Directory (see above) must start with this Prefix.
    • Notes. Some arbitrary notes.
