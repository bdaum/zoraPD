Meta data views

Meta data in images and/or associated XMP sidecars is displayed by the following views:

  • Properties view. This view shows the basic properties of an image, properties that are available also for images that are not equipped with EXIF meta data. These meta data is grouped into the following sections:
    • File. This section contains file related information such as file size, modification date, name, volume, URI.
    • Financial. This section contains financial information such as sales, price, and income. The data can be edited.
    • Image. This section contains image related data derived from the image such as dimensions, samples per pixel, etc. but also information entered when importing analog images.Entries starting with a chevron (») can be edited. Simply click the entry in the value column.
    • ZoRa Photo Director. This section contains data maintained by ZoRa Photo Director such as Color Code, Rating, Status, Track, and User field1 and 2. 
  • Exif view. This view shows Exif data which usually is generated by the camera and by geo-tagging operations. Entries starting with a chevron (») can be edited. Simply click the entry in the value column. The data is organized into different sections:
    • Auxiliary. Contains data such as serial numbers or maker notes (see also Import Preferences)..
    • Camera settings. Contains data such as shutter speed or ISO.
    • GPS. Contains the images geographic coordinates but also speed and altitude.
    • Image data. Contains data such as the image creation date, camera make and model, and more.
  • IPTC view. The IPTC view contains IPTC data. This administrative and descriptive data is usually supplied after the image was taken. Hence, all of the entries of the IPTC view are editable. Simply click the entry in the value column.
    • Administration. Contains data such as the event where the image was taken, the location where it was taken, or the date when the IPTC information was last edited.
    • Classification. Contains data such as Category, Keywords, and Urgency.
    • Description. Contains data such as the title, the persons and objects shown in the image.
    • Origin. Contains data such as the depicted location, the scene code, or the subject code.
    • Rights. Contains data such as the creator and the owner of the image.
  • Metadata view. This view combines the above three views into one and list their entries in three different sections.

Local menu functions
You can access local menu functions by clicking the triangle symbol on the local toolbar.
  • Editable. Only editable entries are shown.
  • Essential. Shows only essential entries. Which those are, can be configured (see below).
  • All. Show all entries.
  • Configure. Jumps to the Metadata preference page, where you can configure which entries are considered as essential.

Local tool buttons
  • Delete sub-entry (). This tool button is used to delete a sub entry in multi-dimensional entries. Simply select the sub-entry and click this button.

Each view has three columns: Name, Value and Action. Icons displayed in the Action column can be clicked to invoke an action:

Find all images with the same value in the selected entry.
Pressing Shift will restrict the search to the current collection.
Show coordinates in the map
Open URL in web browser
Send email to the specified address
Show folder

Note: Entries with a pencil symbol () at the left side of the Name column can be edited. Simply click the entry in the Value column to invoke a dedicated editor.
