Appearance

Color scheme and thumbnails
  • Color scheme.You may choose between black, dark grey, grey, white and platform. Text colors are adapted automatically. "Platform" is the default color scheme of the operating system. 
    Additional color schemes can  be added by placing CSS style sheets into the dropins/ subfolder of the installation directory.
  • Thumbnails
    • Show status mark. Check this option to display a green status mark for images with status "Ready for publishing" or RAW images with status "Corrected".
    • Max. number of displayed regions. Limits the number of regions (face, pet, focus, barcode) shown in a thumbnail.
    • Show location pin. The location pin informs about the presence of location data and allows access to the map view with a single click.
    • Show color code in thumbnails. Select an option to display color codes in thumbnails:
      • No. No colorcodes will be shown in thumbnails
      • Automatic. Color codes are automatically determined by the rules defined in the Automatic Color Coding box (see below). Manually set color codes override the automatic codes.
      • Manual. Only manually set color codes are shown.
    • Show rating in thumbnails. Select by size to show the rating as one differently sized star. Select by count to show the rating as a variable number of stars.
    • Show rotate buttons in thumbnails. Check this option to display the rotate buttons at the bottom of the thumbnail.
    • Show voicenote button in thumbnails. Check this option to display the voicenote button at the bottom of the thumbnail.
    • Show Expand/Collapse button in thumbnails. Check this option to display the + or -  buttons at the top of the thumbnail for controlling the expanded/collapsed state of the thumbnail. Collapsed means that subsequent images with the same name are shown as one thumbnail.
  • Automatic Color Coding. This group is only shown if the automatic color code option was chosen.
    For each color code you may specify a criterion selecting the images for which the color code is to be shown (similar to specifying a collection criterion).

Color codes
  • Show color code in thumbnails. Select an option to display color codes in thumbnails:
    • No. No colorcodes will be shown in thumbnails
    • Automatic. Color codes are automatically determined by the rules defined in the Automatic Color Coding box (see below). Manually set color codes override the automatic codes.
    • Manual. Only manually set color codes are shown.
  • Automatic Color Coding. This group is only shown if the automatic color code option was chosen.
    For each color code you may specify a criterion selecting the images for which the color code is to be shown (similar to specifying a collection criterion).

    Note: Automatic color coding does not modify the meta data of images but only influences the display of color codes in the user interface. The consequence is that if you query for a specific color code in Properties>ZoRa Photo Director>Color code automatic color coding has no effect. But if you use the Color code filter found in the local toolbars of galleries, automatic color coding is considered.
