External programs


General

  • Export metadata before invoking an external program. Check this option if you want to export the meta data stored in the catalog as an XMP sidecar before invoking an external program such as an image editor or image viewer so that the external program can take advantage of the updated meta data.
  • External viewer. If you want to use an external viewer for viewing images at high resolution you can specify its path here. If you leave this entry empty, the internal viewer is used, instead.

File associations

Files associations are used to determine which external editor should be launched for specific image types. 

  • File types. This section lists file types to which editors can be associated. You may add new file types using the Add... button. Each file type can have several file suffixes. For example, *.jpg, *.jpeg, *.jpe, *.jfif are all valid file suffixes for JPEG-files.
  • Associated editors. To associate editors to a file type, select the file type in the upper section. Then press the Add... button in the lower section and navigate to the editors executable. To designate one editor as the default editor, select the editor and press the Default button. Default editors can be launched using the Edit () action in the context menu of a gallery. The other editors can be launched by invoking the Edit with... () action and selecting the editor from a list.
    For convenience, you may press the buttons Apply System Settings to retrieve already existing file associations from the operating system.
  • Remember last editor. If this option is checked, each image will remember the last editor that was used for it. This editor will then be invoked in lieue of the default editor.
