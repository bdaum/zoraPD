Albums

Albums are similar to collections but they are not defined through queries. Instead, images are simply pulled from a gallery into the album icon. The sequence of images within an album is determined by a sort expression, similar to the ordering of collections. If you require an image set where the sequence of images is determined on an image-by-image basis, use a slide show instead.

An image may belong to several albums. To remove an image from an album, use the context function Remove from Album on the image thumbnail.

To create a new album, use the context function Create new Album in the Catalog view.
