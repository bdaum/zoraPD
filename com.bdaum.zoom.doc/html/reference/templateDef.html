Template definition

User defined templates need a name that is entered in the very first field of the dialog.

In the second field the template is constructed from constant text parts and variables. You can type in both, but it is much more comfortable and safe to select the variables from a list by clicking the button Insert variable...

The third field shows the text string resulting from the defined template. The variable values are taken from the very first image of the current image selection.

In case the templates are used for bulk renaming, most metadata fields can be added to the template as well. To do so, click the button Insert metadata...
