This preference page allows you selecting among different similarity algorithms. When you select an algorithm, a short description is displayed below the selection field. Which search method is the best, depends very much on the character of your images. For example, in a catalog containing mostly texture images, you may want to use the Tamura texture filters. With the average photographic content we found the methods CEDD and FCTH quite efficient.

Each query returns with each image a score between 0 and 100 percent. You may limit the amount of returned images either by number or by specifying a minimum score that must be satisfied by each image.

The section Advanced Options allows you configuring the indexing process. Only the selected algorithms will later be available for search. The number of selected algorithms will determine the size of the index and the speed of indexing.
