Basic Concepts

  • Images are never modified by ZoRa Photo Director. All changes applied to images such as the Rotate operation or Raw Recipes affect only thumbnails in the catalog and/or the display of images, but not the image in its stored form.
Exception: Optionally XMP-metadata can be exported into original JPEG files. See (Window>Preferences>Metadata>Export*)
  • No image editing facilities. We believe that image editing is performed best with your favorite image editor. You may register a different editor for each file suffix (Window>Preferences>File Associations*) .
  • Meta data are exported in the form of XMP sidecars. You have the option to export the metadata automatically before invoking an external editor (Window>Preferences>General*) so the external editor will see all changes you have applied to the meta data (provided the external editor supports XMP).
    The exception are JPEG images that are generated by ZoRa Photo Director for upload or e-mai purposesl. In this case, selected meta data is written into the generated JPEG image.
