Face detection and recognition

ZoRa Photo Director does not provide facilities of its own for face detection and recognition but relies on face data obtained from other sources such as Google's Picasa, Windows Live Photo Gallery, Adobe Lightroom or the camera.

ZoRa Photo Director is able to handle face data from mixed sources. Face areas from different sources will be united to one area if appropriate.

Each person will be displayed as a separate album within the Persons group. Images where the face of a specific person was detected are added automatically to the corresponding album.

It is possible to add and remove images to and from person albums in ZoRa Photo Director freely. To add an image to a person album drag the image to the album in the Catalog view. If you grab the image inside a region, the region will be renamed to the name of the album. Otherwise, the unnamed region closest to the mouse pointer will be renamed.
Alternatively, you can click a region to add the image to a person album.

Please be aware that the deleted entries can reappear in an album as soon as the album is re-synchronized after a corresponding Picasa album was modified.

Metadata

When exporting metadata (XMP file) face related data is exported, too. The formats used are the Microsoft Photo Region Schema as used by Windows Live Photo Gallery, and the EXIF MWG format as used by Adobe Lightroom 6. 

Queries

Because images are automatically added to person albums, it is not necessary to define queries regarding persons. However, it is possible to add person names to the IPTC keywords of an image via Catalog>Properties>Keywords, so that images containing certain persons can be found easily via the keyword search. In addition, images containing persons can also be found by using the text search facility (Catalog>Properties>Indexing).
Another option is querying for images containing a given number of faces (Group Properties, Subgroup Image, Field No of detected persons).
 

*) Note for Linux users: On Linux systems Picasa runs under Wine. Depending on your system and the installation method Picasa file locations vary. If you have trouble accessing Picasa data, please configure the file picasaContacts.ini in plug-in com.bdaum.jna.linux.x64 or com.bdaum.jna.linux.x86.
