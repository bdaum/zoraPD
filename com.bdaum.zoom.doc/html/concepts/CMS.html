Color Management

  • When importing images, thumbnails are converted from the source image color space into the sRGB color space. 
  • Viewing of images can be controlled through Window>Preferences>General>Output ICC-profile*:
    • Disable CMS. This is the fastest option but can result in wrong colors
    • sRGB. Use this setting for standard monitors (default)
    • Adobe RGB. Use this setting when working with Wide Gamut Monitors
    • Rec. 709, Rec. 2020, DCI-P3, P3-D60, P3-65. Use these settings when working with cinematographic monitors.
  • Images exported to email and web are always converted to the sRGB color space.
  • Also, printing of thumbnails occurs in the sRGB color space.
