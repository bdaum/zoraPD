Working with keywords

A common problem when defining keywords for a set of images is keyword inflation caused by different word forms, spelling errors, or use of synonyms. For example the keywords "children", "child", "chilrden", or "kid" may all mean the same thing.

For avoiding keyword inflation the keyword definition dialog offers a few aids:

  • A list of recently used keywords. Using a keyword from this list saves you some typing or scrolling, and you can be sure that the selected keyword already exists.
  • A list of available keywords. You may want to scroll through that list and select keywords from there. Again, you can be sure that the selected keyword already exists.
  • The Suggest... button. By clicking that button, a list of proposed keywords is compiled based on visual similarity. You may probably find the right keyword for your purpose in this list.
  • Only if you can't find the right keyword in one of these three lists, enter the new keyword into the designated field. We recommend that you install a dictionary for enabling spell checking.
  • Not an then have a look at the list of all keywords defined in the catalog. You may want to clean up that list by deleting, editing or replacing keywords.

Some tips for keyword selection (inspired by Robert Kneschke):
  • Use both abstract and concrete keywords. For example when keywording an image showing a speed limit road sign, add a keyword "road sign" and "speed limit".
  • Instead of an adjective use a noun, for example instead of "beautiful" use "beauty".
  • If you plan to sell your images via an agent, ask your agent for a list of keywords that sell, and import this list into your catalog.
  • You may want to look onto the web for finding out how others index similar images and what is a good keyword and what is not:
    • Stocktagger suggest similar keywords based on the vocabularies of Getty Images and istockphoto.
    • Photokeywords and Findphotokeywords search the web for similar images and displays their keywords
    • PicNiche analyses the quality of keywords and suggests similar keywords
