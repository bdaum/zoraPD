Installation

Installation under Windows, Linux and Mac OSX

Download the installation package (JAR-file). Then double click this file to execute it*. Alternatively, you may also right click it, select Open with... and open it with Java. A third possibility is to start the installation process from the command line:

java -jar {installationFile}

Then follow the instructions of the installer. Shortcuts can be generated on Windows and Linux*. On the Macintosh you may want to create Aliases manually, instead.

Installation on removable media

It is possible to install ZoRa Photo Director on removable media such as a memory stick. It is recommended to install a JRE into subfolder jre/ for more portability.
Also catalogs can be kept easily on removable media. Using a medium size for image thumbnails each catalog entry will take less than 20 kb disc space. Even large catalogs with several 10,000 images will fit onto a stick.

De-installation

For de-installation of the application go to the folder {installation directory}/Uninstaller/ and double the file ZoRaPDUninstaller.jar. Alternatively, you may also right click it, select Open with... and open it with Java. A third possibility is to start the de-installation process from the command line:

java -jar {installation directory}/Uninstaller/ZoRaPDUninstaller.jar

You then have the option to keep the user defined application data for later use or to delete it together with the application as well.

*) On Ubuntu you need to mark the installation file and generated shortcuts as Executable (Properties>Permissions).
