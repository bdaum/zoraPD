Data entry

A more convenient method for entering large amounts of data is offered by the Data entry view. This view organizes the various metadata into logical groups. It allows editing single images as well as multiple images in one step.


The Data entry view is best invoked via the Data Entry Perspective (). You can switch to this perspective via the Open Perspective button in the main toolbar.

Hint: Many image attributes are not editable, especially those supplied by a digital camera. In rare cases you might want to modify such an attribute. To do so, first go to the EXIF Camera Settings and modify the file source to an analog device such as Film Scanner. Most attributes are then unlocked for editing. After editing don't forget to reset the file source to Digital Camera.
