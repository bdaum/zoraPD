Working with Scanners

ZoRa Photo Director does not feature an import function for TWAIN scanners. Many photographers would rather not scan through the standard TWAIN drivers anyway but use high-quality scan utilities such as SilverFast or VueScan.

If you want to automate the import of scanned images into ZoRa Photo Director you may scan always into a specific directory and configure that directory as a watched folder (Catalog>Properties>Watched Folders).
