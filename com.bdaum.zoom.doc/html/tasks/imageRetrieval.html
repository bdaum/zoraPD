Searching and Finding Images

There are numerous ways to search for images:

  • Collections can group images under various aspects. Collections are listed in the Catalog View. Some collections are system-defined while other collections can be defined by the user. For example you may want to create a collection containing all images in portrait mode with the keyword "skyscraper". Typically, collections are used for image selections that you want to use frequently. The image selection provided by collections can be narrowed down by adding sub-collections. This can be done within both system-defined and user-defined collections. For example, within the 5-star collection you may want to create a sub collection containing all images with color code "red".
  • Queries are typically used for one-time selections:
    • Adhoc Queries () are defined in the same way as a collection (using the same type of criteria) but are by default not added to the Catalog View. 
    • The Keyword Search () is a quick way to search for single keywords or keyword combinations.
    • The Time Search () allows searching for all images in the same time interval as the currently selected images. This is an excellent way to quickly locate all images from a series.
    • The Proximity Search () allows searching for all images in the vicinity of given images - provided that those images are geo-tagged.
    • The Content based Search () is used for searching visually similar images. This can be based on a given image but it is also possible to paint a sample image. Different similarity algorithms can be selected for performing the search.
    • The Duplicate Search () is used for finding duplicates. This search can be based on the image name, on EXIF data, or on visual similarity (see above).
    • The Series Search () can be used to find image series. This search is dominantly based on the creation time of images but can be customized in various ways.
    • The Search for Orphans () finds catalog entries whose image file does not exist any more or has moved to another location.
    • The Text Search () searches in all text elements connected with an image: keywords, title, description, locations, contacts, and depicted artwork. Also the title and description of derived slides, exhibits, and web exhibits are considered. This search allows for complex search expressions and wild cards. It is very fast.
Queries are either performed on the whole catalog or, optionally, on the currently selected collection. By invoking the gallery menu function Save Query As Collection (), you can convert a query into a collection that will be stored into the catalog. Using this method it is possible to define collections based on the Similarity Search or the Text Search, too.
  • By clicking the button Find similar images () in the Properties view, EXIF view, IPTC view, or the Metadata view, you can search for images with matching attributes. Pressing the Shift button in addition will restrict the search to the current collection.
