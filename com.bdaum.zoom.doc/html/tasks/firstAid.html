First aid

  • Should you experience problems with graphics - especially with very large images - try to un-check the option Use quality interpolation under Window>Preferences>General*.

  • If you get "Out of memory messages" messages, close the application immediately. In the installation folder of the application find the file ZoRaPD.ini. Open this file with text editor such as Notepad. Increase the number given in the entry "-Xmx????m". This will increase the amount of memory that the application can allocate. Press "Save" and restart the application.

  • In case of performance problems consult the Tuning chapter.

  • If you get messages that indicate problems with the Lucene index or you have reason to believe that the Similarity Search, the Text Search, or the Duplicate Search are broken, do the following: 
Invoke Catalog>Maintenance>Refresh Index.
  • Your camera is not supported during RAW import? You may need to download the latest version of DCRAW from http://www.insflug.org/raw/Downloads/. After downloading the binaries place dcraw.exe into dropins/ subfolder of the installation folder, for example into C:\Program Files\ZoRaPD\dropins\.

  • If fading of images does not work in slide shows and image viewer under Linux, make sure that a compositing manager (such as compiz-fusion) is installed and that the Desktop Effects are switched on.

  • Under Windows 8 starting ZoRaPD occasionally fails with a gallery showing the error message "Illegal thread access...". In this case, restart your computer.
