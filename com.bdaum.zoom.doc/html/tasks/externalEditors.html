Invoking external editors and viewers

This application is all about image browsing and retrieval. It leaves the task of modifying images to specialized image editors such as Photoshop, Paint Shop Pro, Picture Window, and others. 

By using a keyboard shortcut (Ctrl+E) you can invoke such editors on the selected images quickly. But before you are able to do that, you must associate those editors with file types (Window>Preferences>File Associations*). Each file type may be opened with a different editor. For example, RAW files can automatically be opened with a RAW converter while TIFF files are opened with a classic image editor. You may even assign several editors to the same file type and then, by pressing Ctrl+Shift+E, select one of these editors from a list.

When double-clicking an image, the image will be opened in a full-screen viewer. If you did not define an external viewer, the internal viewer will be used. This viewer can display all supported file types in full resolution. If you want instead to use your favorite third-party viewer, you must specify its location under Window>Preferences>General*. You will still be able to invoke the internal viewer by pressing Shift  while double-clicking.
