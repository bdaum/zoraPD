Reporting errors

Please file error reports at https://bugs.launchpad.net/zora/
                                 

When reporting errors, please provide the following information:

  • A description of what you have done before the error occurred.
  • Information about the error. Open the Log view by clicking on the error icon () in the Fast View bar (bottom left). Then double click the entry describing the error and copy the information into the clipboard by pressing the button Copy Event Details to Clipboard () (top right) to paste this information into your error report.
  • The system configuration. You can use the information under Help>About ZoRa Photo Director>Installation Details*. Simply select the tab Configuration and press the button Copy to Clipboard to paste this information into your error report.

*) On Mac use ZoRa Photo Director>About ZoRa Photo Director>Installation Details.
