Tips and Tricks


Metadata

  • Modifying non-editable metadata
Most metadata supplied by a digital camera are protected against editing. To modify them anyway, change the attribute Exif>Camera Settings>File Source to an analog device such as Film Scanner. Now, you will be able to edit most of the other EXIF metadata attributes. After you have applied your changes, don't forget to set  Exif>Camera Settings>File Source back to Digital Camera.

  • Bad or missing EXIF Image Creation Date
This can be remedied by supplying a valid data in the field IPTC>Origin>Date Created.

  • Restoring a catalog from a back-up
Catalog consist of a .zdb file and a .zix folder. Back-ups are stored in folders with the file extension .backup. To activate such a back-up
  1. close ZoraPD
  2. wait until the ZoraPD icon in the system tray has vanished
  3. replace your existing catalog with the catalog found in the .backup folder (both .zdb file and .zix folder)
  4. open ZoraPD again.
