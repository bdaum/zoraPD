Creating Web Galleries

Web galleries are another option to publish your images on the internet - provided that you have access to a web site you can upload to. 


You start by creating your web gallery locally with one or more storyboards. To do so, first create a new group (or use an existing group containing web galleries) by invoking the function Create New Group () from the Catalog view drop down menu. Then right click this group and invoke Create new presentation>Web Gallery... (). You are then prompted with the Web gallery property dialog.

After placing and arranging the images on the storyboards, setting the various parameters for the whole gallery, each storyboard, and each image, and selecting a web engine, you can deploy the web gallery. This can be done into a local folder which you may want to upload at a later time, or directly to an FTP account.
