Galleriffic

www.twospy.com/galleriffic/

Features

  • Smart image preloading
  • Thumbnail navigation (with pagination)
  • Slideshow
  • Support for image captions
  • Download Originals link
  • Exif display
  • Flexible configuration
  • Javascript/AJAX based. Graceful degradation when Javascript is not available.
  • Multiple storyboards not supported
