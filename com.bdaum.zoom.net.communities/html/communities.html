Exporting to Internet Photo Communities

Exporting to Internet Photo Communities is invoked via File>Export...>Communities>....

You may either export selected images or export all images of a slideshow, exhibition, or web gallery selected in the Catalog View.

Before being able to export images to a photo community you must first create a new account or select an existing account::

  • Name. The account name as agreed with the service provider.
  • Web URL. The web URL of the specific account. This is used for displaying the account content after images have been uploaded.
  • Track exports. Check this box if uploads should be recorded with each image. Later, you may view these records in the Properties view under  File>Track.
  • Privacy. The varous settings for the privacy of the whole account depends on the specific community. Typical are different restriction levels (private, private and friends, private and family, public).
  • Metadata
    • Assign keywords as Tags. Check this box if tags should be automatically created from keywords.
    • Create Photosets from albums. Check this box if photosets should be automatically created from albums.

The following options are available for the export operation:

  • You have the choice to send either originals (which may be rather large) or to downsize the images and compress them into the popular JPEG format. Raw and DNG files can only be uploaded if the particular community supports this.
  • The size of these JPEGs can be adjusted using the scale. Alternatively, you may enter a maximum size (either width for landscape orientation or height for portrait orientation).
  • You can sharpen the downsized JPEGs to counterbalance the loss in sharpness due to the down-sampling process. An Unsharp Mask filter is used for that purpose, where you can adjust the pixel radius, the sharpening strength, and a threshold value.
  •  You can set the JPEG quality between 1 for low visual quality and small storage size and 100 for best visual quality and large storage size. The default value of 75 combines good compression with virtually no visible artifacts.
  • If you want to include metadata (EXIF and IPTC) into the generated JPEG files, check the option "Include metadata". On the following wizard page you can then select the metadata that you want to export. You may include geodata as well to support geotagging.
  • Optionally, you may add watermarks for the protection of you intellectual property to images (not when uploading originals).
