    Description

    ZoRa Photo Director is a photo managerment program. It is targeted at serious amateur photographers and professional photographers.

    Features

    • Powerful database engine allowing for large image catalogs.
    • Image collections based on query expressions.
    • Advanced search options: Geographic Proximity, Time frame, Visual Similarity, Full text, Duplicates, and Series
    • Support for offline and remote media (mobile devices, Internet, etc.)
    • Tracking relationships between images. 
    • Integrated Geo-tagging and Geo-naming.
    • Support for RAW files with integrated DNG conversion and Raw Recipe Reflection (3R). 
    • EXIF/IPTC data extraction and editing.
    • Spell checking.
    • Automatic timeline.
    • Automatic colorcoding
    • Albums and bookmarks. 
    • Voice notes. 
    • Integrated image viewer and Slideshow. 
    • Integration of external image viewers and editors. 
    • Design facilities for slideshows, exhibitions, and webgalleries. 
    • Virtual exhibitions.
    • Export to Email, PDF, FTP, Flickr, Locr, and Google Earth. 
    • Face detection and recognition. 
    • Optional support for videos including upload to YouTube. 
    • Optional peer-to-peer and multi-catalog support. 
    • Optional support for importing foreign catalogs. 


    Requirements

    This version is released under Windows (Window Vista or later), Linux GTK, and macOS cacao (>=10.4). We recommend system memory (RAM) of 8GB or more.

    Windows
    • For Windows Vista Service Pack SP2 is required.

    Mac OSX
    • Java must be installed on the host platform to run the installer. At least Java 1.8.0_111 is required. We recommend the JRE from http://www.java.com or from http://www.azul.com.

    Linux
    • Java must be installed on the host platform to run the installer. At least Java 1.8.0_111 is required. We recommend the JRE from http://www.java.com or from http://www.azul.com.

    • For processing of RAW images, a recent version of Dave Coffin's dcraw should be installed in /usr/bin/.  Installation packages are available at http://linuxappfinder.com/package/dcraw.

    Installation

    Installation under Windows

    • Download the installation package (EXE-file).
    • Execute it as an administrator.
    • Then follow the instructions of the installer.

    Installation under Linux

    • Download the installation package (JAR-file).
    • You may need to mark the installation file as executable (Properties)
    • Open a terminal window and type
      java -jar {installationFile full path}
    • Then follow the instructions of the installer.
    • If you had created desktop shortcuts, you may need to mark them as Executable (Properties).

    Installation under Mac OSX

    • Download the installation package (JAR-file).
    • Open a terminal window and type
      java -jar {installationFile full path}
    • Then follow the instructions of the installer.
    • Shortcuts can be created manually on the Macintosh in form of Aliases.

    Installation on removable media

    It is possible to install ZoRa PhotoDirector on removable media such as a memory stick. It is recommended to install a JRE into subfolder ZoRa/jre/ for more portability.
    Also catalogs can be kept easily on removable media. Using a medium size for image thumbnails each catalog entry will take less than 20 kb disc space. Even large catalogs with several 10,000 images will fit onto a stick.

    De-installation

    For de-installation of the application go to the folder {installation directory}/Uninstaller/ and double the file ZoRaPDUninstaller.jar. Alternatively, you may also right click it, select Open with... and open it with Java. A third possibility is to start the de-installation process from the command line:

    java -jar {installation directory}/Uninstaller/ZoRaPDUninstaller.jar

    (Under Linux you may need to mark ZoRaPDUninstaller.jar as Executable first (Properties)).

    You then have the option to keep the user defined application data for later use or to delete it together with the application as well.

    Known problems

    • If after updating to a new version the application does not start properly, please delete the old installation folder with all its contents, and perform a new installation.
    • Under Windows there are compatibility problems with Google Maps. Please use another map engine (select in map window)
