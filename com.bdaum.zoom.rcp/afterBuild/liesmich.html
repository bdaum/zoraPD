    Beschreibung

    ZoRa Photo Director ist ein Foto-Management-Programm. Es richtet sich an ernsthafte Hobby-Fotografen und professionelle Fotografen.

    Features:

    • Leistungsstarke Datenbank-Engine für große Bild-Kataloge. 
    • Bild-Kollektionen auf der Grundlage von Abfrageausdrücken. 
    • Erweiterte Such-Optionen: Geografische Nähe, Zeitspanne, Viuselle Ähnlichkeit, Volltext, Duplikate und Serien. 
    • Unterstützung für offline- und remote-Medien (mobile Geräte, Internet, etc..)
    • Verfolgung von Beziehungen zwischen Bildern. 
    • Integriertes Geo-Tagging und Geo-Benennung. 
    • Unterstützung für RAW-Dateien mit integrierter DNG-Konvertierung und Raw-Rezept-Reflektion (3R). 
    • Extrahieren und bearbeiten von EXIF-/IPTC-Daten. 
    • Rechtschreibeprüfung. 
    • Automatische Zeitachse. 
    • Automatische Farbkodierung
    • Alben und Lesezeichen. 
    • Sprachnotizen. 
    • Integrierter Bild- und Diashow-Betrachter. 
    • Integration von externen Bildbetrachter und Editoren. 
    • Design von Diashows, Ausstellungen und Webgalerien. 
    • Virtuelle Ausstellungen.
    • Exportieren als E-Mail, PDF, FTP, oder zu Flickr, Locr und Google Earth. 
    • Gesichtserkennung. 
    • Optionale Unterstützung für Videos, einschließlich Veröffentlichung auf YouTube. 
    • Optionale Peer-zu-Peer-Kommunikation und Multi-Katalog-Suche.
    • Optionale Unterstützung für den Import fremder Kataloge.
    • Optionale Unterstützung für KI-basierte Bildkategorisierung und -bewertung

    Systemanforderungen

    Diese Version ist freigegeben unter Windows (Window Vista oder später), Linux GTK und macOS cacao (>=10.4). Wir empfehlen Systemspeicher von 8GB oder mehr.

    Windows
    • Für Windows Vista wird Service Pack SP2 benötigt.
    Mac OSX
    • Java wird für die Ausführung des Installationsprogramms benötigt. Mindestens Java 1.8.0_111 wird benötigt. Wir empfehlen die JRE von http://www.java.com oder von http://www.azul.com.

    Linux
    • Java wird für die Ausführung des Installationsprogramms benötigt. Mindestens Java 1.8.0_111 wird benötigt. Wir empfehlen die JRE von http://www.java.com oder von http://www.azul.com.

    • Für die Verarbeitung von RAW-Bildern sollte eine aktuelle Version von Dave Coffin's dcraw im Verzeichnis /usr/bin/ installiert sein. Installationspakete sind erhältlich unter http://linuxappfinder.com/package/dcraw.

    Installation

    Installation unter Windows

    • Laden Sie die Installationsdatei (EXE-Datei) herunter.
    • Führen Sie diese Datei als Administrator aus.
    • Folgen Sie dann den Anweisungen des Installationsprogramms.

    Installation unter Mac OSX

    • Laden Sie die Installationsdatei (JAR-Datei) herunter.
    • Öffnen Sie ein Terminal-Fenster und geben sie folgendes ein:
      java -jar {Installationsdatei vollständiger Pfad}
    • Folgen Sie dann den Anweisungen des Installationsprogramms.
    • Desktop-Shortcuts können manuell als Aliasse erzeugt werden.

    Installation unter Linux

    • Laden Sie die Installationsdatei (JAR-Datei) herunter.
    • Möglicherweise müssen dann die Installationsdatei als ausführbares Programm markieren (Eigenschaften).
    • Öffnen Sie ein Terminal-Fenster und geben sie folgendes ein:
      java -jar {Installationsdatei vollständiger Pfad}
    • Folgen Sie dann den Anweisungen des Installationsprogramms.
    • Wurden Desktop-Shortcuts erzeugt, müssen Sie diese möglicherweise noch als ausführbares Programm markieren.

    Installation auf Wechseldatenträger

    Es ist möglich, ZoRa Photo Director auf Wechseldatenträger (z.B. Memory Stick) zu installieren. Um portabel zu bleiben, installieren Sie am Besten auch eine JRE im Unterordner ZoRa/jre/.
    Auch Kataloge können leicht auf Wechselmedien untergebracht werden. Bei Verwendung von Vorschaubilder mit mittlerer Größe nimmt jeder Katalogeintrag weniger als 20 KB Speicherplatz ein. Auch große Kataloge mit mehreren 10.000 Bildern finden deshalb auf einem Memory Stick Platz.

    Deinstallation

    Um ZoRa Photo Director zu deinstallieren,  gehen Sie zum Ordner {Installationsverzeichnis}/Uninstaller/ und Doppelklicken die Datei ZoRaPDUninstaller.jar. Alternativ können Sie die Datei auch mit einem Rechtsklick und Auswahl von Öffnen mit... > Java ausführen. Eine dritte Möglichkeit ist, den Installationsprozess von der Kommandzeile aus zu starten:

    java -jar {Installationsverzeichnis}/Uninstaller/ZoRaPDUninstaller.jar

    Unter Linux müssen Sie möglicherweise ZoRaPDUninstaller.jar erst als ausführbares Programm markieren (Eigenschaften).

    Anschließend haben Sie die Option, die benutzerdefinierten Anwendungsdaten im Installationsverzeichnis mit zu löschen - oder auch nicht, falls Sie die Anwendung später noch einmal installieren wollen.


    Bekannte Probleme

    • Falls nach einem Update zu einer neuen Version die Anwendung nicht sauber startet, löschen Sie den alten Installationsordner komplett und installieren neu.
    • Unter Windows gibt es Kompatibiltätsprobleme mit Google Maps. Verwenden Sie eine andere Map-Engine (Auswahl im Kartenfenster)
